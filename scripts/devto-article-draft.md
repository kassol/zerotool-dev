I got tired of opening random online tools that show ads, require accounts, or send my data to a server. So I built [ZeroTool](https://zerotool.dev) — a collection of browser-based developer utilities where everything runs client-side.

## What's in it

15 tools so far:

- **JSON Formatter** — prettify, minify, validate
- **JWT Decoder** — inspect header/payload, see expiry status
- **Regex Tester** — real-time match highlighting, capture groups
- **Diff Checker** — line-by-line comparison with LCS algorithm
- **Password Generator** — cryptographically secure, with entropy meter
- **Hash Generator** — MD5, SHA-1, SHA-256, SHA-384, SHA-512
- **Base64 Encode/Decode**
- **URL Encode/Decode**
- **UUID Generator** — v4, single or batch
- **Text Case Converter** — UPPER, lower, Title, camelCase, snake_case, kebab-case
- **Markdown Preview** — live editor with GFM support
- **Color Converter** — HEX ↔ RGB ↔ HSL
- **Timestamp Converter** — Unix ↔ human-readable, auto-detects ms vs s
- **QR Code Generator** — URL, text, Wi-Fi; download as PNG
- **Cron Expression Parser** — plain-English explanation + next 10 run times

## Why no framework for tool logic?

Each tool is a single `.astro` file with an inline `<script>` tag. No React, no Vue, no build-time complexity per tool. The browser already has everything needed for string manipulation, crypto (`SubtleCrypto`), canvas (QR), and `TextEncoder`.

The site itself is Astro — handles routing, SEO, layouts. Zero JS framework overhead on the client.

## Privacy architecture

The entire value proposition is: **your data never leaves the browser tab.**

- JSON you paste → parsed by `JSON.parse()` locally
- Passwords → generated by `crypto.getRandomValues()`
- Hashes → `crypto.subtle.digest()`
- QR codes → canvas rendering via a bundled library

No `fetch()` calls. No analytics pixels on tool pages (Google Analytics only on the homepage, with partytown for non-blocking load).

## Stack

- **Astro 5** — static output, filesystem routing, MDX for blog
- **Cloudflare Pages** — edge CDN, free tier
- **@astrojs/sitemap** — auto-generated sitemap
- **JSON-LD** — structured data on every page
