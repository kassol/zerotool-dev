Welcome to your new Astro SEO site.

## Getting Started

Edit this file at `src/content/blog/hello-world.mdx` to create your first article.

Add your AdSense publisher ID to `.env.local` and the ads will appear automatically.

<AdUnit slot={import.meta.env.PUBLIC_ADSENSE_SLOT_TOP ?? ''} format="rectangle" />

## Writing Content

Create new `.mdx` files in `src/content/blog/` with the required frontmatter:

- `title` and `description` are required for SEO
- Set `lang: "zh"` for Chinese articles (routed to `/zh/blog/...`)
- Set `draft: true` to exclude from builds
