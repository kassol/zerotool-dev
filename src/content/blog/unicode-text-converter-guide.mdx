## What Unicode Text Conversion Does

Unicode includes many alphabets and symbol sets beyond standard ASCII. Several blocks contain characters that visually resemble styled versions of the Latin alphabet:

- **Mathematical Bold** (`рқ—”рқ—•рқ—–`) вҖ” looks like bold text
- **Mathematical Italic** (`рқҗҙрқҗөрқҗ¶`) вҖ” looks like italic text
- **Mathematical Bold Italic** (`рқ‘Ёрқ‘©рқ‘Ә`) вҖ” bold and italic combined
- **Mathematical Monospace** (`рқҷ°рқҷұрқҷІ`) вҖ” looks like code/terminal font
- **Fullwidth** (`пјЎ пјў пјЈ`) вҖ” wide characters used in East Asian typography
- **Small Caps** (`бҙҖ Кҷ бҙ„`) вҖ” uppercase letterforms at lowercase size
- **Circled** (`в’¶ в’· в’ё`) вҖ” letters in circles
- **Squared** (`рҹ„° рҹ„ұ рҹ„І`) вҖ” letters in squares

These are not formatting instructions вҖ” they are actual Unicode characters. That's why they work in plain-text environments where HTML and Markdown are stripped.

## Accessibility and Limitations

### Screen Readers

Screen readers handle Unicode styled text inconsistently. A screen reader may read "рқ—ӣрқ—Ірқ—№рқ—№рқ—ј" as "mathematical bold capital H mathematical bold small e..." rather than "Hello". For content that must be accessible, use HTML/CSS formatting instead.

### Searchability

Unicode bold "рқ—ӣрқ—Ірқ—№рқ—№рқ—ј" is not the same string as "Hello". Search engines and in-page search will not find bold Unicode text when searching for the plain version. Keep this in mind for SEO-critical content.

### Missing Characters

Not all characters have Unicode styled equivalents. Numbers (0вҖ“9) have bold/monospace variants but may not have italic versions. Special characters, accented letters, and non-Latin scripts typically don't have Mathematical styled equivalents.

### Rendering Across Platforms

Most modern operating systems and browsers render these Unicode characters correctly. Older systems or environments with limited font support may show replacement characters (в–Ў) instead.

## Unicode Ranges by Style

For developers who want to implement this programmatically:

| Style | Uppercase start | Lowercase start |
|-------|----------------|----------------|
| Bold | U+1D400 | U+1D41A |
| Italic | U+1D434 | U+1D44E |
| Bold Italic | U+1D468 | U+1D482 |
| Monospace | U+1D670 | U+1D68A |
| Fullwidth | U+FF21 | U+FF41 |

```javascript
function toBold(char) {
  const code = char.charCodeAt(0);
  if (code >= 65 && code <= 90) {  // A-Z
    return String.fromCodePoint(0x1D400 + code - 65);
  }
  if (code >= 97 && code <= 122) {  // a-z
    return String.fromCodePoint(0x1D41A + code - 97);
  }
  return char;
}

"Hello".split("").map(toBold).join("");  // рқ—ӣрқ—Ірқ—№рқ—№рқ—ј
```
