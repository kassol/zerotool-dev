<h2>使用方法</h2>
<ol>
  <li>在输入框中粘贴或输入任意 URL。</li>
  <li>工具在输入时即时解析，无需点击按钮。</li>
  <li>点击任意字段旁的 <strong>Copy</strong> 按钮可复制对应值。</li>
  <li>若 URL 包含查询参数，会在下方单独以键值表格展示。</li>
</ol>

<h2>各字段含义</h2>
<p>
  以 URL <code>https://user:pass@example.com:8080/path?key=val#section</code> 为例：
</p>
<ul>
  <li><strong>Protocol（协议）</strong> — <code>https:</code>（scheme）</li>
  <li><strong>Username / Password（用户名/密码）</strong> — URL 中内嵌的凭据</li>
  <li><strong>Host</strong> — <code>example.com:8080</code>（主机名+端口）</li>
  <li><strong>Hostname（主机名）</strong> — <code>example.com</code>（不含端口）</li>
  <li><strong>Port（端口）</strong> — <code>8080</code>（若为协议默认端口则为空）</li>
  <li><strong>Pathname（路径）</strong> — <code>/path</code></li>
  <li><strong>Search（查询字符串）</strong> — <code>?key=val</code></li>
  <li><strong>Hash（哈希）</strong> — <code>#section</code></li>
  <li><strong>Origin</strong> — <code>https://example.com:8080</code>（协议+主机）</li>
</ul>
