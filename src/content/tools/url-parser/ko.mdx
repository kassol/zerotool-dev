<h2>사용 방법</h2>
<ol>
  <li>입력 필드에 URL을 붙여넣거나 입력합니다.</li>
  <li>입력과 동시에 즉시 파싱됩니다. 버튼을 누를 필요가 없습니다.</li>
  <li>각 필드 옆 <strong>Copy</strong> 버튼을 클릭하면 해당 값을 복사할 수 있습니다.</li>
  <li>쿼리 파라미터가 있으면 아래에 별도 키-값 테이블로 표시됩니다.</li>
</ol>

<h2>각 필드 의미</h2>
<p>
  URL <code>https://user:pass@example.com:8080/path?key=val#section</code>의 경우:
</p>
<ul>
  <li><strong>Protocol(프로토콜)</strong> — <code>https:</code> (스킴)</li>
  <li><strong>Username / Password</strong> — URL에 내장된 인증 정보</li>
  <li><strong>Host</strong> — <code>example.com:8080</code> (호스트명+포트)</li>
  <li><strong>Hostname</strong> — <code>example.com</code> (포트 제외)</li>
  <li><strong>Port</strong> — <code>8080</code> (기본 포트면 비어 있음)</li>
  <li><strong>Pathname</strong> — <code>/path</code></li>
  <li><strong>Search</strong> — <code>?key=val</code> (전체 쿼리 문자열)</li>
  <li><strong>Hash</strong> — <code>#section</code></li>
  <li><strong>Origin</strong> — <code>https://example.com:8080</code> (스킴+호스트)</li>
</ul>
