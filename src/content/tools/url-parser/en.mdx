<h2>How to Use</h2>
<ol>
  <li>Paste or type any URL into the input field.</li>
  <li>The tool parses it instantly as you type — no button needed.</li>
  <li>Click <strong>Copy</strong> next to any field to copy that value.</li>
  <li>If the URL has query parameters, they appear in a separate key-value table below.</li>
</ol>

<h2>What Each Field Means</h2>
<p>
  The URL <code>https://user:pass@example.com:8080/path?key=val#section</code> breaks down as:
</p>
<ul>
  <li><strong>Protocol</strong> — <code>https:</code> (the scheme)</li>
  <li><strong>Username / Password</strong> — credentials embedded in the URL</li>
  <li><strong>Host</strong> — <code>example.com:8080</code> (hostname + port)</li>
  <li><strong>Hostname</strong> — <code>example.com</code> (no port)</li>
  <li><strong>Port</strong> — <code>8080</code> (empty if default for the scheme)</li>
  <li><strong>Pathname</strong> — <code>/path</code></li>
  <li><strong>Search</strong> — <code>?key=val</code> (full query string)</li>
  <li><strong>Hash</strong> — <code>#section</code></li>
  <li><strong>Origin</strong> — <code>https://example.com:8080</code> (scheme + host)</li>
</ul>
