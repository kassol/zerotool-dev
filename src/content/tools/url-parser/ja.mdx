<h2>使い方</h2>
<ol>
  <li>入力欄に URL を貼り付けるか入力します。</li>
  <li>入力と同時に即座に解析されます。ボタンは不要です。</li>
  <li>各フィールド横の <strong>Copy</strong> ボタンをクリックしてその値をコピーできます。</li>
  <li>クエリパラメータがある場合は、下部に個別のキーバリューテーブルとして表示されます。</li>
</ol>

<h2>各フィールドの意味</h2>
<p>
  URL <code>https://user:pass@example.com:8080/path?key=val#section</code> の場合：
</p>
<ul>
  <li><strong>Protocol（プロトコル）</strong> — <code>https:</code>（スキーム）</li>
  <li><strong>Username / Password</strong> — URL に埋め込まれた認証情報</li>
  <li><strong>Host</strong> — <code>example.com:8080</code>（ホスト名+ポート）</li>
  <li><strong>Hostname</strong> — <code>example.com</code>（ポートなし）</li>
  <li><strong>Port</strong> — <code>8080</code>（スキームのデフォルトポートの場合は空）</li>
  <li><strong>Pathname</strong> — <code>/path</code></li>
  <li><strong>Search</strong> — <code>?key=val</code>（クエリ文字列全体）</li>
  <li><strong>Hash</strong> — <code>#section</code></li>
  <li><strong>Origin</strong> — <code>https://example.com:8080</code>（スキーム+ホスト）</li>
</ul>
