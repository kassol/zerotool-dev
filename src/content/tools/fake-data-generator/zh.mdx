<h2>使用方法</h2>
<ol>
  <li>勾选所需数据字段（如全名、邮箱、电话）。</li>
  <li>设置生成数量（1–100，默认 10）。</li>
  <li>选择输出格式：<strong>JSON</strong> 或 <strong>CSV</strong>。</li>
  <li>点击<strong>生成</strong>。</li>
  <li>点击<strong>复制</strong>复制到剪贴板，或点击<strong>下载</strong>保存文件。</li>
</ol>

<h2>使用场景</h2>
<p>
  假数据在软件开发各阶段不可或缺：用于填充开发数据库而不污染真实用户数据，为前端开发模拟 API 响应，或为需要看起来真实的 QA 测试和 UI 演示填充电子表格。
</p>
<p>
  所有生成均在客户端完成，零依赖。姓名、地址等值从精选词库中随机组合。UUID 生成使用浏览器的 <code>crypto.randomUUID()</code> API，具备加密级别的随机性。
</p>
