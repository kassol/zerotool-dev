<h2>How to Use</h2>
<ol>
  <li>Paste your SQL query into the input area.</li>
  <li>Click <strong>Format</strong> to beautify or <strong>Minify</strong> to compress.</li>
  <li>Adjust indent size and keyword casing with the options.</li>
  <li>Click <strong>Copy</strong> to copy the formatted result.</li>
</ol>

<h2>Worked Examples</h2>
<h3>Before formatting</h3>

```sql
select u.id, u.name, o.total from users u join orders o on u.id = o.user_id where o.total > 100 and u.active = 1 order by o.total desc limit 10;
```

<h3>After formatting</h3>

```sql
SELECT
  u.id,
  u.name,
  o.total
FROM users u
JOIN orders o ON u.id = o.user_id
WHERE
  o.total > 100
  AND u.active = 1
ORDER BY o.total DESC
LIMIT 10;
```

<h2>Why Format SQL?</h2>
<p>Formatted SQL is easier to read, review, and debug. Consistent formatting helps teams maintain coding standards and makes complex queries with multiple JOINs and subqueries much more manageable.</p>
