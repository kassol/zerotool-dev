<h2>사용 방법</h2>
<ol>
  <li>SQL 쿼리를 입력창에 붙여넣습니다.</li>
  <li><strong>정렬</strong>을 클릭하여 읽기 쉬운 SQL 형식으로 변환합니다.</li>
  <li><strong>압축</strong>을 클릭하여 공백을 제거한 한 줄 형식으로 만듭니다.</li>
  <li>결과를 <strong>복사</strong>합니다.</li>
</ol>

<h2>SQL 포매팅에 대하여</h2>
<p>정렬된 SQL은 읽기 쉽고 코드 리뷰나 디버깅이 용이해집니다. 또한 ORM이 생성한 압축된 SQL을 읽을 때 정렬 도구가 매우 유용합니다. 반면 로그나 API 파라미터로 전달할 때는 압축 형식이 적합합니다.</p>
