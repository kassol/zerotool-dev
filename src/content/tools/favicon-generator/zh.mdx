<h2>为什么仍然需要一整套 favicon</h2>

<p>网站图标早已不只是浏览器标签上的小图。它出现在搜索结果中域名旁边、收藏夹里、iOS 用户把网站添加到主屏幕时的图标位、Android 的 PWA 安装卡片上。每个场景需要的尺寸不同，少数场景还要求略不同的格式。只放一个 <code>favicon.ico</code> 在视网膜屏和可安装 PWA 上会留下肉眼可见的毛刺。</p>

<p>本工具一键产出整套现代图标：兼容旧浏览器的 ICO、对应现代 <code>&lt;link rel="icon"&gt;</code> 栈的多尺寸 PNG、iOS 用的 apple-touch-icon，以及含 maskable Android 图标声明的 webmanifest。</p>

<h2>使用方法</h2>

<ol>
  <li>选择一种素材。四个标签分别接受 emoji（预设或自由粘贴）、1–4 个排版字符、上传的 PNG/JPG/SVG 图片（≤ 10 MB），或粘贴的 SVG 代码。</li>
  <li>设置背景与形状。透明适合带蒙版的设计，纯色适合品牌标识。形状提供方形、圆角（12.5% 圆角半径）和圆形三种。Padding 滑块让内容远离安全区边缘。</li>
  <li>盯紧实时预览。四个关键尺寸 16/32/64/180 px 同步渲染。16 和 32 px 决定你的图标在标签栏是否清晰；180 px 是 iOS 主屏图标。</li>
  <li>点击 <strong>Generate package</strong>。浏览器约 1 秒内产出 9 个文件。<strong>Download ZIP</strong> 一键保存；<strong>Copy HTML snippet</strong> 复制需粘贴到 <code>&lt;head&gt;</code> 的 <code>&lt;link&gt;</code> 标签。</li>
</ol>

<h2>HTML 片段说明</h2>

<p>生成的代码片段约 7 行，覆盖每个现代场景：</p>

<ul>
  <li><code>&lt;link rel="icon" type="image/x-icon" href="/favicon.ico"&gt;</code> — 老浏览器和仍优先解析 favicon.ico 的收藏夹的兜底。</li>
  <li><code>&lt;link rel="icon" type="image/png" sizes="..."&gt;</code> 16/32/48 三档 — 现代浏览器优先选择匹配 sizes 属性的 PNG，跳过 ICO。</li>
  <li><code>&lt;link rel="apple-touch-icon"&gt;</code> 180×180 — iOS Safari 在「添加到主屏幕」时使用。</li>
  <li><code>&lt;link rel="manifest"&gt;</code> — Android Chrome 和 PWA 安装器读取 site.webmanifest，里面用 <code>purpose: "any maskable"</code> 声明 192 与 512 PNG。</li>
  <li><code>&lt;meta name="theme-color"&gt;</code> — 控制移动端浏览器地址栏与已安装 PWA 标题栏的颜色。</li>
</ul>

<h2>图标包是怎么生成的</h2>

<p>渲染器对每个目标尺寸单独建立 canvas，应用所选的背景与形状蒙版，按 padding 偏移把素材居中绘制，再调用 <code>canvas.toBlob('image/png')</code> 取得紧凑的 PNG 数据。16/32/48 三个 PNG 通过 PNG 压缩条目格式打包成单一 favicon.ico——三段 PNG 数据接在 ICONDIR 头和三条 16 字节 ICONDIRENTRY 之后。</p>

<p>ZIP 写入器使用 STORED 方法（不做 DEFLATE）：PNG 本身已经是压缩格式，再做 DEFLATE 收益不足 1%，却要多带 5 KB 代码。文件名在通用位标志中标记 UTF-8，确保未来你给文件改名时非 ASCII 名称也不会乱码。</p>

<ul>

  <li>页面缓存后可离线工作。Canvas API、ZIP 与 ICO 写入器全部内联在页面包里。</li>

<h2>常见坑</h2>

<ul>
  <li><strong>SVG 含外链图片。</strong> 当 SVG 引用 <code>&lt;image href="https://..."&gt;</code> 时浏览器拒绝把 canvas 读回为 blob（跨域污染）。粘贴前先把外链图片转成 Data URI 内联。</li>
  <li><strong>Linux 工作机上 emoji 显示空框。</strong> 某些 IT 管控的 Linux 发行版未预装彩色 emoji 字体。预览会显示空框，切到 Text 或 Image 标签即可绕过。</li>
  <li><strong>16 px 上的细节会消失。</strong> 线宽小于 1.5 px 在 16×16 上会消失。如果 32 上清晰、16 上没了，把设计简化或用背景色提供分割。</li>
</ul>
