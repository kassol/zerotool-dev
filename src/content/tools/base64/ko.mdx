<h2>사용 방법</h2>
<ol>
  <li><strong>인코딩</strong> 또는 <strong>디코딩</strong> 모드를 선택합니다.</li>
  <li>텍스트 또는 Base64 문자열을 입력창에 붙여넣습니다.</li>
  <li>작업 버튼을 클릭하여 변환합니다.</li>
  <li><strong>복사</strong> 버튼을 클릭하여 결과를 클립보드에 복사합니다.</li>
  <li><strong>스왑</strong>으로 출력을 입력창으로 이동하여 연속 작업이 가능합니다.</li>
</ol>

<h2>주요 사용 사례</h2>
<ul>
  <li><strong>API 인증:</strong> Basic Auth 헤더에는 Base64 인코딩된 인증 정보가 사용됩니다.</li>
  <li><strong>Data URI:</strong> CSS/HTML에 인라인으로 삽입하는 이미지나 폰트는 Base64 인코딩되어 있습니다.</li>
  <li><strong>JWT 토큰:</strong> 헤더와 페이로드는 Base64url 인코딩되어 있습니다.</li>
  <li><strong>이메일:</strong> MIME 첨부 파일은 Base64 인코딩을 사용합니다.</li>
</ul>
