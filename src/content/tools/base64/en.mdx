<h2>Common Use Cases</h2>
<ul>
  <li><strong>API authentication:</strong> Basic Auth headers use Base64-encoded credentials.</li>
  <li><strong>Data URIs:</strong> Inline images and fonts in CSS/HTML are Base64-encoded.</li>
  <li><strong>JWT tokens:</strong> The header and payload sections are Base64url-encoded.</li>
  <li><strong>Email:</strong> MIME attachments are Base64-encoded.</li>
</ul>
