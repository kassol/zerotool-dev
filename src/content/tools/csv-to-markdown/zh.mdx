<h2>使用方法</h2>
<ol>
  <li>将 CSV 数据粘贴到左侧面板。</li>
  <li>选择列对齐方式：<strong>左对齐</strong>、<strong>居中</strong>或<strong>右对齐</strong>。</li>
  <li>右侧面板自动显示 Markdown 表格。</li>
  <li>点击<strong>复制</strong>将结果复制到剪贴板。</li>
</ol>

<h2>Markdown 表格语法</h2>

Markdown 表格由表头行、分隔行和数据行组成，竖线符（`|`）分隔列。

```
| 姓名  | 年龄 | 城市     |
| :---- | :--- | :------- |
| Alice | 30   | 纽约     |
| Bob   | 25   | 伦敦     |
```

分隔行控制对齐：`:---` = 左对齐，`:---:` = 居中，`---:` = 右对齐。

<h2>使用技巧</h2>
<ul>
  <li>单元格值中的竖线（<code>|</code>）会自动转义为 <code>\|</code>。</li>
  <li>引号 CSV 字段内的换行符在输出中会被合并为空格。</li>
  <li>如需 JSON 输出，请使用 <a href="/tools/csv-json">CSV to JSON</a> 工具。</li>
</ul>
