<h2>How to Use</h2>
<ol>
  <li>Paste your CSV data into the left panel.</li>
  <li>Choose a column alignment: <strong>Left</strong>, <strong>Center</strong>, or <strong>Right</strong>.</li>
  <li>The Markdown table appears in the right panel automatically.</li>
  <li>Click <strong>Copy</strong> to copy the result to your clipboard.</li>
</ol>

<h2>Markdown Table Syntax</h2>

A Markdown table consists of a header row, a separator row, and data rows. Pipe characters (`|`) delimit columns.

```
| Name  | Age | City     |
| :---- | :-- | :------- |
| Alice | 30  | New York |
| Bob   | 25  | London   |
```

The separator row controls alignment: `:---` = left, `:---:` = center, `---:` = right.

<h2>Tips</h2>
<ul>
  <li>Pipe characters (<code>|</code>) inside cell values are automatically escaped as <code>\|</code>.</li>
  <li>Newlines within quoted CSV fields are collapsed to a single space in the output.</li>

</ul>
