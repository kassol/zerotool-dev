<h2>사용 방법</h2>
<ol>
  <li>왼쪽 패널에 CSV 데이터를 붙여 넣습니다.</li>
  <li>열 정렬을 선택합니다: <strong>왼쪽</strong>, <strong>가운데</strong>, 또는 <strong>오른쪽</strong>.</li>
  <li>오른쪽 패널에 자동으로 Markdown 표가 표시됩니다.</li>
  <li><strong>복사</strong>를 클릭해 결과를 클립보드에 복사합니다.</li>
</ol>

<h2>Markdown 표 문법</h2>

Markdown 표는 헤더 행, 구분 행, 데이터 행으로 구성되며 파이프 문자(`|`)로 열을 구분합니다.

```
| Name  | Age | City     |
| :---- | :-- | :------- |
| Alice | 30  | New York |
| Bob   | 25  | London   |
```

구분 행으로 정렬을 제어합니다: `:---` = 왼쪽, `:---:` = 가운데, `---:` = 오른쪽.

<h2>팁</h2>
<ul>
  <li>셀 값 내의 파이프 문자(<code>|</code>)는 자동으로 <code>\|</code>로 이스케이프됩니다.</li>
  <li>따옴표 CSV 필드 내의 줄바꿈은 출력에서 공백 하나로 변환됩니다.</li>
  <li>JSON 출력이 필요한 경우 <a href="/tools/csv-json">CSV to JSON</a> 도구를 사용하세요.</li>
</ul>
