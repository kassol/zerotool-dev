<h2>使用方法</h2>
<ol>
  <li>填入网络名称（<strong>SSID</strong>）—— 和路由器中显示完全一致，包括大小写与空格。</li>
  <li>填入 <strong>密码</strong>，点眼睛图标可临时显示密码核对。</li>
  <li>选择路由器使用的 <strong>加密方式</strong>（WPA 选项覆盖 WPA / WPA2 / WPA3）。</li>
  <li>如果 SSID 不广播，请勾选 <strong>隐藏网络</strong>。</li>
  <li>点击 <strong>生成二维码</strong>，用任意手机摄像头扫一下试试。</li>
  <li>点击 <strong>下载 PNG</strong>，保存图片用于打印或分享。</li>
</ol>

<h2>Wi-Fi 二维码字符串格式</h2>

<p>Wi-Fi 二维码遵循一种纯文本标准格式，手机摄像头能原生解析：</p>

<pre><code>WIFI:T:WPA;S:&lt;ssid&gt;;P:&lt;password&gt;;;</code></pre>

<ul>
  <li><code>T</code> — 加密类型（<code>WPA</code> / <code>WEP</code> / <code>nopass</code>）</li>
  <li><code>S</code> — SSID（网络名称）</li>
  <li><code>P</code> — 密码（<code>T</code> 为 <code>nopass</code> 时省略）</li>
  <li><code>H:true</code> — 可选，标记隐藏网络</li>
  <li><code>;;</code> — 结尾终止符</li>
</ul>

<p>密码中如果包含 <code>{'\\'}</code>、<code>;</code>、<code>,</code>、<code>{'"'}</code>、<code>:</code> 等保留字符，会自动加反斜杠转义，含标点的密码也能正常工作。</p>
