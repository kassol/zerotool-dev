<h2>사용 방법</h2>
<ol>
  <li>네트워크 이름(<strong>SSID</strong>)을 입력합니다 — 라우터에 표시된 그대로 대소문자와 공백까지 일치시키세요.</li>
  <li><strong>비밀번호</strong>를 입력합니다. 눈 아이콘을 누르면 일시적으로 표시해 확인할 수 있습니다.</li>
  <li>라우터에서 사용하는 <strong>암호화 방식</strong>을 선택합니다 (WPA 옵션은 WPA / WPA2 / WPA3 모두 포함).</li>
  <li>SSID 가 브로드캐스트되지 않으면 <strong>숨김 네트워크</strong>를 체크하세요.</li>
  <li><strong>QR 코드 생성</strong>을 클릭한 뒤 휴대폰 카메라로 스캔해 동작을 확인합니다.</li>
  <li><strong>PNG 다운로드</strong>로 이미지를 저장해 인쇄·공유에 사용하세요.</li>
</ol>

<h2>Wi-Fi QR 문자열 포맷</h2>

<p>Wi-Fi QR 코드는 스마트폰 카메라가 기본 인식하는 다음 표준 평문 포맷을 따릅니다:</p>

<pre><code>WIFI:T:WPA;S:&lt;ssid&gt;;P:&lt;password&gt;;;</code></pre>

<ul>
  <li><code>T</code> — 암호화 유형 (<code>WPA</code> / <code>WEP</code> / <code>nopass</code>)</li>
  <li><code>S</code> — SSID (네트워크 이름)</li>
  <li><code>P</code> — 비밀번호 (<code>T</code> 가 <code>nopass</code> 면 생략)</li>
  <li><code>H:true</code> — 선택, 숨김 네트워크 표시</li>
  <li><code>;;</code> — 종료 마커</li>
</ul>

<p>비밀번호에 <code>{'\\'}</code>, <code>;</code>, <code>,</code>, <code>{'"'}</code>, <code>:</code> 같은 예약 문자가 포함되면 백슬래시로 자동 이스케이프되어, 특수문자가 든 비밀번호도 정상 동작합니다.</p>

<h2>주요 활용 사례</h2>
<ul>
  <li><strong>카페·레스토랑·에어비앤비:</strong>QR 을 인쇄해 카운터에 두면 손님이 비밀번호 입력 없이 접속할 수 있습니다.</li>
  <li><strong>사무실 게스트 네트워크:</strong>회의실 벽에 붙여 두면 방문자가 셀프 접속.</li>
  <li><strong>가정용:</strong>스마트폰에 저장해 두면 손님이 왔을 때 즉시 공유 가능.</li>
  <li><strong>IoT 기기 페어링:</strong>스마트홈 기기 페어링 과정에서 Wi-Fi QR 스캔을 활용할 수 있습니다.</li>
</ul>
