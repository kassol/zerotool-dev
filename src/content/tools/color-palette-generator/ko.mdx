<h2>사용 방법</h2>
<ol>
  <li>입력 필드에 <strong>HEX 값</strong>(예: <code>#3b82f6</code>)을 입력하거나 <strong>색상 선택기</strong>로 기본 색상을 선택합니다.</li>
  <li>4가지 팔레트 카드(<strong>보색</strong>, <strong>유사색</strong>, <strong>삼각 배색</strong>, <strong>사각 배색</strong>)가 즉시 업데이트됩니다.</li>
  <li>색상 스와치를 클릭하여 <strong>HEX 값을 복사</strong>합니다.</li>
  <li>카드 오른쪽 상단의 <strong>모두 복사</strong> 버튼으로 해당 배색의 모든 HEX 값을 복사합니다.</li>
</ol>

<h2>색상 조화 방식 설명</h2>

<h3>보색 (Complementary)</h3>
<p>색상환에서 180° 반대편에 위치한 2색. 강한 대비와 시각적 임팩트를 제공합니다. CTA 버튼, 강조 요소, 대비가 필요한 디자인에 적합합니다.</p>

<h3>유사색 (Analogous)</h3>
<p>기본 색상 좌우 ±30° 범위의 3색. 자연스럽고 조화로운 분위기. 자연 친화적 디자인과 시각적 편안함을 중시하는 UI에 잘 어울립니다.</p>

<h3>삼각 배색 (Triadic)</h3>
<p>색상환을 120° 간격으로 3등분한 3색. 균형감과 활기를 동시에 가집니다. 다채롭고 에너지 있는 브랜드 디자인에 적합합니다.</p>

<h3>사각 배색 (Tetradic)</h3>
<p>색상환을 90° 간격으로 4등분한 4색. 가장 풍부한 팔레트로 다양한 색상 표현이 가능합니다. 하나의 주색과 나머지 3색을 보조색으로 활용하는 것이 일반적입니다.</p>

<h2>HSL 색상 모델</h2>
<ul>
  <li><strong>색조 (H):</strong> 색상환의 각도 (0–360°). 빨강=0°, 초록=120°, 파랑=240°.</li>
  <li><strong>채도 (S):</strong> 색상 선명도. 0%=회색, 100%=최고 채도.</li>
  <li><strong>명도 (L):</strong> 0%=검정, 50%=순색, 100%=흰색.</li>
</ul>
