<h2>使用方法</h2>
<ol>
  <li>在左侧面板粘贴或输入 Markdown，右侧会实时显示预览。</li>
  <li>可在顶部输入框中修改文件名（默认为 <code>document</code>）。</li>
  <li>点击<strong>转换并下载 .docx</strong>。</li>
  <li>用 Microsoft Word、Google Docs 或 LibreOffice 打开下载的文件。</li>
</ol>

<h2>支持的 Markdown 元素</h2>
<ul>
  <li><strong>标题</strong> — ATX 风格（<code>#</code> 至 <code>######</code>）和 setext 风格（用 <code>=</code> 或 <code>-</code> 下划线）</li>
  <li><strong>行内格式</strong> — 粗体（<code>**文字**</code>）、斜体（<code>*文字*</code>）、粗+斜体（<code>***文字***</code>）、行内代码（<code>`代码`</code>）</li>
  <li><strong>列表</strong> — 无序（<code>-</code>、<code>*</code>、<code>+</code>）和有序（<code>1.</code>、<code>2.</code>），支持嵌套</li>
  <li><strong>代码块</strong> — 三反引号或波浪线围栏，在 Word 中使用等宽字体</li>
  <li><strong>表格</strong> — GFM 管道表格，带表头行和列对齐</li>
  <li><strong>引用块</strong> — <code>&gt; 文字</code>，在 Word 中渲染为带左边框的斜体样式</li>
  <li><strong>分隔线</strong> — <code>---</code> 转换为 Word 中的段落边框</li>
  <li><strong>链接</strong> — <code>[文字](url)</code>，渲染为带下划线的蓝色文本</li>
</ul>

<h2>隐私</h2>
<p>所有处理均在客户端完成，文档内容不会被上传、存储或传输到任何地方。</p>
