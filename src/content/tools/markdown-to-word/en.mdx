<h2>How to Use</h2>
<ol>
  <li>Paste or type your Markdown in the left pane — a preview appears on the right.</li>
  <li>Optionally change the filename in the top field (defaults to <code>document</code>).</li>
  <li>Click <strong>Convert &amp; Download .docx</strong>.</li>
  <li>Open the downloaded file in Microsoft Word, Google Docs, or LibreOffice.</li>
</ol>

<h2>Supported Markdown Elements</h2>
<ul>
  <li><strong>Headings</strong> — ATX style (<code>#</code> through <code>######</code>) and setext style (underlined with <code>=</code> or <code>-</code>)</li>
  <li><strong>Inline formatting</strong> — bold (<code>**text**</code>), italic (<code>*text*</code>), bold+italic (<code>***text***</code>), inline code (<code>`code`</code>)</li>
  <li><strong>Lists</strong> — unordered (<code>-</code>, <code>*</code>, <code>+</code>) and ordered (<code>1.</code>, <code>2.</code>), with nesting</li>
  <li><strong>Fenced code blocks</strong> — triple backtick or tilde fences, monospace font in Word</li>
  <li><strong>Tables</strong> — GFM pipe tables with header row and column alignment</li>
  <li><strong>Blockquotes</strong> — <code>&gt; text</code> rendered with left border and italic style</li>
  <li><strong>Horizontal rules</strong> — <code>---</code> becomes a paragraph border in Word</li>
  <li><strong>Links</strong> — <code>[text](url)</code> rendered as underlined blue text</li>
</ul>
