<h2>使用方法</h2>
<ol>
  <li>在正则表达式输入框中填写你的表达式（不含斜杠）。</li>
  <li>在 <strong>Test String</strong> 区域粘贴或输入要测试的文本。</li>
  <li>切换标志位：<code>g</code>（全局匹配）、<code>i</code>（忽略大小写）、<code>m</code>（多行）、<code>s</code>（dotAll）。</li>
  <li>匹配结果会实时高亮显示，捕获组列在下方。</li>
</ol>

<h2>常用正则表达式</h2>
<ul>
  <li><code>\d+</code> — 一个或多个数字</li>
  <li><code>\w+</code> — 一个或多个单词字符（字母、数字、下划线）</li>
  <li><code>[a-zA-Z]+</code> — 仅字母</li>
  <li><code>^\s+|\s+$</code> — 首尾空白字符</li>
  <li><code>(\w+)@(\w+\.\w+)</code> — 简单邮箱捕获</li>
  <li><code>https?://\S+</code> — URL 链接</li>
</ul>
