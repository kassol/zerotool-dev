    answer: "Each vertex is expressed as two percentages: X% from the left edge and Y% from the top edge of the element's bounding box. A point at 50% 0% is the top-center, and 100% 100% is the bottom-right corner. Values outside 0–100% are valid and clip beyond the element boundary."

<h2>Understanding the Generated CSS</h2>
<p>
  The output sets <code>clip-path</code> on an <code>.element</code> class. Apply this property directly
  to any element — an <code>img</code>, a <code>div</code>, or a pseudo-element. All coordinates use
  percentages relative to the element's own width and height, so the shape scales correctly when the
  element is resized.
</p>

<h2>Common Use Cases</h2>
<p>
  CSS clip-path is used to create non-rectangular image crops, diagonal section dividers, decorative
  hero shapes, animated shape reveals, icon masks, and scroll-triggered morphing effects. The polygon
  mode is especially flexible — any convex or concave polygon with up to twelve vertices can be expressed
  as a clip-path without any SVG or canvas.
</p>
