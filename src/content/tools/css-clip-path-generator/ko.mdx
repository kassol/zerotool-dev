<h2>사용 방법</h2>
<ol>
  <li>탭으로 도형 유형을 선택합니다: <strong>폴리곤</strong>, <strong>원</strong>, <strong>타원</strong>, 또는 <strong>인셋</strong>.</li>
  <li><strong>폴리곤</strong>: 미리보기의 흰색 핸들을 드래그하거나 꼭짓점 목록에 X/Y 값을 직접 입력합니다. 프리셋 버튼(육각형·별·화살표·마름모·삼각형)을 시작점으로 활용하세요.</li>
  <li><strong>원</strong>: 슬라이더로 반경과 중심 위치를 조정합니다.</li>
  <li><strong>타원</strong>: 가로 반경·세로 반경·중심 위치를 독립적으로 설정합니다.</li>
  <li><strong>인셋</strong>: 각 가장자리 오프셋과 선택적 모서리 반경을 설정합니다.</li>
  <li><strong>색상</strong>과 <strong>이미지</strong> 배경을 전환하며 클리핑 결과를 확인합니다.</li>
  <li><strong>복사</strong> 버튼으로 생성된 CSS를 클립보드에 복사합니다.</li>
</ol>

<h2>생성된 CSS 이해하기</h2>
<p>
  출력은 <code>.element</code> 클래스에 <code>clip-path</code>를 설정합니다. 이 속성은 <code>img</code>·<code>div</code>·가상 요소 등 모든 요소에 직접 적용할 수 있습니다. 모든 좌표는 요소 자체의 너비와 높이에 대한 퍼센트로 표현되므로 요소 크기 변경 시 도형이 올바르게 스케일됩니다.
</p>

<h2>주요 활용 사례</h2>
<p>
  CSS clip-path는 비직사각형 이미지 크롭, 대각선 섹션 구분선, 장식적인 히어로 도형, 도형 나타내기 애니메이션, 아이콘 마스크, 스크롤 트리거 모핑 효과 등에 활용됩니다. 폴리곤 모드는 특히 유연하여 최대 12개 꼭짓점의 볼록·오목 다각형을 SVG나 Canvas 없이 표현할 수 있습니다.
</p>
