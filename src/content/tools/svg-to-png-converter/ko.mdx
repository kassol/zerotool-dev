<h2>사용 방법</h2>
<ol>
  <li>드래그 앤 드롭 또는 클릭으로 <strong>.svg 파일</strong>을 업로드하거나, 「SVG 코드 붙여넣기」 탭으로 전환하여 마크업을 붙여넣습니다.</li>
  <li><strong>출력 크기</strong>를 선택합니다: Original(1x), 2x, 4x, 또는 사용자 정의 픽셀.</li>
  <li><strong>배경</strong>을 선택합니다: 투명, 흰색, 또는 사용자 정의 색상.</li>
  <li><strong>Convert to PNG</strong>를 클릭하여 변환합니다.</li>
  <li>미리보기를 확인한 후 <strong>Download PNG</strong>를 클릭하여 저장합니다.</li>
</ol>

<h2>SVG를 PNG로 변환하는 이유</h2>
<ul>
  <li><strong>호환성:</strong> PNG는 브라우저, 오피스 소프트웨어, 문서 편집기에서 광범위하게 지원됩니다.</li>
  <li><strong>소셜 미디어:</strong> Twitter/X, LinkedIn, Slack 등의 플랫폼은 이미지 업로드 시 래스터 형식을 요구합니다.</li>
  <li><strong>고정 해상도:</strong> PNG는 지정된 해상도로 SVG를 스냅샷하여 의도치 않은 크기 변경을 방지합니다.</li>
  <li><strong>이메일 클라이언트:</strong> 대부분의 이메일 클라이언트는 인라인 SVG를 지원하지 않으므로 PNG가 더 안전한 선택입니다.</li>
</ul>
<p>
  모든 처리는 브라우저 내에서 완전히 이루어집니다. 파일이 서버에 업로드되지 않으므로
  기밀 로고나 상업적 아트워크에도 안심하고 사용할 수 있습니다.
</p>
