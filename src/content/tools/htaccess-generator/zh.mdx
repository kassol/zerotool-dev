<h2>使用方法</h2>
<ol>
  <li>用复选框开启或关闭各配置区域。</li>
  <li>在各区域内设置具体选项（缓存时长、www 偏好等）。</li>
  <li>底部的 <code>.htaccess</code> 输出框实时更新。</li>
  <li>点击 <strong>Copy</strong>，粘贴到你的 <code>.htaccess</code> 文件中即可。</li>
</ol>

<h2>各配置区域说明</h2>
<ul>
  <li><strong>强制 HTTPS：</strong>添加 RewriteRule，将所有 <code>http://</code> 请求 301 跳转到 <code>https://</code>。需要服务器启用 <code>mod_rewrite</code>。</li>
  <li><strong>WWW 跳转：</strong>统一域名的访问形式——强制带 <code>www.</code> 或不带 <code>www.</code>，避免重复内容问题。</li>
  <li><strong>目录索引：</strong>设置 Apache 在请求目录 URL 时默认返回的文件列表，多个文件名形成回退顺序。</li>
  <li><strong>浏览器缓存：</strong>使用 <code>mod_expires</code> 告知浏览器图片、CSS/JS 和字体的缓存有效期，减少重复请求，加快页面加载。</li>
  <li><strong>安全：</strong>禁止目录浏览（<code>Options -Indexes</code>），阻止直接访问 <code>.htaccess</code> 和 <code>.env</code> 文件，并可选择添加 XSS 和点击劫持防护响应头。</li>
  <li><strong>自定义跳转：</strong>使用 Apache <code>Redirect</code> 指令创建单路径跳转，可选 301（永久）或 302（临时）。</li>
</ul>
