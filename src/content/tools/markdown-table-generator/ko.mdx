<h2>사용 방법</h2>
<ol>
  <li><strong>시각적 편집</strong>: 열과 행을 추가하고 셀에 직접 입력합니다.</li>
  <li><strong>CSV 가져오기</strong>: CSV 데이터를 붙여넣어 표를 자동 생성합니다.</li>
  <li><strong>JSON 가져오기</strong>: 객체 배열 JSON을 붙여넣어 표를 생성합니다.</li>
  <li>생성된 Markdown 표를 <strong>복사</strong>합니다.</li>
</ol>

<h2>Markdown 표에 대하여</h2>
<p>Markdown 표는 기술 문서·README·Wiki 등에서 정보를 정리하여 표시하는 데 편리합니다. GitHub, GitLab, Notion, Obsidian 등 많은 Markdown 렌더러에서 올바르게 표시됩니다. 수작업으로 파이프(|)를 나열하는 것은 번거롭기 때문에 이 도구로 자동 생성하는 것을 권장합니다.</p>
