<h2>How to Use</h2>

<h3>Manual Editor</h3>
<ol>
  <li>Click any cell to edit it inline. The Markdown output updates in real time.</li>
  <li>Use <strong>+ Row</strong> / <strong>+ Column</strong> to add rows or columns, and <strong>− Row</strong> / <strong>− Column</strong> to remove them.</li>
  <li>In the header row, use the alignment buttons to set column alignment (left / center / right).</li>
  <li>Click <strong>Copy</strong> to copy the Markdown table to your clipboard.</li>
</ol>

<h3>CSV Paste</h3>
<ol>
  <li>Switch to the <strong>CSV Paste</strong> tab.</li>
  <li>Paste your CSV data — the first row is treated as headers.</li>
  <li>The tool auto-detects commas, tabs, or semicolons as delimiters and populates the table immediately.</li>
</ol>

<h3>JSON Paste</h3>
<ol>
  <li>Switch to the <strong>JSON Paste</strong> tab.</li>
  <li>Paste a JSON array of objects. Keys become column headers; each object becomes a data row.</li>
  <li>The table and Markdown output update instantly as you type.</li>
</ol>

<h2>Supported Formats</h2>
<p>
  This tool generates <strong>GitHub Flavored Markdown (GFM)</strong> tables, compatible with GitHub,
  GitLab, Notion, Obsidian, VS Code, and most modern Markdown renderers.
  Pipe characters (<code>|</code>) in cell content are automatically escaped.
</p>
