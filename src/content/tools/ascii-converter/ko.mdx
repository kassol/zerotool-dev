<h2>사용 방법</h2>
<ol>
  <li><strong>텍스트 → ASCII</strong>: 변환할 텍스트를 입력창에 붙여넣습니다.</li>
  <li><strong>ASCII → 텍스트</strong>: ASCII 코드(공백 구분)를 입력합니다.</li>
  <li>10진수·16진수·2진수 표시 형식을 선택할 수 있습니다.</li>
</ol>

<h2>ASCII 코드 활용 예시</h2>
<p>프로그래밍 학습, 디버깅, 데이터 통신 프로토콜 분석 등에 활용됩니다. 특수 문자나 제어 문자의 실제 값을 확인하고 싶을 때 편리합니다. 예를 들어 줄바꿈 코드의 차이(CRLF vs LF)를 확인하는 경우 등에 사용할 수 있습니다.</p>
