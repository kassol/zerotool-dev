<h2>What is ASCII?</h2>
<p>ASCII (American Standard Code for Information Interchange) is a character encoding standard that assigns numbers to 128 characters including letters, digits, punctuation, and control characters. It forms the foundation of modern text encoding systems like UTF-8.</p>
