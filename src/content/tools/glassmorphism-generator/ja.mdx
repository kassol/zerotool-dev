<h2>使い方</h2>
<ol>
  <li><strong>背景色</strong>を選択し、<strong>背景の不透明度</strong>を設定します（通常10〜25%）。</li>
  <li><strong>ブラー</strong>を大きくして（10〜20px推奨）すりガラス効果を得ます。</li>
  <li><strong>彩度</strong>でガラスの後ろに見える色の鮮やかさを調整します。</li>
  <li><strong>ボーダー色</strong>・<strong>ボーダーの不透明度</strong>・<strong>ボーダー幅</strong>を設定します。細い半透明のボーダーがガラスの縁を明確にします。</li>
  <li><strong>角丸</strong>でカードの角を丸めます（12〜20pxが一般的）。</li>
  <li>柔らかな<strong>シャドウ</strong>を加えてカードを背景から浮かせます。</li>
  <li>背景プリセット（サンセット・オーシャン・フォレスト・ナイト・オーロラ）を切り替えて様々な背景でエフェクトを確認します。</li>
  <li><strong>コピー</strong>をクリックして生成されたCSSをクリップボードにコピーします。</li>
</ol>

<h2>CSS出力リファレンス</h2>

```css
.glass {
  background: rgba(255, 255, 255, 0.15);
  backdrop-filter: blur(12px) saturate(180%);
  -webkit-backdrop-filter: blur(12px) saturate(180%);
  border: 1px solid rgba(255, 255, 255, 0.30);
  border-radius: 16px;
  box-shadow: 0 4px 24px rgba(0, 0, 0, 0.20);
}
```

<h2>よく使われるグラスモーフィズムのプリセット</h2>
<ul>
  <li><strong>ライトガラス（白）：</strong>background rgba(255,255,255,0.15)、blur 12px、border rgba(255,255,255,0.3)</li>
  <li><strong>ダークガラス：</strong>background rgba(0,0,0,0.25)、blur 16px、border rgba(255,255,255,0.1)</li>
  <li><strong>強いすりガラス：</strong>background rgba(255,255,255,0.25)、blur 20px、saturation 200%</li>
  <li><strong>ミニマル：</strong>background rgba(255,255,255,0.08)、blur 8px、ボーダーなし</li>
</ul>

<h2>ブラウザサポートについて</h2>
<p>backdrop-filterは主要ブラウザすべてでサポートされています（Chrome 76+、Firefox 103+、Safari 9+、Edge 79+）。必ず<code>backdrop-filter</code>と<code>-webkit-backdrop-filter</code>を両方記述してください。非対応ブラウザ向けのフォールバックとして、半透明の単色背景を用意することをお勧めします。</p>
