<h2>사용 방법</h2>
<ol>
  <li><strong>배경 색상</strong>을 선택하고 <strong>배경 투명도</strong>를 설정합니다 (보통 10–25%).</li>
  <li><strong>블러</strong>를 높여서 (10–20px 권장) frosted-glass 효과를 만듭니다.</li>
  <li><strong>채도</strong>로 유리 뒤에 보이는 색상의 선명도를 조정합니다.</li>
  <li><strong>테두리 색상</strong>, <strong>테두리 투명도</strong>, <strong>테두리 너비</strong>를 설정합니다 — 얇고 반투명한 테두리가 유리 가장자리를 정의합니다.</li>
  <li><strong>모서리 반경</strong>으로 카드 모서리를 둥글게 합니다 (12–20px가 일반적).</li>
  <li>부드러운 <strong>그림자</strong>를 추가하여 카드를 배경에서 들어올립니다.</li>
  <li>배경 프리셋 (선셋, 오션, 포레스트, 나이트, 오로라)을 전환하여 다양한 배경에서 효과를 테스트합니다.</li>
  <li><strong>복사</strong>를 클릭하여 생성된 CSS를 클립보드에 복사합니다.</li>
</ol>

<h2>CSS 출력 참조</h2>

```css
.glass {
  background: rgba(255, 255, 255, 0.15);
  backdrop-filter: blur(12px) saturate(180%);
  -webkit-backdrop-filter: blur(12px) saturate(180%);
  border: 1px solid rgba(255, 255, 255, 0.30);
  border-radius: 16px;
  box-shadow: 0 4px 24px rgba(0, 0, 0, 0.20);
}
```

<h2>일반적인 글래스모피즘 프리셋</h2>
<ul>
  <li><strong>밝은 유리 (흰색):</strong> background rgba(255,255,255,0.15), blur 12px, border rgba(255,255,255,0.3)</li>
  <li><strong>어두운 유리:</strong> background rgba(0,0,0,0.25), blur 16px, border rgba(255,255,255,0.1)</li>
  <li><strong>강한 frosted:</strong> background rgba(255,255,255,0.25), blur 20px, saturation 200%</li>
  <li><strong>미니멀:</strong> background rgba(255,255,255,0.08), blur 8px, 테두리 없음</li>
</ul>

<h2>브라우저 지원 안내</h2>
