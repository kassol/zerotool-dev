<h2>使用方法</h2>
<ol>
  <li>选择<strong>背景颜色</strong>并设置<strong>背景透明度</strong>（通常 10–25%）。</li>
  <li>增大<strong>模糊</strong>（推荐 10–20px）以获得磨砂玻璃效果。</li>
  <li>调整<strong>饱和度</strong>控制玻璃后方的色彩鲜艳程度。</li>
  <li>设置<strong>边框颜色</strong>、<strong>边框透明度</strong>和<strong>边框宽度</strong>——细的半透明边框能清晰定义玻璃边缘。</li>
  <li>用<strong>圆角</strong>圆润卡片边角（12–20px 是常用值）。</li>
  <li>添加柔和<strong>阴影</strong>让卡片从背景中浮起。</li>
  <li>切换背景预设（日落、海洋、森林、深夜、极光）测试不同场景下的效果。</li>
  <li>点击<strong>复制</strong>将生成的 CSS 复制到剪贴板。</li>
</ol>

<h2>CSS 输出参考</h2>

```css
.glass {
  background: rgba(255, 255, 255, 0.15);
  backdrop-filter: blur(12px) saturate(180%);
  -webkit-backdrop-filter: blur(12px) saturate(180%);
  border: 1px solid rgba(255, 255, 255, 0.30);
  border-radius: 16px;
  box-shadow: 0 4px 24px rgba(0, 0, 0, 0.20);
}
```
