  - question: "What does backdrop-filter do?"
    answer: "The CSS backdrop-filter property applies graphical effects (blur, saturation, contrast, etc.) to the area behind an element, not the element itself. For glassmorphism, blur() and saturate() are the two key functions: blur creates the frosted-glass look, and saturate keeps colors vivid beneath the translucent surface."
  - question: "Why is -webkit-backdrop-filter needed?"

<h2>How to Use</h2>
<ol>
  <li>Pick a <strong>Background Color</strong> and set <strong>Background Opacity</strong> (10–25% is typical).</li>
  <li>Increase <strong>Blur</strong> (10–20px recommended) to get the frosted-glass effect.</li>
  <li>Adjust <strong>Saturation</strong> to control color vibrancy behind the glass.</li>
  <li>Set a <strong>Border Color</strong>, <strong>Border Opacity</strong>, and <strong>Border Width</strong> — a thin semi-transparent border defines the glass edge.</li>
  <li>Round the corners with <strong>Border Radius</strong> (12–20px is a common sweet spot).</li>
  <li>Add a soft <strong>Shadow</strong> to lift the card off the background.</li>
  <li>Switch between background presets (Sunset, Ocean, Forest, Night, Aurora) to test your effect.</li>
  <li>Click <strong>Copy</strong> to copy the generated CSS to your clipboard.</li>
</ol>

<h2>CSS Output Reference</h2>

```css
.glass {
  background: rgba(255, 255, 255, 0.15);
  backdrop-filter: blur(12px) saturate(180%);
  -webkit-backdrop-filter: blur(12px) saturate(180%);
  border: 1px solid rgba(255, 255, 255, 0.30);
  border-radius: 16px;
  box-shadow: 0 4px 24px rgba(0, 0, 0, 0.20);
}
```

<h2>Common Glassmorphism Presets</h2>
<ul>
  <li><strong>Light glass (white):</strong> background rgba(255,255,255,0.15), blur 12px, border rgba(255,255,255,0.3)</li>
  <li><strong>Dark glass:</strong> background rgba(0,0,0,0.25), blur 16px, border rgba(255,255,255,0.1)</li>
  <li><strong>Frosted strong:</strong> background rgba(255,255,255,0.25), blur 20px, saturation 200%</li>
  <li><strong>Minimal:</strong> background rgba(255,255,255,0.08), blur 8px, no border</li>
</ul>

<h2>Browser Support Notes</h2>
<p>backdrop-filter is supported in all major browsers (Chrome 76+, Firefox 103+, Safari 9+, Edge 79+). Always include both <code>backdrop-filter</code> and <code>-webkit-backdrop-filter</code>. For unsupported browsers, consider a fallback with a semi-opaque solid background.</p>
