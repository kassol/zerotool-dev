<h2>使い方</h2>
<ol>
  <li><strong>変数プレフィックス</strong>を設定します（デフォルト <code>--</code>、<code>--brand-</code> などに変更可）。</li>
  <li>各グループの事前入力済みトークンを編集するか、<strong>+ トークン追加</strong>で新規追加します。</li>
  <li>カラートークンはカラーピッカーか16進数値の直接入力が使えます。</li>
  <li><strong>生成された CSS</strong> ブロックはリアルタイムで更新されます。</li>
  <li><strong>コピー</strong>をクリックして <code>:root &#123; ... &#125;</code> ブロックをクリップボードにコピーします。</li>
</ol>

<h2>デザイントークンとは？</h2>
<p>
  デザイントークンはデザイン上の意思決定（カラーパレット・タイプスケール・スペーシングリズム・角丸）の
  唯一の信頼できる情報源です。<code>:root</code> ブロック内の CSS カスタムプロパティとして管理することで、
  スタイルシート全体からアクセスが可能になります。一箇所の変更がそのトークンを使用するすべてのコンポーネントに
  自動的に反映されます。
</p>

<h2>トークングループの説明</h2>
<ul>
  <li><strong>カラー</strong> — ブランドパレット・背景色・テキスト色・状態色、カラーピッカー付き。</li>
  <li><strong>タイポグラフィ</strong> — フォントファミリースタックとベース/スケールフォントサイズ。</li>
  <li><strong>スペーシング</strong> — 一貫したスペーシングスケール（4px、8px、16px など）。</li>
  <li><strong>角丸</strong> — 控えめなものからピル形状まで、角丸半径の値。</li>
</ul>
