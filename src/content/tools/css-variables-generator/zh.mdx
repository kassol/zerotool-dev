<h2>使用方法</h2>
<ol>
  <li>设置<strong>变量前缀</strong>（默认 <code>--</code>，可改为 <code>--brand-</code> 等）。</li>
  <li>编辑各分组中预填的 Token，或点击<strong>+ 添加 Token</strong>新增。</li>
  <li>颜色 Token 可使用颜色选择器，也可直接输入十六进制值。</li>
  <li><strong>生成的 CSS</strong> 代码块实时更新。</li>
  <li>点击<strong>复制</strong>将 <code>:root &#123; ... &#125;</code> 代码块复制到剪贴板。</li>
</ol>

<h2>什么是设计 Token？</h2>
<p>
  设计 Token 是设计决策的唯一真实来源——颜色调色板、字号比例、间距节奏和圆角大小。
  将它们存储为 <code>:root</code> 块中的 CSS 自定义属性，可使整个样式表都能访问这些值。
  修改一个 Token，所有使用它的组件都会同步更新。
</p>

<h2>Token 分组说明</h2>
<ul>
  <li><strong>颜色</strong> — 品牌调色板、背景色、文字颜色及状态色，带颜色选择器。</li>
  <li><strong>字体排版</strong> — 字体族栈及基础/比例字号。</li>
  <li><strong>间距</strong> — 统一的间距比例（4px、8px、16px 等）。</li>
  <li><strong>圆角</strong> — 从微妙到胶囊形状的圆角半径值。</li>
</ul>
