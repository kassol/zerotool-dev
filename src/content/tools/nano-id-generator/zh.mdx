<h2>使用方法</h2>
<ol>
  <li>设置<strong>批量数量</strong>（1–100），一次生成多个 ID。</li>
  <li>设置<strong>长度</strong>（1–128 个字符，默认 21）。</li>
  <li>选择<strong>字符集</strong>预设，或选择<em>自定义</em>输入自己的字符。</li>
  <li>点击<strong>生成</strong>。</li>
  <li>点击任意行的<strong>复制</strong>复制单个 ID，或点击<strong>复制全部</strong>以换行分隔格式复制全部 ID。</li>
</ol>

<h2>Nano ID vs UUID 对比</h2>

| 特性 | Nano ID（默认）| UUID v4 |
|---|---|---|
| 长度 | 21 字符 | 36 字符（含连字符）|
| 随机性 | 约 126 位 | 122 位 |
| URL 安全 | ✓ 是 | ✓ 是（去除连字符后）|
| 自定义字符集 | ✓ 是 | ✗ 否 |
| 可排序 | ✗ 否 | ✗ 否 |
| 内嵌时间戳 | ✗ 否 | ✗ 否 |
