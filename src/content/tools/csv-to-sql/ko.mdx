<h2>사용 방법</h2>
<ol>
  <li>CSV 데이터를 <strong>CSV 입력</strong> 텍스트 영역에 붙여넣거나 <strong>CSV 업로드</strong>를 클릭하여 파일을 불러옵니다.</li>
  <li><strong>SQL 방언</strong>(MySQL, PostgreSQL, SQLite)을 선택합니다.</li>
  <li><strong>테이블명</strong>을 입력합니다. 기본값은 <code>my_table</code>입니다.</li>
  <li><strong>배치 INSERT</strong>(하나의 문에 여러 행) 또는 <strong>개별 INSERT</strong>(행마다 하나의 문)를 선택합니다.</li>
  <li>SQL 출력이 즉시 업데이트됩니다. <strong>복사</strong>를 클릭하여 클립보드에 복사합니다.</li>
</ol>

  <thead>
    <tr>

    <tr>
      <td>SQLite</td>
      <td>큰따옴표</td>
      <td><code>INSERT INTO "users" ("name") VALUES ('Alice');</code></td>
    </tr>
  </tbody>
</table>

<h2>NULL 값 처리</h2>
<ul>
  <li>CSV의 빈 필드는 SQL 출력에서 따옴표 없는 <code>NULL</code>이 됩니다.</li>
  <li>공백만 있는 필드는 문자열로 처리되며 NULL이 되지 않습니다.</li>
  <li>순수한 숫자(정수 및 소수)는 따옴표 없이 출력되어 숫자 타입으로 저장됩니다.</li>
  <li>다른 모든 값은 작은따옴표로 감싸지며, 내부의 작은따옴표는 <code>''</code>로 이스케이프됩니다.</li>
</ul>
