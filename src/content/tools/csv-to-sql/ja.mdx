<h2>使い方</h2>
<ol>
  <li>CSVデータを<strong>CSV入力</strong>テキストエリアに貼り付けるか、<strong>CSVアップロード</strong>をクリックしてファイルを読み込みます。</li>
  <li><strong>SQLダイアレクト</strong>（MySQL、PostgreSQL、SQLite）を選択します。</li>
  <li><strong>テーブル名</strong>を入力します。デフォルトは <code>my_table</code> です。</li>
  <li><strong>バッチINSERT</strong>（1文に複数行）または<strong>個別INSERT</strong>（行ごとに1文）を選択します。</li>
  <li>SQL出力は即座に更新されます。<strong>コピー</strong>をクリックしてクリップボードにコピーします。</li>
</ol>

<h2>SQLダイアレクトの違い</h2>

<table>
  <thead>
    <tr>
      <th>ダイアレクト</th>
      <th>識別子の引用</th>
      <th>例</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>MySQL</td>
      <td>バックティック</td>
      <td><code>INSERT INTO `users` (`name`) VALUES ('Alice');</code></td>
    </tr>
    <tr>
      <td>PostgreSQL</td>
      <td>ダブルクォート</td>
      <td><code>INSERT INTO "users" ("name") VALUES ('Alice');</code></td>
    </tr>
    <tr>
      <td>SQLite</td>
      <td>ダブルクォート</td>
      <td><code>INSERT INTO "users" ("name") VALUES ('Alice');</code></td>
    </tr>
  </tbody>
</table>

<h2>NULL値の処理</h2>
<ul>
  <li>CSVの空フィールドはSQL出力で引用符なしの <code>NULL</code> になります。</li>
  <li>空白のみのフィールドは文字列として扱われ、NULLにはなりません。</li>
  <li>純粋な数値（整数・小数）は引用符なしで出力され、数値型として保存されます。</li>
  <li>その他の値はシングルクォートで囲まれ、内部のシングルクォートは <code>''</code> にエスケープされます。</li>
</ul>
