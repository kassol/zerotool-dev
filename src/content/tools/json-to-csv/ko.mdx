<h2>사용 방법</h2>
<ol>
  <li>왼쪽 패널에 JSON 객체 배열을 붙여넣으면 자동으로 변환됩니다.</li>
  <li><strong>구분자</strong>를 선택: 쉼표(기본), 세미콜론, 또는 탭.</li>
  <li><strong>중첩 객체 펼치기</strong>를 켜면 중첩 키가 점 경로 열(예: <code>address.city</code>)로 펼쳐집니다.</li>
  <li><strong>헤더 포함</strong>으로 헤더 행 포함 여부를 전환합니다.</li>
  <li><strong>복사</strong>로 클립보드에 복사하거나 <strong>.csv 다운로드</strong>로 파일을 저장합니다.</li>
</ol>

<h2>예제</h2>

<h3>기본 변환</h3>
<p>입력 JSON:</p>
<pre><code>{`[{"id":1,"name":"Alice"},{"id":2,"name":"Bob"}]`}</code></pre>
<p>출력 CSV:</p>
<pre><code>{`id,name\n1,Alice\n2,Bob`}</code></pre>

<h3>중첩 객체 펼치기</h3>
<p>입력 JSON:</p>
<pre><code>{`[{"user":{"name":"Alice","age":30}}]`}</code></pre>
<p>출력 CSV (펼치기 켬):</p>
<pre><code>{`user.name,user.age\nAlice,30`}</code></pre>

<h2>RFC 4180 준수</h2>
<p>
  이 도구는 RFC 4180 CSV 표준을 따릅니다. 구분자, 큰따옴표, 줄바꿈을 포함한 필드는 큰따옴표로 감싸지며, 필드 내 큰따옴표는 <code>""</code>로 이스케이프됩니다.
</p>
