<h2>使い方</h2>
<ol>
  <li>左パネルにJSONを貼り付けるか入力します。リアルタイムで検証されます。</li>
  <li>任意で<strong>ルート構造体名</strong>を設定します（デフォルト：<code>RootObject</code>）。</li>
  <li><strong>Go Struct を生成</strong>をクリックするか、リアルタイム変換を待ちます。</li>
  <li><strong>コピー</strong>をクリックして出力をクリップボードにコピーします。</li>
</ol>

<h2>型マッピング</h2>
<ul>
  <li><code>null</code> → <code>interface{"{}"}</code></li>
  <li>文字列 → <code>string</code></li>
  <li>整数 → <code>int</code></li>
  <li>浮動小数点数 → <code>float64</code></li>
  <li>真偽値 → <code>bool</code></li>
  <li>配列 → <code>[]T</code></li>
  <li>nullable/省略可能 → ポインタ（<code>*T</code>）</li>
  <li>ネストオブジェクト → 独立した名前付き <code>struct</code></li>
</ul>
