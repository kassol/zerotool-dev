<h2>How to Use</h2>
<ol>
  <li>Paste or type your JSON into the left panel. The tool validates it in real time.</li>
  <li>Optionally set a <strong>Root struct name</strong> (default: <code>RootObject</code>).</li>
  <li>Click <strong>Generate Go Struct</strong> or wait for real-time conversion.</li>
  <li>Click <strong>Copy</strong> to copy the output to your clipboard.</li>
</ol>

<h2>Type Mapping</h2>
<ul>
  <li><code>null</code> → <code>interface{"{}"}</code></li>
  <li>string → <code>string</code></li>
  <li>integer number → <code>int</code></li>
  <li>floating-point number → <code>float64</code></li>
  <li>boolean → <code>bool</code></li>
  <li>array → <code>[]T</code></li>
  <li>nullable/optional → pointer (<code>*T</code>)</li>
  <li>nested object → separate named <code>struct</code></li>
</ul>

<h2>Example</h2>
<p>Given this JSON:</p>
<pre><code>{`{"name": "Alice", "age": 30, "address": {"city": "London"}}`}</code></pre>
<p>The tool generates:</p>
<pre><code>{`type Address struct {\n\tCity string \`json:"city"\`\n}\n\ntype RootObject struct {\n\tName    string  \`json:"name"\`\n\tAge     int     \`json:"age"\`\n\tAddress Address \`json:"address"\`\n}`}</code></pre>
