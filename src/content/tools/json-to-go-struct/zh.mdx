<h2>使用方法</h2>
<ol>
  <li>将 JSON 粘贴或输入到左侧面板，工具会实时验证。</li>
  <li>可选：设置<strong>根结构体名称</strong>（默认：<code>RootObject</code>）。</li>
  <li>点击<strong>生成 Go Struct</strong> 或等待实时转换。</li>
  <li>点击<strong>复制</strong>将输出复制到剪贴板。</li>
</ol>

<h2>类型映射</h2>
<ul>
  <li><code>null</code> → <code>interface{"{}"}</code></li>
  <li>字符串 → <code>string</code></li>
  <li>整数 → <code>int</code></li>
  <li>浮点数 → <code>float64</code></li>
  <li>布尔值 → <code>bool</code></li>
  <li>数组 → <code>[]T</code></li>
  <li>可空/可选 → 指针（<code>*T</code>）</li>
  <li>嵌套对象 → 独立的命名 <code>struct</code></li>
</ul>
