<h2>How to Use</h2>
<ol>
  <li>Adjust the <strong>Length</strong> slider (4–128 characters).</li>
  <li>Select which character sets to include.</li>
  <li>A password is generated automatically — click <strong>Generate</strong> for a new one.</li>
  <li>The strength meter shows entropy in bits. Aim for <strong>80+ bits</strong> for strong security.</li>
  <li>Click the copy icon to copy to clipboard, or use <strong>Generate 10</strong> for a batch.</li>
</ol>

<h2>Password Security Tips</h2>
<ul>
  <li>Use a unique password for every account. A password manager makes this practical.</li>
  <li>Longer passwords are exponentially stronger — prefer length over complexity alone.</li>
  <li>Enable two-factor authentication wherever possible, regardless of password strength.</li>
  <li>Never use personal information (birthdays, names) in passwords — even partially.</li>
</ul>
