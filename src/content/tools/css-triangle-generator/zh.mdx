<h2>使用方法</h2>
<ol>
  <li>点击 3×3 方向网格中的按钮，选择三角形的朝向。</li>
  <li>设置<strong>宽度</strong>和<strong>高度</strong>（像素）以控制三角形尺寸。</li>
  <li>通过颜色选择器或直接输入十六进制颜色值来选色。</li>
  <li>实时预览区域会即时更新，并提供对比背景，方便在明暗模式下都能清晰看到三角形。</li>
  <li>点击 CSS 区块旁的<strong>复制</strong>按钮即可复制 <code>.triangle &#123; ... &#125;</code> 规则，或单独复制 HTML 用法示例。</li>
</ol>

<h2>理解生成的 CSS</h2>
<p>
  生成的规则始终以 <code>width: 0</code> 和 <code>height: 0</code> 开始。
  对于四个基本方向（上、右、下、左），两个透明 border 定义三角形的基础宽度，一个实色 border 定义高度。
  对于对角方向（如左上），一个实色 border 和一个透明 border 从角落裁出一个直角三角形。
  将输出粘贴到样式表中，并将 <code>.triangle</code> 改为你自己的类名。
</p>
