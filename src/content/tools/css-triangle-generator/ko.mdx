<h2>사용 방법</h2>
<ol>
  <li>3×3 방향 그리드에서 버튼을 클릭하여 삼각형이 가리킬 방향을 선택합니다.</li>
  <li><strong>너비</strong>와 <strong>높이</strong>를 픽셀로 설정하여 삼각형 크기를 조정합니다.</li>
  <li>색상 피커를 사용하거나 16진수 색상 값을 직접 입력합니다.</li>
  <li>라이브 미리보기는 대비 배경과 함께 즉시 업데이트되어 라이트/다크 모드 모두에서 삼각형을 명확하게 확인할 수 있습니다.</li>
  <li>CSS 블록 옆의 <strong>복사</strong>를 클릭하여 <code>.triangle &#123; ... &#125;</code> 규칙을 복사하거나, HTML 사용법 스니펫을 별도로 복사합니다.</li>
</ol>

<h2>생성된 CSS 이해하기</h2>
<p>
  생성된 규칙은 항상 <code>width: 0</code>과 <code>height: 0</code>으로 시작합니다.
  기본 방향(위, 오른쪽, 아래, 왼쪽)에서는 두 개의 투명 border가 삼각형의 밑변 너비를 정의하고,
  하나의 실색 border가 높이를 정의합니다.
  모서리 방향(예: 왼쪽 위)에서는 하나의 실색 border와 하나의 투명 border가 모서리에서 직각삼각형을 형성합니다.
  출력을 스타일시트에 붙여넣고 <code>.triangle</code>을 자신의 클래스 이름으로 변경하세요.
</p>

<h2>주요 사용 사례</h2>
<p>
  CSS 삼각형은 툴팁 화살표, 드롭다운 캐럿 지시자, 장식적인 섹션 구분선,
  브레드크럼 구분자, 말풍선 꼬리 등에 사용됩니다.
  이미지 없이 순수 CSS로 구현되어 즉시 로드되며 CSS 변수로 손쉽게 테마를 적용할 수 있습니다.
</p>
