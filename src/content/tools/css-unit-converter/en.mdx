<h2>When to Use Each Unit</h2>
<ul>
  <li><strong>px</strong> — absolute size, consistent across devices. Good for borders, shadows, and fine details.</li>
  <li><strong>rem</strong> — scales with the user's browser font size setting. Preferred for font sizes and spacing in accessible designs.</li>
  <li><strong>em</strong> — relative to the parent element's font size. Useful for component-scoped sizing.</li>
  <li><strong>vw</strong> — relative to viewport width. Good for fluid typography and full-bleed layouts.</li>
</ul>
