<h2>사용 방법</h2>
<ol>
  <li><strong>Value</strong> 필드에 숫자를 입력합니다.</li>
  <li>원본 단위(px, rem, em, 또는 vw)를 선택합니다.</li>
  <li>네 가지 단위의 변환 결과가 즉시 업데이트됩니다.</li>
  <li>필요하면 <strong>루트 폰트 크기</strong>나 <strong>뷰포트 너비</strong>를 프로젝트 설정에 맞게 조정합니다.</li>
  <li>결과 옆의 <strong>Copy</strong> 버튼을 클릭해 클립보드에 복사합니다.</li>
</ol>

<h2>자주 쓰는 변환 예시</h2>
<p>기본 설정(루트 폰트 크기 16px, 뷰포트 1920px) 기준:</p>
<ul>
  <li><code>16px</code> = <code>1rem</code> = <code>1em</code> = <code>0.833vw</code></li>
  <li><code>1rem</code> = <code>16px</code></li>
  <li><code>100vw</code> = <code>1920px</code> = <code>120rem</code></li>
</ul>

<h2>각 단위 사용 시나리오</h2>
<ul>
  <li><strong>px</strong> — 절대 크기. 테두리, 그림자 등 세부 요소에 적합합니다.</li>
  <li><strong>rem</strong> — 사용자 브라우저 폰트 크기 설정에 따라 스케일됩니다. 접근성 중심 디자인에서 폰트 크기와 간격에 권장.</li>
  <li><strong>em</strong> — 부모 요소의 폰트 크기를 기준으로 합니다. 컴포넌트 내부 상대 사이징에 유용합니다.</li>
  <li><strong>vw</strong> — 뷰포트 너비를 기준으로 합니다. 유동적 타이포그래피와 전체 너비 레이아웃에 적합합니다.</li>
</ul>
