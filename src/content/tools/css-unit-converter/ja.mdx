<h2>使い方</h2>
<ol>
  <li><strong>Value</strong> フィールドに数値を入力します。</li>
  <li>元の単位（px・rem・em・vw）を選択します。</li>
  <li>4種類の単位の変換結果が即座に更新されます。</li>
  <li>必要に応じて<strong>ルートフォントサイズ</strong>や<strong>ビューポート幅</strong>をプロジェクトの設定に合わせて調整します。</li>
  <li>各結果の横にある <strong>Copy</strong> ボタンでクリップボードにコピーします。</li>
</ol>

<h2>よく使う変換の例</h2>
<p>デフォルト設定（ルートフォントサイズ16px、ビューポート1920px）の場合：</p>
<ul>
  <li><code>16px</code> = <code>1rem</code> = <code>1em</code> = <code>0.833vw</code></li>
  <li><code>1rem</code> = <code>16px</code></li>
  <li><code>100vw</code> = <code>1920px</code> = <code>120rem</code></li>
</ul>

<h2>各単位の使いどころ</h2>
<ul>
  <li><strong>px</strong> — 絶対サイズ。ボーダーや細部のデザインに適します。</li>
  <li><strong>rem</strong> — ブラウザのフォントサイズ設定に連動。アクセシブルなデザインでフォントサイズや余白に推奨。</li>
  <li><strong>em</strong> — 親要素のフォントサイズを基準にします。コンポーネント内の相対サイジングに便利。</li>
  <li><strong>vw</strong> — ビューポート幅を基準にします。流動的なタイポグラフィや全幅レイアウトに適します。</li>
</ul>
