<h2>使用方法</h2>
<ol>
  <li>在 <strong>Value</strong> 输入框中输入数值。</li>
  <li>选择源单位（px、rem、em 或 vw）。</li>
  <li>下方四种单位的转换结果即时更新。</li>
  <li>可选：调整<strong>根字体大小</strong>或<strong>视口宽度</strong>以匹配项目设置。</li>
  <li>点击任意结果旁的 <strong>Copy</strong> 按钮复制到剪贴板。</li>
</ol>

<h2>常用转换参考</h2>
<p>默认设置（根字体大小 16px，视口 1920px）：</p>
<ul>
  <li><code>16px</code> = <code>1rem</code> = <code>1em</code> = <code>0.833vw</code></li>
  <li><code>1rem</code> = <code>16px</code></li>
  <li><code>100vw</code> = <code>1920px</code> = <code>120rem</code></li>
</ul>

<h2>各单位使用场景</h2>
<ul>
  <li><strong>px</strong> — 绝对尺寸，跨设备一致。适合边框、阴影等细节。</li>
  <li><strong>rem</strong> — 随用户浏览器字体大小缩放。无障碍设计中字体和间距的首选。</li>
  <li><strong>em</strong> — 相对于父元素字体大小。适合组件内部的相对尺寸。</li>
  <li><strong>vw</strong> — 相对于视口宽度。适合流体排版和全宽布局。</li>
</ul>
