<h2>사용 방법</h2>
<ol>
  <li>변환할 CSS 속성을 입력창에 붙여넣습니다(예: <code>color: red; font-size: 16px;</code>).</li>
  <li>대응하는 Tailwind 클래스가 즉시 표시됩니다.</li>
  <li>결과를 <strong>복사</strong>하여 HTML에 붙여넣습니다.</li>
</ol>

<h2>Tailwind CSS에 대하여</h2>
<p>Tailwind CSS는 유틸리티 퍼스트 CSS 프레임워크로, 미리 정의된 클래스를 HTML에 직접 적용하여 스타일링합니다. 기존 CSS를 Tailwind 프로젝트로 마이그레이션할 때나 Tailwind 클래스를 빠르게 찾고 싶을 때 이 도구가 유용합니다.</p>
