faqItems:
  - question: "What is a cron expression?"
    answer: "A cron expression is a string with 5 fields (minute, hour, day-of-month, month, day-of-week) that defines a recurring schedule. It is used by cron, crontab, Kubernetes, GitHub Actions, and many other systems."
  - question: "What does * mean in a cron expression?"
    answer: "An asterisk (*) means 'every valid value'. For example, * in the minute field means 'every minute'. Combined with other fields it creates intervals."
  - question: "What is the difference between '0 0 * * 0' and '0 0 * * 7'?"
    answer: "Both mean Sunday midnight. The day-of-week field accepts 0 and 7 as aliases for Sunday."
  - question: "Does this support seconds or @reboot?"
    answer: "This tool supports the standard 5-field cron format (minute hour day month weekday). Vixie-style 6-field cron with seconds and @reboot are not supported."
