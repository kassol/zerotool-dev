<h2>How to Use</h2>
<ol>
  <li>Paste your original text into the <strong>Original</strong> panel.</li>
  <li>Paste your modified text into the <strong>Modified</strong> panel.</li>
  <li>Click <strong>Compare</strong> to see the diff.</li>
  <li>Lines highlighted in <span style="color:#c0392b">red</span> were removed; lines in <span style="color:#057a55">green</span> were added.</li>
  <li>Use <strong>Swap</strong> to reverse the comparison direction.</li>
</ol>

<h2>About This Tool</h2>
<p>
  This diff checker performs line-by-line comparison using the Longest Common Subsequence (LCS) algorithm —
  the same principle used by <code>git diff</code> and Unix <code>diff</code>. It's ideal for comparing
  code, configuration files, log output, or any text where line-level changes matter.
</p>
