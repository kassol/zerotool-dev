<h2>사용 방법</h2>
<ol>
  <li>TypeScript <code>interface</code> 또는 <code>type</code> 선언을 왼쪽 패널에 붙여넣습니다.</li>
  <li>입력이 멈추면 300ms 후 자동으로 유효성 검사 및 변환이 실행됩니다.</li>
  <li><strong>Zod 스키마 생성</strong> 버튼을 클릭해 강제로 변환할 수 있습니다.</li>
  <li><strong>복사</strong> 버튼을 클릭해 Zod 출력을 클립보드에 복사합니다.</li>
  <li>결과를 프로젝트에 붙여넣으세요. 출력에는 이미 <code>import</code> 문과 <code>export type</code> 타입 추론 줄이 포함되어 있습니다.</li>
</ol>

<h2>지원하는 TypeScript 기능</h2>
<ul>
  <li><strong>기본 타입</strong>: <code>string</code>, <code>number</code>, <code>boolean</code>, <code>null</code>, <code>undefined</code>, <code>any</code>, <code>unknown</code>, <code>void</code>, <code>never</code>, <code>bigint</code>, <code>symbol</code></li>
  <li><strong>선택적 속성</strong>: <code>prop?: type</code> → <code>.optional()</code></li>
  <li><strong>유니언 타입</strong>: <code>A | B</code> → <code>z.union([A, B])</code></li>
  <li><strong>문자열 열거형</strong>: <code>"a" | "b" | "c"</code> → <code>z.enum(["a", "b", "c"])</code></li>
  <li><strong>배열</strong>: <code>T[]</code> / <code>Array&lt;T&gt;</code> → <code>z.array(T)</code></li>
  <li><strong>중첩 객체</strong>: 인라인 객체 타입 → <code>z.object(&#123;&#125;)</code></li>
  <li><strong>상호 참조</strong>: 선언된 타입 간의 참조 → Schema 상수</li>
  <li><strong>readonly</strong>: <code>readonly</code> 수식어를 허용하며, 변환 시 무시(Zod 스키마는 기본적으로 불변)</li>
  <li><strong>유틸리티 타입</strong>: <code>Partial&lt;T&gt;</code>, <code>Record&lt;K, V&gt;</code>, <code>Promise&lt;T&gt;</code></li>
  <li><strong>교차 타입</strong>: <code>A &amp; B</code> → <code>z.intersection(A, B)</code></li>
  <li><strong>튜플</strong>: <code>[A, B]</code> → <code>z.tuple([A, B])</code></li>
  <li><strong>리터럴 타입</strong>: <code>"value"</code>, <code>42</code>, <code>true</code> → <code>z.literal()</code></li>
</ul>

<h2>예시</h2>
<p>다음 TypeScript를 입력하면:</p>
<pre><code>{`interface Address {
  street: string;
  city: string;
  zipCode?: string;
}

export interface User {
  id: number;
  name: string;
  role: "admin" | "user" | "guest";
  age?: number;
  isActive: boolean;
  tags: string[];
  address: Address;
}

export type Status = "active" | "inactive" | "pending";`}</code></pre>
<p>다음과 같이 생성됩니다:</p>
<pre><code>{`import { z } from "zod";

export const AddressSchema = z.object({
  street: z.string(),
  city: z.string(),
  zipCode: z.string().optional(),
});
export type Address = z.infer<typeof AddressSchema>;

export const UserSchema = z.object({
  id: z.number(),
  name: z.string(),
  role: z.enum(["admin", "user", "guest"]),
  age: z.number().optional(),
  isActive: z.boolean(),
  tags: z.array(z.string()),
  address: AddressSchema,
});
export type User = z.infer<typeof UserSchema>;

export const StatusSchema = z.enum(["active", "inactive", "pending"]);
export type Status = z.infer<typeof StatusSchema>;`}</code></pre>
