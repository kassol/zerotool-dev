<h2>使用方法</h2>
<ol>
  <li>在左侧编辑面板中输入或粘贴 Markdown。</li>
  <li>右侧会实时显示渲染后的 HTML 预览。</li>
  <li>点击 <strong>Copy HTML</strong> 可将渲染后的 HTML 复制到剪贴板。</li>
  <li>点击 <strong>Clear</strong> 可清空内容重新开始。</li>
</ol>

<h2>支持的语法</h2>
<ul>
  <li>标题：<code># H1</code> 至 <code>###### H6</code></li>
  <li>粗体：<code>**文本**</code> 或 <code>__文本__</code></li>
  <li>斜体：<code>*文本*</code> 或 <code>_文本_</code></li>
  <li>删除线：<code>~~文本~~</code></li>
  <li>行内代码：<code>`code`</code></li>
  <li>代码块：三个反引号围栏，可指定语言</li>
  <li>链接：<code>[文本](url)</code></li>
  <li>图片：<code>{"![替代文本](图片地址)"}</code></li>
  <li>无序列表：<code>- 项目</code></li>
  <li>有序列表：<code>1. 项目</code></li>
  <li>引用块：<code>&gt; 引文</code></li>
  <li>表格（GFM）</li>
  <li>任务列表：<code>- [x] 已完成</code> / <code>- [ ] 待办</code></li>
  <li>水平线：<code>---</code></li>
</ul>
