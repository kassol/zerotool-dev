<h2>How to Use</h2>
<ol>
  <li>Type or paste Markdown in the left editor pane.</li>
  <li>The rendered HTML preview updates in real time on the right.</li>
  <li>Click <strong>Copy HTML</strong> to copy the rendered HTML to clipboard.</li>
  <li>Click <strong>Clear</strong> to start fresh.</li>
</ol>

<h2>Supported Syntax</h2>
<ul>
  <li>Headings: <code># H1</code> through <code>###### H6</code></li>
  <li>Bold: <code>**text**</code> or <code>__text__</code></li>
  <li>Italic: <code>*text*</code> or <code>_text_</code></li>
  <li>Strikethrough: <code>~~text~~</code></li>
  <li>Inline code: <code>`code`</code></li>
  <li>Code block: triple backtick fence with optional language</li>
  <li>Links: <code>[label](url)</code></li>
  <li>Images: <code>{"![alt](src)"}</code></li>
  <li>Unordered lists: <code>- item</code></li>
  <li>Ordered lists: <code>1. item</code></li>
  <li>Blockquotes: <code>&gt; quote</code></li>
  <li>Tables (GFM)</li>
  <li>Task lists: <code>- [x] done</code> / <code>- [ ] todo</code></li>
  <li>Horizontal rule: <code>---</code></li>
</ul>
