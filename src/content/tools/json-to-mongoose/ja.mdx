<h2>使い方</h2>
<ol>
  <li>左パネルにJSONを貼り付けるか入力します。リアルタイムで検証されます。</li>
  <li><strong>モデル名</strong>を設定します（デフォルト：<code>User</code>）。スキーマ変数とMongooseモデルの名前になります。</li>
  <li><strong>JavaScript</strong>または<strong>TypeScript</strong>出力モードを選択します。</li>
  <li><strong>timestamps</strong>をオン/オフで切り替えます。</li>
  <li><strong>スキーマを生成</strong>をクリックするか、リアルタイム変換を待ちます。</li>
  <li><strong>コピー</strong>をクリックして出力をクリップボードにコピーします。</li>
</ol>

<h2>型マッピング</h2>
<ul>
  <li><code>null</code> → <code>mongoose.Schema.Types.Mixed</code></li>
  <li>文字列 → <code>String</code></li>
  <li>数値 → <code>Number</code></li>
  <li>真偽値 → <code>Boolean</code></li>
  <li>プリミティブ配列 → <code>[String]</code> / <code>[Number]</code> / <code>[Boolean]</code></li>
  <li>オブジェクト配列 → <code>[subSchema]</code>（独立サブスキーマを生成）</li>
  <li>ネストオブジェクト → 独立スキーマ変数をインライン参照</li>
</ul>

<h2>例</h2>
<p>モデル名 <code>User</code>、以下のJSONを入力した場合：</p>
<pre><code>{`{"username": "alice", "age": 28, "tags": ["admin"], "address": {"city": "London"}}`}</code></pre>
<p>JavaScript出力（timestampsオン）：</p>
<pre><code>{`const mongoose = require('mongoose');\nconst { Schema } = mongoose;\n\nconst addressSchema = new Schema({\n  city: { type: String },\n});\n\nconst userSchema = new Schema({\n  username: { type: String },\n  age: { type: Number },\n  tags: [String],\n  address: addressSchema,\n}, { timestamps: true });\n\nmodule.exports = mongoose.model('User', userSchema);`}</code></pre>
