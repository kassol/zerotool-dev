<h2>How to Use</h2>
<ol>
  <li>Paste or type your JSON into the left panel. The tool validates it in real time.</li>
  <li>Set a <strong>Model name</strong> (default: <code>User</code>). This names the schema variable and the Mongoose model.</li>
  <li>Choose <strong>JavaScript</strong> or <strong>TypeScript</strong> output mode.</li>
  <li>Toggle <strong>timestamps</strong> On or Off.</li>
  <li>Click <strong>Generate Schema</strong> or wait for real-time conversion.</li>
  <li>Click <strong>Copy</strong> to copy the output to your clipboard.</li>
</ol>

<h2>Type Mapping</h2>
<ul>
  <li><code>null</code> → <code>mongoose.Schema.Types.Mixed</code></li>
  <li>string → <code>String</code></li>
  <li>number → <code>Number</code></li>
  <li>boolean → <code>Boolean</code></li>
  <li>array of primitives → <code>[String]</code> / <code>[Number]</code> / <code>[Boolean]</code></li>
  <li>array of objects → <code>[subSchema]</code> (separate schema generated)</li>
  <li>nested object → separate schema variable referenced inline</li>
</ul>

<h2>Example</h2>
<p>Given this JSON with model name <code>User</code>:</p>
<pre><code>{`{"username": "alice", "age": 28, "tags": ["admin"], "address": {"city": "London"}}`}</code></pre>
<p>JavaScript output (timestamps On):</p>
<pre><code>{`const mongoose = require('mongoose');\nconst { Schema } = mongoose;\n\nconst addressSchema = new Schema({\n  city: { type: String },\n});\n\nconst userSchema = new Schema({\n  username: { type: String },\n  age: { type: Number },\n  tags: [String],\n  address: addressSchema,\n}, { timestamps: true });\n\nmodule.exports = mongoose.model('User', userSchema);`}</code></pre>
