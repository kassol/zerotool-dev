<h2>사용 방법</h2>
<ol>
  <li>왼쪽 패널에 JSON을 붙여넣거나 입력합니다. 실시간으로 유효성이 검사됩니다.</li>
  <li><strong>모델 이름</strong>을 설정합니다(기본값: <code>User</code>). 스키마 변수와 Mongoose 모델 이름으로 사용됩니다.</li>
  <li><strong>JavaScript</strong> 또는 <strong>TypeScript</strong> 출력 모드를 선택합니다.</li>
  <li><strong>timestamps</strong>를 켜거나 끕니다.</li>
  <li><strong>스키마 생성</strong>을 클릭하거나 실시간 변환 결과를 기다립니다.</li>
  <li><strong>복사</strong>를 클릭하여 출력을 클립보드에 복사합니다.</li>
</ol>

<h2>타입 매핑</h2>
<ul>
  <li><code>null</code> → <code>mongoose.Schema.Types.Mixed</code></li>
  <li>문자열 → <code>String</code></li>
  <li>숫자 → <code>Number</code></li>
  <li>불리언 → <code>Boolean</code></li>
  <li>기본 타입 배열 → <code>[String]</code> / <code>[Number]</code> / <code>[Boolean]</code></li>
  <li>객체 배열 → <code>[subSchema]</code>(별도 서브스키마 생성)</li>
  <li>중첩 객체 → 별도 스키마 변수를 인라인으로 참조</li>
</ul>

<h2>예시</h2>
<p>모델 이름 <code>User</code>, 다음 JSON 입력 시:</p>
<pre><code>{`{"username": "alice", "age": 28, "tags": ["admin"], "address": {"city": "London"}}`}</code></pre>
<p>JavaScript 출력(timestamps 켜짐):</p>
<pre><code>{`const mongoose = require('mongoose');\nconst { Schema } = mongoose;\n\nconst addressSchema = new Schema({\n  city: { type: String },\n});\n\nconst userSchema = new Schema({\n  username: { type: String },\n  age: { type: Number },\n  tags: [String],\n  address: addressSchema,\n}, { timestamps: true });\n\nmodule.exports = mongoose.model('User', userSchema);`}</code></pre>
