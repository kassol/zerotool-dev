<h2>使用方法</h2>
<ol>
  <li>将 JSON 粘贴或输入到左侧面板，工具会实时验证。</li>
  <li>设置<strong>模型名称</strong>（默认：<code>User</code>），用于命名 schema 变量和 Mongoose model。</li>
  <li>选择 <strong>JavaScript</strong> 或 <strong>TypeScript</strong> 输出模式。</li>
  <li>切换 <strong>timestamps</strong> 开启或关闭。</li>
  <li>点击<strong>生成 Schema</strong>，或等待实时转换结果。</li>
  <li>点击<strong>复制</strong>将输出复制到剪贴板。</li>
</ol>

<h2>类型映射</h2>
<ul>
  <li><code>null</code> → <code>mongoose.Schema.Types.Mixed</code></li>
  <li>字符串 → <code>String</code></li>
  <li>数字 → <code>Number</code></li>
  <li>布尔值 → <code>Boolean</code></li>
  <li>基本类型数组 → <code>[String]</code> / <code>[Number]</code> / <code>[Boolean]</code></li>
  <li>对象数组 → <code>[subSchema]</code>（生成独立子 schema）</li>
  <li>嵌套对象 → 独立 schema 变量，内联引用</li>
</ul>

<h2>示例</h2>
<p>给定如下 JSON，模型名称为 <code>User</code>：</p>
<pre><code>{`{"username": "alice", "age": 28, "tags": ["admin"], "address": {"city": "London"}}`}</code></pre>
<p>JavaScript 输出（timestamps 开启）：</p>
<pre><code>{`const mongoose = require('mongoose');\nconst { Schema } = mongoose;\n\nconst addressSchema = new Schema({\n  city: { type: String },\n});\n\nconst userSchema = new Schema({\n  username: { type: String },\n  age: { type: Number },\n  tags: [String],\n  address: addressSchema,\n}, { timestamps: true });\n\nmodule.exports = mongoose.model('User', userSchema);`}</code></pre>
