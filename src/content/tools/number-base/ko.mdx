<h2>사용 방법</h2>
<ol>
  <li>임의의 입력창에 숫자를 입력합니다(예: 10진수 창에 "255").</li>
  <li>다른 진법(2진수: 11111111, 16진수: ff)으로 즉시 변환됩니다.</li>
</ol>

<h2>진법에 대하여</h2>
<p>컴퓨터는 내부적으로 2진수(0과 1)로 처리합니다. 프로그래밍에서 2진수는 비트 연산이나 플래그 관리, 8진수는 Unix 퍼미션, 16진수는 메모리 주소·색상 코드·바이너리 데이터 표현에 자주 사용됩니다.</p>
<ul>
  <li><strong>2진수(binary)</strong>: 기수 2, 0과 1만 사용</li>
  <li><strong>8진수(octal)</strong>: 기수 8, 0–7 사용</li>
  <li><strong>10진수(decimal)</strong>: 기수 10, 0–9 사용</li>
  <li><strong>16진수(hexadecimal)</strong>: 기수 16, 0–9와 A–F 사용</li>
</ul>
