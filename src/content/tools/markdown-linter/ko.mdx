<h2>사용 방법</h2>
<ol>
  <li>왼쪽 패널에 Markdown을 붙여넣거나 입력합니다.</li>
  <li>버튼을 누르지 않아도 오른쪽에 실시간으로 Lint 결과가 표시됩니다.</li>
  <li>결과를 클릭하면 에디터의 해당 줄로 이동합니다.</li>
  <li>문제를 수정하면 결과가 즉시 업데이트됩니다.</li>
</ol>

<h2>검사 내용</h2>
<p>
  <a href="https://github.com/DavidAnson/markdownlint/blob/main/doc/Rules.md" target="_blank" rel="noopener">markdownlint 전체 규칙</a>을 적용합니다. 주요 내용:
</p>
<ul>
  <li><strong>MD001</strong> — 제목 레벨은 한 번에 1단계씩만 증가</li>
  <li><strong>MD009</strong> — 줄 끝에 공백을 두지 않음</li>
  <li><strong>MD010</strong> — 하드 탭 사용 금지 (공백 사용)</li>
  <li><strong>MD012</strong> — 연속 빈 줄을 여러 개 두지 않음</li>
  <li><strong>MD022</strong> — 제목 앞뒤에 빈 줄 삽입</li>
  <li><strong>MD025</strong> — 문서 내 최상위 제목은 하나만</li>
  <li><strong>MD031</strong> — 코드 펜스 앞뒤에 빈 줄 삽입</li>
  <li><strong>MD034</strong> — 노출된 URL 사용 금지, 링크 문법 사용</li>
  <li><strong>MD041</strong> — 첫 줄은 최상위 제목이어야 함</li>
</ul>
