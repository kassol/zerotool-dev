<h2>사진을 공유하기 전에 EXIF를 제거해야 하는 이유</h2>

<p>대부분의 스마트폰과 카메라는 사진마다 정확한 GPS 좌표, 카메라 시리얼 번호, 촬영 타임스탬프를 조용히 기록합니다. EXIF를 제거하지 않은 채 게시판·블로그·자동 정리 기능이 없는 메신저에 사진을 올리면, 무료 도구만으로도 집·사무실·숙소 위치가 몇 미터 단위로 노출될 수 있습니다.</p>

<p>이 도구는 사진이 어떤 정보를 무심코 누설하는지 빠르게 확인하고, 메타데이터가 제거된 깨끗한 사본을 다운로드할 수 있게 해줍니다. 모든 처리는 지금 열려 있는 브라우저 탭 안에서 완결됩니다.</p>

<h2>사용 방법</h2>
<ol>
  <li>JPEG 사진을 업로드 영역에 드롭하거나 클릭해서 파일을 선택하거나 <code>Ctrl + V</code>로 붙여넣습니다.</li>
  <li>파싱된 EXIF를 확인합니다: 바디·렌즈, 노출 설정, 원본 촬영 시각, GPS 좌표(Google 지도 링크 포함), 파일을 편집한 소프트웨어.</li>
  <li><strong>메타데이터 제거된 JPEG 다운로드</strong>를 클릭해 EXIF / XMP / IPTC 마커가 제거된 사본을 저장합니다. 이미지 데이터는 바이트 단위로 동일하며 픽셀이 재인코딩되지 않습니다.</li>
  <li>원본 메타데이터를 기록으로 남기고 싶다면 <strong>EXIF JSON 복사</strong>로 파싱 결과를 먼저 보관할 수 있습니다.</li>
</ol>

<h2>제거되는 세그먼트</h2>

<p>제거 단계는 JPEG 마커 스트림을 훑으면서 개인정보가 들어가기 쉬운 세그먼트를 떨어냅니다:</p>

<ul>
  <li><strong>APP1</strong> — EXIF(카메라, GPS, 타임스탬프)와 XMP(Adobe 메타데이터, 편집 이력)</li>
  <li><strong>APP13</strong> — Photoshop / IPTC(캡션, 저작권, 연락처)</li>
  <li><strong>APP14</strong> — Adobe 색상 변환 마커</li>
</ul>

<p>브라우저가 이미지를 올바르게 디코딩하는 데 필요한 세그먼트는 그대로 둡니다: SOI 시작 마커, JFIF(APP0), ICC 색상 프로파일(APP2), SOS 스캔 마커, 압축 이미지 데이터, EOI 종료 마커. 색감은 원본과 동일합니다.</p>

<h2>표시되는 EXIF 필드</h2>

<ul>
  <li><strong>카메라</strong>: 제조사, 모델, 렌즈 모델 — 예: Apple iPhone 15 Pro, iPhone back camera 6.86mm f/1.78</li>
  <li><strong>시간</strong>: 원본 촬영 시각, 마지막 수정 시각</li>
  <li><strong>노출</strong>: 조리개(F값), 셔터, ISO, 초점거리</li>
  <li><strong>GPS</strong>: 위도, 경도, 고도, Google 지도 직링크</li>
  <li><strong>소프트웨어</strong>: 파일을 기록한 OS 또는 편집 앱</li>
</ul>
