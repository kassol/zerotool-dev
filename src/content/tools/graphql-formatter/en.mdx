<h2>How to Use</h2>
<ol>
  <li>Paste your GraphQL query, mutation, subscription, or SDL schema into the input area, or click <strong>Load sample</strong> to start from an example.</li>
  <li>Click <strong>Format</strong> to beautify, <strong>Minify</strong> to compress, or <strong>Validate</strong> to check syntax only.</li>
  <li>Switch indent between 2 and 4 spaces — the default 2 matches graphql-js native output.</li>
  <li>Click <strong>Copy</strong> or <strong>Download .graphql</strong> to take the result with you. Press <kbd>Cmd/Ctrl + Enter</kbd> to format from the keyboard.</li>
</ol>

<h2>Worked Examples</h2>

<h3>Before formatting</h3>

```graphql
query GetUser($id:ID!,$withPosts:Boolean!){user(id:$id){id name email posts @include(if:$withPosts){...PostSummary}}} fragment PostSummary on Post{id title publishedAt}
```

<h3>After formatting</h3>

```graphql
query GetUser($id: ID!, $withPosts: Boolean!) {
  user(id: $id) {
    id
    name
    email
    posts @include(if: $withPosts) {
      ...PostSummary
    }
  }
}

fragment PostSummary on Post {
  id
  title
  publishedAt
}
```

<h3>SDL schema</h3>

```graphql
type Post {
  id: ID!
  title: String!
  publishedAt: DateTime
  author: User!
}

type Query {
  post(id: ID!): Post
  posts(authorId: ID, limit: Int = 20): [Post!]!
}
```

<h2>Why Format GraphQL?</h2>
<p>Consistent GraphQL formatting matters for code review, schema diffs, and storing operations in <code>.graphql</code> files. Minified queries shrink wire payloads when you cannot rely on persisted queries; pretty-printed queries are easier to scan in PRs. Validating a query in isolation — before piping it to an endpoint — surfaces a typo in a fragment name in seconds rather than after a round-trip.</p>

<h2>Behind the Scenes</h2>
<p>This tool runs <code>graphql-js</code> (the reference implementation maintained by The GraphQL Foundation) entirely inside your browser. <code>parse()</code> builds an AST from your text and reports the first syntax error with line and column. <code>print()</code> serializes the AST back to canonical GraphQL. <code>stripIgnoredCharacters()</code> removes every token the spec marks as ignored — that is the minified form. Nothing is sent anywhere, and there is no server round-trip.</p>
