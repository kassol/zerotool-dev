<h2>How to Use</h2>
<ol>
  <li>To convert <strong>JSON → XML</strong>: paste your JSON into the left panel and click <strong>JSON → XML</strong>, or type and conversion runs automatically.</li>
  <li>To convert <strong>XML → JSON</strong>: paste your XML into the right panel and click <strong>XML → JSON</strong>.</li>
  <li>Customize the <strong>Root element</strong> name used when wrapping JSON output in XML.</li>
  <li>Toggle <strong>Pretty print</strong> to control whether output is indented or compact.</li>
  <li>Use <strong>Ctrl+Enter</strong> (or <strong>Cmd+Enter</strong> on Mac) as a shortcut from whichever panel you're editing.</li>
</ol>

<h2>Worked Examples</h2>

<h3>Simple object</h3>
<p>Input JSON:</p>

```json
{"name": "Alice", "age": 30, "active": true}
```

<p>Output XML (root element: <code>root</code>):</p>

```xml
<?xml version="1.0" encoding="UTF-8"?>
<root>
  <name>Alice</name>
  <age>30</age>
  <active>true</active>
</root>
```

<h3>Nested objects</h3>
<p>Input JSON:</p>

```json
{"server": {"host": "localhost", "port": 8080}}
```

<p>Output XML:</p>

```xml
<?xml version="1.0" encoding="UTF-8"?>
<root>
  <server>
    <host>localhost</host>
    <port>8080</port>
  </server>
</root>
```

<h3>Arrays</h3>
<p>JSON arrays produce repeated XML elements with the tag name <code>item</code> at the top level, or the parent key name for nested arrays:</p>

```json
{"tags": ["js", "xml", "api"]}
```

```xml
<?xml version="1.0" encoding="UTF-8"?>
<root>
  <tags>js</tags>
  <tags>xml</tags>
  <tags>api</tags>
</root>
```

<h2>JSON vs XML</h2>
<p>
  <strong>JSON</strong> (JavaScript Object Notation) is the standard data format for REST APIs and web services — compact, human-readable, and natively supported in every language.
  <strong>XML</strong> (eXtensible Markup Language) remains widely used in enterprise systems, SOAP APIs, configuration files (Maven, Spring), and document formats (SVG, XHTML, RSS).
  This tool helps you bridge the two formats when integrating systems that speak different data languages.
</p>
