<h2>使い方</h2>
<ol>
  <li><strong>JSON → XML</strong>：左パネルにJSONを貼り付けて <strong>JSON → XML</strong> ボタンをクリック、または入力するとリアルタイムに変換されます。</li>
  <li><strong>XML → JSON</strong>：右パネルにXMLを貼り付けて <strong>XML → JSON</strong> ボタンをクリック。</li>
  <li><strong>ルート要素</strong> 入力欄でXML出力のルート要素名をカスタマイズできます。</li>
  <li><strong>整形出力</strong> でインデント付き/コンパクト出力を切り替えます。</li>
  <li>どちらのパネルでも <strong>Ctrl+Enter</strong>（Mac: <strong>Cmd+Enter</strong>）で変換できます。</li>
</ol>

<h2>変換例</h2>

<h3>シンプルなオブジェクト</h3>
<p>入力 JSON：</p>

```json
{"name": "Alice", "age": 30, "active": true}
```

<p>出力 XML（ルート要素: <code>root</code>）：</p>

```xml
<?xml version="1.0" encoding="UTF-8"?>
<root>
  <name>Alice</name>
  <age>30</age>
  <active>true</active>
</root>
```

<h3>ネストされたオブジェクト</h3>

```json
{"server": {"host": "localhost", "port": 8080}}
```

```xml
<?xml version="1.0" encoding="UTF-8"?>
<root>
  <server>
    <host>localhost</host>
    <port>8080</port>
  </server>
</root>
```

<h3>配列</h3>

```json
{"tags": ["js", "xml", "api"]}
```

```xml
<?xml version="1.0" encoding="UTF-8"?>
<root>
  <tags>js</tags>
  <tags>xml</tags>
  <tags>api</tags>
</root>
```

<h2>JSONとXML</h2>
<p>
  <strong>JSON</strong>はREST APIとWebサービスの標準データ形式で、コンパクトで人間が読みやすく、あらゆる言語でネイティブにサポートされています。
  <strong>XML</strong>はエンタープライズシステム、SOAP API、設定ファイル（Maven・Spring）、ドキュメント形式（SVG・XHTML・RSS）で今も広く使われています。
  このツールは、異なるデータ形式を使うシステム間の統合をサポートします。
</p>
