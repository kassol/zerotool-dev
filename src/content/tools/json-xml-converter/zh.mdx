<h2>使用方法</h2>
<ol>
  <li><strong>JSON → XML</strong>：将 JSON 粘贴到左侧面板，点击 <strong>JSON → XML</strong> 按钮，或直接输入即可实时转换。</li>
  <li><strong>XML → JSON</strong>：将 XML 粘贴到右侧面板，点击 <strong>XML → JSON</strong> 按钮。</li>
  <li>在 <strong>根元素名</strong> 输入框自定义 XML 输出的根元素名称。</li>
  <li>切换 <strong>格式化输出</strong> 控制是否缩进输出。</li>
  <li>在任意面板中使用 <strong>Ctrl+Enter</strong>（Mac 为 <strong>Cmd+Enter</strong>）快捷键触发转换。</li>
</ol>

<h2>示例</h2>

<h3>简单对象</h3>
<p>输入 JSON：</p>

```json
{"name": "Alice", "age": 30, "active": true}
```

<p>输出 XML（根元素：<code>root</code>）：</p>

```xml
<?xml version="1.0" encoding="UTF-8"?>
<root>
  <name>Alice</name>
  <age>30</age>
  <active>true</active>
</root>
```

<h3>嵌套对象</h3>

```json
{"server": {"host": "localhost", "port": 8080}}
```

```xml
<?xml version="1.0" encoding="UTF-8"?>
<root>
  <server>
    <host>localhost</host>
    <port>8080</port>
  </server>
</root>
```

<h3>数组</h3>

```json
{"tags": ["js", "xml", "api"]}
```

```xml
<?xml version="1.0" encoding="UTF-8"?>
<root>
  <tags>js</tags>
  <tags>xml</tags>
  <tags>api</tags>
</root>
```
