<h2>使い方</h2>
<ol>
  <li><strong>タイトル</strong>、<strong>説明文</strong>、<strong>正規 URL</strong> の 3 つの中核フィールドを入力します。</li>
  <li><strong>og:image URL</strong> を設定してソーシャルプレビューを正しく表示。1200x630 の PNG / JPEG が万能です。</li>
  <li><strong>Twitter カード形式</strong> を選択：ヒーロー画像付き投稿は <code>summary_large_image</code>、コンパクト表示なら <code>summary</code>。</li>
  <li>必要に応じて <strong>Schema.org @type</strong> を選び、最小の JSON-LD ブロックを出力します。</li>
  <li>右側プレビューのタブを切り替え、Google SERP / Facebook / Twitter (X) / Discord での展開を確認。</li>
  <li>生成された HTML をコピーし、ページの <code>&lt;head&gt;</code> 内に貼り付けます。</li>
</ol>

<h2>生成される内容</h2>
<ul>
  <li><strong>基本 SEO メタ</strong>——<code>title</code>、<code>description</code>、<code>canonical</code>、<code>author</code>、<code>keywords</code>、<code>theme-color</code>、<code>viewport</code>、<code>robots</code>。</li>
  <li><strong>Open Graph</strong>——<code>og:type</code>、<code>og:title</code>、<code>og:description</code>、<code>og:url</code>、<code>og:site_name</code>、<code>og:image</code>（width / height / alt 含む）、<code>og:locale</code>。</li>
  <li><strong>Twitter カード</strong>——<code>twitter:card</code>、<code>twitter:site</code>、<code>twitter:creator</code>、<code>twitter:title</code>、<code>twitter:description</code>、<code>twitter:image</code>。</li>
  <li><strong>Schema.org JSON-LD</strong>——最小の <code>WebPage</code> / <code>Article</code> / <code>Product</code> / <code>Organization</code> ブロック（name / description / url / image）。</li>
</ul>

<h2>出力をどこに貼るか</h2>
<p>生成されたメタブロックは <code>&lt;head&gt;</code> 内、<code>&lt;meta charset&gt;</code> の直後・サードパーティスクリプトの前に置くのがおすすめ。クローラーは順序を気にしませんが、ソーシャルメタを 1 か所にまとめておくと監査が楽になります。</p>
