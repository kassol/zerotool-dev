<h2>使い方</h2>
<ol>
  <li>左側の <strong>JSON 入力</strong> 欄に JSON を貼り付けまたは入力します（サンプル users データが事前にロード済み）。</li>
  <li><strong>jq フィルタ</strong> 欄に式を入力するか、下の <strong>例</strong> pill をクリック。</li>
  <li>結果は <strong>結果</strong> 欄に JSON シンタックスハイライト付きで表示されます。</li>
  <li><strong>コピー</strong> ボタンで整形済み出力をクリップボードに転送できます。</li>
</ol>

<h2>jq チートシート</h2>
<ul>
  <li><code>.</code> — 恒等フィルタ、入力をそのまま出力</li>
  <li><code>.field</code> — プロパティ値の取得</li>
  <li><code>.field?</code> — オプショナル取得（フィールド欠落時もエラーにしない）</li>
  <li><code>.[]</code> — 配列・オブジェクトの値を反復</li>
  <li><code>.[start:end]</code> — 配列スライス</li>
  <li><code>f | g</code> — f の出力を g にパイプ</li>
  <li><code>f, g</code> — f と g の両方を出力</li>
  <li><code>select(cond)</code> — cond が真の値のみ残す</li>
  <li><code>map(f)</code> — 配列の各要素に f を適用</li>
  <li><code>group_by(.k)</code> — key でグループ化</li>
  <li><code>sort_by(.k)</code> — key でソート</li>
  <li><code>length</code>、<code>keys</code>、<code>values</code>、<code>type</code>、<code>has(k)</code> — イントロスペクション</li>
  <li><code>add</code>、<code>min</code>、<code>max</code>、<code>unique</code> — 集計</li>
  <li><code>tonumber</code>、<code>tostring</code>、<code>ascii_downcase</code>、<code>split("/")</code> — 変換</li>
  <li><code>reduce</code>、<code>foreach</code>、<code>def</code> — 制御フローとカスタム関数</li>
</ul>

<h2>主な用途</h2>
<p>
  jq は shell パイプラインで JSON を扱う標準ツールです：<code>curl</code> による API レスポンス、Kubernetes
  マニフェスト、CI/CD ログ、AWS CLI 出力、IaC ステート、データエクスポート。この Playground は小さなサンプル
  でフィルタを試作してから、表現をそのまま <code>jq -r '...'</code> で大きなファイルに流すための場所です。
</p>
<p>
  よくあるレシピ：<code>.items | map(&#123;id, name&#125;)</code> で API リスト応答を平坦化、
  <code>group_by(.tag) | map(&#123;tag: .[0].tag, count: length&#125;)</code> でイベントを tag 別にバケット化、
  <code>[.[] | .duration_ms] | add / length</code> で数値フィールドの平均を計算。
</p>
