<h2>使用方法</h2>
<ol>
  <li>把 JSON 粘贴或输入到左侧 <strong>JSON 输入</strong> 区域（预置了示例 users 数据）。</li>
  <li>在 <strong>jq 过滤器</strong> 输入框写过滤器，或点击下方任意 <strong>示例</strong> pill。</li>
  <li>结果会在 <strong>结果</strong> 区域以高亮 JSON 渲染。</li>
  <li>点 <strong>复制</strong> 把格式化输出放进剪贴板。</li>
</ol>

<h2>jq 速查</h2>
<ul>
  <li><code>.</code> — 恒等过滤器，输入原样输出</li>
  <li><code>.field</code> — 取属性值</li>
  <li><code>.field?</code> — 可选取值，字段缺失时不报错</li>
  <li><code>.[]</code> — 迭代数组或对象的值</li>
  <li><code>.[start:end]</code> — 数组切片</li>
  <li><code>f | g</code> — 把 f 的输出管道给 g</li>
  <li><code>f, g</code> — 同时输出 f 和 g</li>
  <li><code>select(cond)</code> — 保留 cond 为 true 的值</li>
  <li><code>map(f)</code> — 对数组每个元素应用 f</li>
  <li><code>group_by(.k)</code> — 按 key 分组</li>
  <li><code>sort_by(.k)</code> — 按 key 排序</li>
  <li><code>length</code>、<code>keys</code>、<code>values</code>、<code>type</code>、<code>has(k)</code> — 内省</li>
  <li><code>add</code>、<code>min</code>、<code>max</code>、<code>unique</code> — 聚合</li>
  <li><code>tonumber</code>、<code>tostring</code>、<code>ascii_downcase</code>、<code>split("/")</code> — 转换</li>
  <li><code>reduce</code>、<code>foreach</code>、<code>def</code> — 控制流与自定义函数</li>
</ul>

<h2>常见用途</h2>
<p>
  jq 是 shell 管道处理 JSON 的标配工具：<code>curl</code> 拿到的 API 响应、Kubernetes manifest、CI/CD 日志、
  AWS CLI 输出、基础设施 state、数据导出。这个 Playground 用来在小样本上调好过滤器，然后把表达式直接
  复制到 <code>jq -r '...'</code> 处理大文件。
</p>
<p>
  常用配方：<code>.items | map(&#123;id, name&#125;)</code> 把 API 列表打平；
  <code>group_by(.tag) | map(&#123;tag: .[0].tag, count: length&#125;)</code> 按 tag 分桶；
  <code>[.[] | .duration_ms] | add / length</code> 算平均耗时。
</p>
