<h2>사용 방법</h2>
<ol>
  <li>왼쪽 <strong>JSON 입력</strong> 패널에 JSON을 붙여넣거나 입력합니다(샘플 users 데이터가 미리 로드되어 있음).</li>
  <li><strong>jq 필터</strong> 입력란에 식을 작성하거나 아래의 <strong>예시</strong> pill을 클릭하세요.</li>
  <li>결과는 <strong>결과</strong> 영역에 JSON 구문 강조 표시와 함께 렌더링됩니다.</li>
  <li><strong>복사</strong> 버튼으로 포맷된 출력을 클립보드로 가져갈 수 있습니다.</li>
</ol>

<h2>jq 치트시트</h2>
<ul>
  <li><code>.</code> — 항등 필터, 입력을 그대로 출력</li>
  <li><code>.field</code> — 속성 값 추출</li>
  <li><code>.field?</code> — 선택적 추출(필드 누락 시 오류 없음)</li>
  <li><code>.[]</code> — 배열 또는 객체 값 순회</li>
  <li><code>.[start:end]</code> — 배열 슬라이스</li>
  <li><code>f | g</code> — f의 출력을 g로 파이프</li>
  <li><code>f, g</code> — f와 g를 모두 출력</li>
  <li><code>select(cond)</code> — cond가 참인 값만 유지</li>
  <li><code>map(f)</code> — 배열의 각 요소에 f 적용</li>
  <li><code>group_by(.k)</code> — key로 그룹화</li>
  <li><code>sort_by(.k)</code> — key로 정렬</li>
  <li><code>length</code>, <code>keys</code>, <code>values</code>, <code>type</code>, <code>has(k)</code> — 인트로스펙션</li>
  <li><code>add</code>, <code>min</code>, <code>max</code>, <code>unique</code> — 집계</li>
  <li><code>tonumber</code>, <code>tostring</code>, <code>ascii_downcase</code>, <code>split("/")</code> — 변환</li>
  <li><code>reduce</code>, <code>foreach</code>, <code>def</code> — 제어 흐름 및 사용자 함수</li>
</ul>

<h2>주요 사용 사례</h2>
<p>
  jq는 shell 파이프라인에서 JSON을 다루는 표준 도구입니다: <code>curl</code>로 받은 API 응답, Kubernetes
  매니페스트, CI/CD 로그, AWS CLI 출력, 인프라 코드 상태, 데이터 내보내기. 이 Playground는 작은 샘플로
  필터를 시제품화한 뒤, 표현식을 그대로 <code>jq -r '...'</code>에 사용해 큰 파일을 처리하는 용도입니다.
</p>
<p>
  자주 쓰는 레시피: <code>.items | map(&#123;id, name&#125;)</code>로 API 리스트 응답을 평탄화,
  <code>group_by(.tag) | map(&#123;tag: .[0].tag, count: length&#125;)</code>로 이벤트를 tag별로 버킷화,
  <code>[.[] | .duration_ms] | add / length</code>로 숫자 필드의 평균 계산.
</p>
