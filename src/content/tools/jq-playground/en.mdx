<h2>How to Use</h2>
<ol>
  <li>Paste or type your JSON into the <strong>JSON Input</strong> panel (a sample users dataset is preloaded).</li>
  <li>Type a jq filter in the <strong>jq Filter</strong> field, or click one of the <strong>Example</strong> pills.</li>
  <li>Results render with JSON syntax highlighting under <strong>Result</strong>.</li>
  <li>Click <strong>Copy</strong> to copy the formatted output to your clipboard.</li>
</ol>

<h2>jq Cheat Sheet</h2>
<ul>
  <li><code>.</code> — the identity filter, returns the input unchanged</li>
  <li><code>.field</code> — extract a property value</li>
  <li><code>.field?</code> — extract optionally (no error when field is missing)</li>
  <li><code>.[]</code> — iterate over array or object values</li>
  <li><code>.[start:end]</code> — array slice</li>
  <li><code>f | g</code> — pipe output of f into g</li>
  <li><code>f, g</code> — produce both f and g outputs</li>
  <li><code>select(cond)</code> — keep values where cond is true</li>
  <li><code>map(f)</code> — apply f to each element of an array</li>
  <li><code>group_by(.k)</code> — group an array by key</li>
  <li><code>sort_by(.k)</code> — sort an array by key</li>
  <li><code>length</code>, <code>keys</code>, <code>values</code>, <code>type</code>, <code>has(k)</code> — introspection</li>
  <li><code>add</code>, <code>min</code>, <code>max</code>, <code>unique</code> — aggregation</li>
  <li><code>tonumber</code>, <code>tostring</code>, <code>ascii_downcase</code>, <code>split("/")</code> — conversion</li>
  <li><code>reduce</code>, <code>foreach</code>, <code>def</code> — control flow and custom functions</li>
</ul>

<h2>Common Use Cases</h2>
<p>
  jq is the standard tool for shell pipelines that touch JSON: API responses from <code>curl</code>, Kubernetes
  manifests, CI/CD logs, AWS CLI output, infrastructure-as-code state, and analytics exports. This playground
  is for prototyping a filter on a small sample before piping a giant file through the CLI — copy the filter
  out and use it verbatim with <code>jq -r '...'</code>.
</p>
<p>
  Typical recipes: <code>.items | map(&#123;id, name&#125;)</code> to flatten API list responses,
  <code>group_by(.tag) | map(&#123;tag: .[0].tag, count: length&#125;)</code> to bucket events,
  <code>[.[] | .duration_ms] | add / length</code> to average a numeric field.
</p>
