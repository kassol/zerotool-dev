<h2>使用方法</h2>
<ol>
  <li>在左侧 <strong>JSON 输入</strong> 区域粘贴您的 JSON。</li>
  <li>右侧 <strong>JSON Schema</strong> 区域会自动更新结果。</li>
  <li>点击 <strong>复制</strong> 将 Schema 复制到剪贴板。</li>
  <li>按需调整 Schema——添加约束条件、描述说明或修改必填字段。</li>
</ol>

<h2>理解输出结果</h2>
<p>
  生成的 Schema 始终包含指向 draft-07 的 <code>$schema</code> 字段。
  对象属性根据 JSON 的键名推断，<code>required</code> 数组列出所有非 null 键。
  数组的 <code>items</code> 根据实际元素推断类型。
</p>

<h2>生成后的优化建议</h2>
<p>
  生成的 Schema 只是起点，常见的优化包括：为字段添加 <code>description</code> 说明、
  用 <code>minLength</code> 或 <code>pattern</code> 收紧字符串约束、
  将可选字段从 <code>required</code> 中移除，
  以及为枚举类字段添加 <code>enum</code> 限制。
</p>
