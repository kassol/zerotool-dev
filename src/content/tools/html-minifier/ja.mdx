<h2>使い方</h2>
<ol>
  <li>HTML（断片でも doctype 付きの完全な文書でも可）を入力欄に貼り付けます。</li>
  <li><strong>圧縮</strong> でコメント削除＋空白圧縮、<strong>整形</strong> で再インデントを実行します。</li>
  <li>整形時はインデント幅を 2 スペース・4 スペース・Tab から選択できます。</li>
  <li>圧縮後はステータス行に削減量（例：<code>12.4 KB → 8.1 KB (−34.6%)</code>）が表示されます。</li>
  <li>出力パネルの <strong>コピー</strong> ボタンで結果を取得できます。</li>
</ol>

<h2>サンプル</h2>
<h3>圧縮前</h3>

```html
<!DOCTYPE html>
<html lang="ja">
  <head>
    <!-- ページメタ情報 -->
    <title>こんにちは</title>
  </head>
  <body>
    <h1>Hello World</h1>
    <p>太字を含む <strong>テキスト</strong>。</p>
  </body>
</html>
```

<h3>圧縮後</h3>

```html
<!doctype html><html lang="ja"><head><title>こんにちは</title></head><body><h1>Hello World</h1><p>太字を含む <strong>テキスト</strong>。</p></body></html>
```

<h2>圧縮されるもの</h2>
<ul>
  <li>タグ間の空白（inter-tag whitespace）が折りたたまれます。</li>
  <li>HTML コメントは削除されます — IE 条件付きコメント（<code>&lt;!--[if IE]&gt;</code>）は例外として保持されます。</li>
  <li><code>disabled=""</code> のような boolean 属性は <code>disabled</code> に簡略化されます。</li>
  <li>テキストノードの前後空白を切り詰め、内部の連続空白は単一スペースに圧縮します。</li>
</ul>

<h2>そのまま保持されるもの</h2>
<ul>
  <li><code>&lt;script&gt;</code>、<code>&lt;style&gt;</code>、<code>&lt;pre&gt;</code>、<code>&lt;textarea&gt;</code>、<code>&lt;code&gt;</code>、<code>&lt;samp&gt;</code>、<code>&lt;kbd&gt;</code> 内のコンテンツ。</li>
  <li>属性値（引用符内の空白や、インラインイベントハンドラを含む）。</li>
  <li><code>&lt;!DOCTYPE&gt;</code> 宣言。</li>
</ul>

<h2>HTML を手動圧縮するタイミング</h2>
<p>モダンなビルドチェーン（Vite、Webpack、Astro、Next.js）はバンドル時に HTML を自動圧縮します。本ツールは単発の作業に向いています — CMS からエクスポートされたコード、圧縮前後のサイズ比較、手書きテンプレートの確認など。整形は日常的に使える機能で、本番環境の 1 行 HTML を貼り付けてデバッグや diff のための可読版を得られます。</p>
