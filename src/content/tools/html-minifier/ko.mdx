<h2>사용 방법</h2>
<ol>
  <li>HTML(조각 또는 doctype 포함 완전 문서)을 입력 패널에 붙여넣습니다.</li>
  <li><strong>압축</strong> 으로 주석을 제거하고 공백을 축소하거나, <strong>정렬</strong> 으로 재들여쓰기 합니다.</li>
  <li>정렬 시 들여쓰기 너비를 2 칸, 4 칸, Tab 중에서 선택할 수 있습니다.</li>
  <li>압축 후 상태 표시줄에 절감 정보(예: <code>12.4 KB → 8.1 KB (−34.6%)</code>)가 표시됩니다.</li>
  <li>출력 패널의 <strong>복사</strong> 버튼으로 결과를 가져갈 수 있습니다.</li>
</ol>

<h2>예시</h2>
<h3>압축 전</h3>

```html
<!DOCTYPE html>
<html lang="ko">
  <head>
    <!-- 페이지 메타 정보 -->
    <title>안녕하세요</title>
  </head>
  <body>
    <h1>Hello World</h1>
    <p>굵게 <strong>강조</strong> 한 문장.</p>
  </body>
</html>
```

<h3>압축 후</h3>

```html
<!doctype html><html lang="ko"><head><title>안녕하세요</title></head><body><h1>Hello World</h1><p>굵게 <strong>강조</strong> 한 문장.</p></body></html>
```

<h2>압축 대상</h2>
<ul>
  <li>태그 사이의 공백(inter-tag whitespace)을 축소합니다.</li>
  <li>HTML 주석은 제거됩니다 — IE 조건부 주석(<code>&lt;!--[if IE]&gt;</code>)은 보존됩니다.</li>
  <li><code>disabled=""</code> 와 같은 boolean 속성은 <code>disabled</code> 로 단순화됩니다.</li>
  <li>텍스트 노드의 앞뒤 공백을 잘라내고 내부 연속 공백은 단일 공백으로 압축합니다.</li>
</ul>

<h2>그대로 보존되는 항목</h2>
<ul>
  <li><code>&lt;script&gt;</code>, <code>&lt;style&gt;</code>, <code>&lt;pre&gt;</code>, <code>&lt;textarea&gt;</code>, <code>&lt;code&gt;</code>, <code>&lt;samp&gt;</code>, <code>&lt;kbd&gt;</code> 안의 내용.</li>
  <li>속성 값(따옴표 내부의 공백, 인라인 이벤트 핸들러 포함).</li>
  <li><code>&lt;!DOCTYPE&gt;</code> 선언.</li>
</ul>

<h2>HTML 을 수동으로 압축해야 할 때</h2>
<p>최신 빌드 체인(Vite, Webpack, Astro, Next.js)은 번들 단계에서 HTML 을 자동으로 압축합니다. 이 도구는 1 회성 작업에 적합합니다 — CMS 에서 내보낸 코드, 압축 전후 크기 비교, 수작업 템플릿 점검 등. 정렬 기능은 일상적으로 사용할 수 있으며, 프로덕션 환경의 한 줄짜리 HTML 을 붙여넣어 디버깅이나 diff 를 위한 가독 가능한 사본을 얻을 수 있습니다.</p>
