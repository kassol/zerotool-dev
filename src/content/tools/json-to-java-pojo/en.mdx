<h2>How to Use</h2>
<ol>
  <li>Paste or type your JSON into the left panel. The tool validates it in real time.</li>
  <li>Optionally set a <strong>Root class name</strong> (default: <code>RootObject</code>).</li>
  <li>Choose an annotation mode: <strong>None</strong>, <strong>Jackson</strong>, <strong>Gson</strong>, or <strong>Lombok (@Data)</strong>.</li>
  <li>Click <strong>Generate Java</strong> or wait for real-time conversion.</li>
  <li>Click <strong>Copy</strong> to copy the output to your clipboard.</li>
</ol>

<h2>Type Mapping</h2>
<ul>
  <li><code>null</code> → <code>Object</code></li>
  <li>string → <code>String</code></li>
  <li>integer number → <code>int</code></li>
  <li>floating-point number → <code>double</code></li>
  <li>boolean → <code>boolean</code></li>
  <li>array → <code>List&lt;T&gt;</code></li>
  <li>nested object → separate <code>public class</code></li>
</ul>

<h2>Example</h2>
<p>Given this JSON:</p>
<pre><code>{`{"user": {"first_name": "Alice", "age": 30}, "tags": ["admin"]}`}</code></pre>
<p>The tool generates (Lombok mode):</p>
<pre><code>{`import java.util.List;\nimport lombok.Data;\n\n@Data\npublic class User {\n    private String firstName;\n    private int age;\n}\n\n@Data\npublic class RootObject {\n    private User user;\n    private List<String> tags;\n}`}</code></pre>
