<h2>使用方法</h2>
<ol>
  <li>将 JSON 粘贴或输入到左侧面板，工具会实时验证。</li>
  <li>可选设置<strong>根类名称</strong>（默认：<code>RootObject</code>）。</li>
  <li>选择注解模式：<strong>无注解</strong>、<strong>Jackson</strong>、<strong>Gson</strong> 或 <strong>Lombok (@Data)</strong>。</li>
  <li>点击<strong>生成 Java</strong>，或等待实时转换结果。</li>
  <li>点击<strong>复制</strong>将输出复制到剪贴板。</li>
</ol>

<h2>类型映射</h2>
<ul>
  <li><code>null</code> → <code>Object</code></li>
  <li>字符串 → <code>String</code></li>
  <li>整数 → <code>int</code></li>
  <li>浮点数 → <code>double</code></li>
  <li>布尔值 → <code>boolean</code></li>
  <li>数组 → <code>List&lt;T&gt;</code></li>
  <li>嵌套对象 → 独立 <code>public class</code></li>
</ul>

<h2>示例</h2>
<p>给定如下 JSON：</p>
<pre><code>{`{"user": {"first_name": "Alice", "age": 30}, "tags": ["admin"]}`}</code></pre>
<p>Lombok 模式下生成：</p>
<pre><code>{`import java.util.List;\nimport lombok.Data;\n\n@Data\npublic class User {\n    private String firstName;\n    private int age;\n}\n\n@Data\npublic class RootObject {\n    private User user;\n    private List<String> tags;\n}`}</code></pre>
