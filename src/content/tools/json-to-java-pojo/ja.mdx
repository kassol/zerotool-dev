<h2>使い方</h2>
<ol>
  <li>左パネルにJSONを貼り付けるか入力します。リアルタイムで検証されます。</li>
  <li>必要に応じて<strong>ルートクラス名</strong>を設定します（デフォルト：<code>RootObject</code>）。</li>
  <li>アノテーションモードを選択します：<strong>なし</strong>、<strong>Jackson</strong>、<strong>Gson</strong>、または<strong>Lombok (@Data)</strong>。</li>
  <li><strong>Javaを生成</strong>をクリックするか、リアルタイム変換を待ちます。</li>
  <li><strong>コピー</strong>をクリックして出力をクリップボードにコピーします。</li>
</ol>

<h2>型マッピング</h2>
<ul>
  <li><code>null</code> → <code>Object</code></li>
  <li>文字列 → <code>String</code></li>
  <li>整数 → <code>int</code></li>
  <li>浮動小数点数 → <code>double</code></li>
  <li>真偽値 → <code>boolean</code></li>
  <li>配列 → <code>List&lt;T&gt;</code></li>
  <li>ネストオブジェクト → 独立した <code>public class</code></li>
</ul>

<h2>例</h2>
<p>以下のJSONを入力した場合：</p>
<pre><code>{`{"user": {"first_name": "Alice", "age": 30}, "tags": ["admin"]}`}</code></pre>
<p>Lombokモードで生成される出力：</p>
<pre><code>{`import java.util.List;\nimport lombok.Data;\n\n@Data\npublic class User {\n    private String firstName;\n    private int age;\n}\n\n@Data\npublic class RootObject {\n    private User user;\n    private List<String> tags;\n}`}</code></pre>
