<h2>사용 방법</h2>
<ol>
  <li>왼쪽 패널에 JSON을 붙여넣거나 입력합니다. 실시간으로 유효성이 검사됩니다.</li>
  <li>필요에 따라 <strong>루트 클래스 이름</strong>을 설정합니다(기본값: <code>RootObject</code>).</li>
  <li>어노테이션 모드를 선택합니다: <strong>없음</strong>, <strong>Jackson</strong>, <strong>Gson</strong>, 또는 <strong>Lombok (@Data)</strong>.</li>
  <li><strong>Java 생성</strong>을 클릭하거나 실시간 변환 결과를 기다립니다.</li>
  <li><strong>복사</strong>를 클릭하여 출력을 클립보드에 복사합니다.</li>
</ol>

<h2>타입 매핑</h2>
<ul>
  <li><code>null</code> → <code>Object</code></li>
  <li>문자열 → <code>String</code></li>
  <li>정수 → <code>int</code></li>
  <li>부동소수점 → <code>double</code></li>
  <li>불리언 → <code>boolean</code></li>
  <li>배열 → <code>List&lt;T&gt;</code></li>
  <li>중첩 객체 → 독립된 <code>public class</code></li>
</ul>

<h2>예시</h2>
<p>다음 JSON 입력 시:</p>
<pre><code>{`{"user": {"first_name": "Alice", "age": 30}, "tags": ["admin"]}`}</code></pre>
<p>Lombok 모드로 생성되는 출력:</p>
<pre><code>{`import java.util.List;\nimport lombok.Data;\n\n@Data\npublic class User {\n    private String firstName;\n    private int age;\n}\n\n@Data\npublic class RootObject {\n    private User user;\n    private List<String> tags;\n}`}</code></pre>
