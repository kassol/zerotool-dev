<h2>텍스트 케이스 변환에 대하여</h2>
<p>프로그래밍에서는 변수명·함수명·상수명에 각각 다른 명명 규칙이 사용됩니다. 이 도구를 사용하면 기존 텍스트를 원하는 형식으로 빠르게 변환할 수 있습니다.</p>
<ul>
  <li><strong>camelCase</strong>: JavaScript 변수, 객체 속성에 사용</li>
  <li><strong>PascalCase</strong>: 클래스명, 컴포넌트명에 사용</li>
  <li><strong>snake_case</strong>: Python 변수, 데이터베이스 열 이름에 사용</li>
  <li><strong>kebab-case</strong>: CSS 클래스명, URL 슬러그에 사용</li>
  <li><strong>CONSTANT_CASE</strong>: 상수, 환경 변수에 사용</li>
</ul>
