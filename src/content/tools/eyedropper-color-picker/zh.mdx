<h2>如何使用</h2>
<ol>
  <li>点击 <strong>从屏幕吸色</strong>。光标会变成放大镜 — 点击任意像素即可采样。</li>
  <li>或者使用 <strong>原生拾色器</strong> 按钮打开浏览器内置的取色对话框（所有浏览器均可用）。</li>
  <li>HEX、RGB、HSL 字段会同步更新。编辑任一字段，另外两个会自动重新计算。</li>
  <li>点击任一格式旁边的 <strong>复制</strong> 按钮，即可将值发送到剪贴板。</li>
  <li>最近 8 个采样颜色会保留在 <strong>最近颜色</strong> 条中 — 点击色块即可重新载入。</li>
</ol>

<h2>60 秒看懂 EyeDropper API</h2>
<p>
  <code>EyeDropper</code> 是浏览器原生的 Web API，允许页面从用户屏幕任意位置采样像素。页面调用 <code>new EyeDropper().open()</code>；浏览器打开系统级放大镜；点击后，Promise resolve 出一个对象，其 <code>sRGBHex</code> 属性即为所拾取的颜色。
</p>
<p>
  该 API 保护隐私：页面无法获取像素数据，只能拿到用户选中的那一个颜色。每次取色必须由用户点击触发 — 没有任何无头或脚本化的采样方式。
</p>

<h2>不同模式如何选择</h2>
<ul>
  <li><strong>从屏幕吸色</strong> — 当颜色出现在其他应用、截图或视频画面里时使用。</li>
  <li><strong>原生拾色器</strong> — 想要新建颜色或使用熟悉的 HSV 色盘时使用。</li>
  <li><strong>在字段中输入</strong> — 已经有精确数值（如设计稿规范），想要换算到其他格式时使用。</li>
</ul>
