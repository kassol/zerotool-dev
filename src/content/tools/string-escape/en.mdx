<h2>JavaScript Escape Sequences</h2>
<ul>
  <li><code>\n</code> — newline (LF)</li>
  <li><code>\r</code> — carriage return (CR)</li>
  <li><code>\t</code> — horizontal tab</li>
  <li><code>\\</code> — literal backslash</li>
  <li><code>\"</code> — double quote</li>
  <li><code>\'</code> — single quote</li>
  <li><code>\uXXXX</code> — Unicode code point (4 hex digits)</li>
  <li><code>\xHH</code> — Latin-1 character (2 hex digits)</li>
  <li><code>\0</code> — null character</li>
</ul>

<h2>JSON Escaping</h2>
<p>JSON escaping follows <a href="https://www.rfc-editor.org/rfc/rfc8259" rel="noopener noreferrer">RFC 8259</a>. The tool strips the surrounding double-quotes that <code>JSON.stringify</code> adds, so the output is the raw escaped string content. To unescape, wrap the input in quotes and parse it as a JSON string value.</p>

<h2>HTML Entity Reference</h2>
<ul>
  <li><code>&amp;amp;</code> → &amp;</li>
  <li><code>&amp;lt;</code> → &lt;</li>
  <li><code>&amp;gt;</code> → &gt;</li>
  <li><code>&amp;quot;</code> → "</li>
  <li><code>&amp;#39;</code> → '</li>
  <li><code>&amp;nbsp;</code> → non-breaking space</li>
  <li><code>&amp;#233;</code> → é (decimal numeric reference)</li>
  <li><code>&amp;#xE9;</code> → é (hex numeric reference)</li>
</ul>
