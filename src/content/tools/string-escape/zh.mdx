<h2>JavaScript 转义序列</h2>
<ul>
  <li><code>\n</code> — 换行（LF）</li>
  <li><code>\r</code> — 回车（CR）</li>
  <li><code>\t</code> — 水平制表符</li>
  <li><code>\\</code> — 字面反斜杠</li>
  <li><code>\"</code> — 双引号</li>
  <li><code>\'</code> — 单引号</li>
  <li><code>\uXXXX</code> — Unicode 码点（4 位十六进制）</li>
  <li><code>\xHH</code> — Latin-1 字符（2 位十六进制）</li>
  <li><code>\0</code> — 空字符</li>
</ul>

<h2>JSON 转义说明</h2>
<p>JSON 转义遵循 RFC 8259 规范。工具会去除 <code>JSON.stringify</code> 添加的外层双引号，输出的是原始转义后的字符串内容。反转义时，工具会在输入外侧添加引号并将其作为 JSON 字符串值解析。</p>

<h2>HTML 实体参考</h2>
<ul>
  <li><code>&amp;amp;</code> → &amp;</li>
  <li><code>&amp;lt;</code> → &lt;</li>
  <li><code>&amp;gt;</code> → &gt;</li>
  <li><code>&amp;quot;</code> → "</li>
  <li><code>&amp;#39;</code> → '</li>
  <li><code>&amp;nbsp;</code> → 不间断空格</li>
  <li><code>&amp;#233;</code> → é（十进制数字引用）</li>
  <li><code>&amp;#xE9;</code> → é（十六进制数字引用）</li>
</ul>
