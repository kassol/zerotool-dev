<h2>JavaScript 이스케이프 시퀀스</h2>
<ul>
  <li><code>\n</code> — 줄 바꿈 (LF)</li>
  <li><code>\r</code> — 캐리지 리턴 (CR)</li>
  <li><code>\t</code> — 수평 탭</li>
  <li><code>\\</code> — 백슬래시</li>
  <li><code>\"</code> — 큰따옴표</li>
  <li><code>\'</code> — 작은따옴표</li>
  <li><code>\uXXXX</code> — 유니코드 코드 포인트 (16진수 4자리)</li>
  <li><code>\xHH</code> — Latin-1 문자 (16진수 2자리)</li>
  <li><code>\0</code> — 널 문자</li>
</ul>

<h2>JSON 이스케이프 안내</h2>
<p>JSON 이스케이프는 RFC 8259를 따릅니다. 이 도구는 <code>JSON.stringify</code>가 추가하는 외부 큰따옴표를 제거하여 이스케이프된 문자열 내용만 출력합니다. 언이스케이프 시에는 입력을 따옴표로 감싸서 JSON 문자열 값으로 파싱합니다.</p>

<h2>HTML 엔티티 참조</h2>
<ul>
  <li><code>&amp;amp;</code> → &amp;</li>
  <li><code>&amp;lt;</code> → &lt;</li>
  <li><code>&amp;gt;</code> → &gt;</li>
  <li><code>&amp;quot;</code> → "</li>
  <li><code>&amp;#39;</code> → '</li>
  <li><code>&amp;nbsp;</code> → 줄 바꿈 없는 공백</li>
  <li><code>&amp;#233;</code> → é (10진수 숫자 참조)</li>
  <li><code>&amp;#xE9;</code> → é (16진수 숫자 참조)</li>
</ul>
