<h2>使い方</h2>
<ol>
  <li>モードタブを選択：<strong>JavaScript</strong>・<strong>JSON</strong>・<strong>HTMLエンティティ</strong>。</li>
  <li>方向を選択：<strong>エスケープ</strong>または<strong>アンエスケープ</strong>。</li>
  <li>左の入力欄にテキストを入力または貼り付けると、出力がリアルタイムで更新されます。</li>
  <li><strong>コピー</strong>で結果をコピー、<strong>入れ替え</strong>で出力を入力欄に移動できます。</li>
</ol>

<h2>JavaScriptエスケープシーケンス</h2>
<ul>
  <li><code>\n</code> — 改行（LF）</li>
  <li><code>\r</code> — キャリッジリターン（CR）</li>
  <li><code>\t</code> — 水平タブ</li>
  <li><code>\\</code> — バックスラッシュ</li>
  <li><code>\"</code> — ダブルクォート</li>
  <li><code>\'</code> — シングルクォート</li>
  <li><code>\uXXXX</code> — Unicodeコードポイント（16進4桁）</li>
  <li><code>\xHH</code> — Latin-1文字（16進2桁）</li>
  <li><code>\0</code> — ヌル文字</li>
</ul>

<h2>JSONエスケープにつ��て</h2>
<p>JSONエスケープはRFC 8259に準拠しています。ツールは<code>JSON.stringify</code>が追加する外側のダブルクォートを除去し、エスケープされた文字列の内容のみを出力します。アンエスケープ時は入力を引用符で囲んでJSONの文字列値として解析します。</p>

<h2>HTMLエンティティ参照</h2>
<ul>
  <li><code>&amp;amp;</code> → &amp;</li>
  <li><code>&amp;lt;</code> → &lt;</li>
  <li><code>&amp;gt;</code> → &gt;</li>
  <li><code>&amp;quot;</code> → "</li>
  <li><code>&amp;#39;</code> → '</li>
  <li><code>&amp;nbsp;</code> → ノーブレークスペース</li>
  <li><code>&amp;#233;</code> → é（10進数値参照）</li>
  <li><code>&amp;#xE9;</code> → é（16進数値参照）</li>
</ul>
