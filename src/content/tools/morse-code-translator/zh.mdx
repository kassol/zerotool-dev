<h2>使用方法</h2>
<ol>
  <li><strong>文字 → 摩斯码</strong>：在输入框中输入或粘贴文字，右侧即时显示摩斯码。</li>
  <li><strong>摩斯码 → 文字</strong>：用 <code>.</code> 表示点，<code>-</code> 表示划，字母间用空格，单词间用 <code> / </code>。</li>

<h2>摩斯码格式说明</h2>
<p>每个字母用一串点和划表示，字母编码之间用空格分隔，单词之间用 <code> / </code> 分隔。例如：<code>.... . .-.. .-.. ---</code> 解码为 <code>HELLO</code>。</p>

<h2>什么是摩斯电码？</h2>
<p>摩斯电码是 1830 年代为电报通信开发的字符编码系统，用点（短信号）和划（长信号）的组合来表示字母和数字。至今仍在业余无线电（CW 模式）和无障碍输入法中使用。</p>
