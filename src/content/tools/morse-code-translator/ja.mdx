<h2>使い方</h2>
<ol>
  <li><strong>テキスト → モールス符号</strong>：左側にテキストを入力すると、右側にモールス符号が即座に表示されます。</li>
  <li><strong>モールス符号 → テキスト</strong>：<code>.</code>で点、<code>-</code>でダッシュを入力。文字間はスペース、単語間は<code> / </code>で区切ります。</li>
  <li><strong>コピー</strong>をクリックして結果をコピーします。</li>
</ol>

<h2>モールス符号のフォーマット</h2>
<p>各文字は点とダッシュのシーケンスで表され、文字コード間はスペースで区切られます。単語は<code> / </code>で区切られます。例：<code>.... . .-.. .-.. ---</code>は<code>HELLO</code>にデコードされます。</p>

<h2>モールス符号とは？</h2>
<p>モールス符号は1830年代に電信通信のために開発された文字符号化システムです。短点（·）と長線（—）の組み合わせで文字と数字を表します。現在もアマチュア無線（CWモード）やアクセシビリティ入力として使用されています。</p>
