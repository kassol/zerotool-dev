<h2>How to Use</h2>
<ol>
  <li><strong>Text → Morse</strong>: Type or paste text in the input panel. The Morse code appears instantly on the right.</li>
  <li><strong>Morse → Text</strong>: Enter Morse code using <code>.</code> for dots and <code>-</code> for dashes. Separate letters with spaces, words with <code> / </code>.</li>

</ol>

<h2>Morse Code Format</h2>
<p>Each letter is represented as a sequence of dots and dashes separated by spaces. Words are delimited by <code> / </code>. Example: <code>.... . .-.. .-.. --- / .-- --- .-. .-.. -..</code> decodes to <code>HELLO WORLD</code>.</p>

<h2>What Is Morse Code?</h2>
<p>Morse code is a character encoding system developed in the 1830s for electrical telegraph communication. It represents letters and numbers as sequences of short (dot) and long (dash) signals. It is still used today in amateur radio (CW mode) and as an accessibility input method.</p>
