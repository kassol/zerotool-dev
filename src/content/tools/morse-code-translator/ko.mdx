<h2>사용 방법</h2>
<ol>
  <li><strong>텍스트 → 모스 부호</strong>: 왼쪽에 텍스트를 입력하면 오른쪽에 모스 부호가 즉시 표시됩니다.</li>
  <li><strong>모스 부호 → 텍스트</strong>: <code>.</code>으로 점, <code>-</code>으로 대시를 입력합니다. 문자 사이는 공백, 단어 사이는 <code> / </code>로 구분하세요.</li>
  <li><strong>복사</strong>를 클릭하여 결과를 복사합니다.</li>
</ol>

<h2>모스 부호 형식</h2>
<p>각 문자는 점과 대시의 시퀀스로 표시되며, 문자 코드 사이는 공백으로 구분됩니다. 단어는 <code> / </code>로 구분됩니다. 예: <code>.... . .-.. .-.. ---</code>는 <code>HELLO</code>로 디코딩됩니다.</p>

<h2>모스 부호란?</h2>
<p>모스 부호는 1830년대 전신 통신을 위해 개발된 문자 인코딩 시스템입니다. 짧은 점(·)과 긴 선(—)의 조합으로 문자와 숫자를 표현합니다. 현재도 아마추어 무선(CW 모드)과 접근성 입력 방법으로 사용됩니다.</p>
