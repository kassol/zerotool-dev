<h2>사용 방법</h2>
<ol>
  <li>XML을 입력창에 붙여넣습니다.</li>
  <li><strong>정렬</strong>을 클릭하여 들여쓰기가 있는 읽기 쉬운 형식으로 변환합니다.</li>
  <li><strong>압축</strong>을 클릭하여 공백을 제거한 최소화 형식으로 만듭니다.</li>

<h2>XML에 대하여</h2>
<p>XML(eXtensible Markup Language)은 데이터 구조화·교환에 사용되는 마크업 언어입니다. HTML과 형태는 비슷하지만 데이터 기술에 특화되어 있습니다. 설정 파일(Maven, Spring, Android의 manifest 등), Web API(SOAP), RSS/Atom 피드, Office 문서 형식(DOCX/XLSX의 내부) 등에 널리 사용됩니다.</p>
