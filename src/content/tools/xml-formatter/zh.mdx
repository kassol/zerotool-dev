<h2>使用方法</h2>
<ol>
  <li>将 XML 粘贴到左侧输入面板。</li>
  <li>点击 <strong>Format</strong> 美化或 <strong>Minify</strong> 压缩。</li>
  <li>从选项中选择偏好的缩进方式。</li>
  <li>点击输出面板的 <strong>Copy</strong> 按钮复制结果。</li>
</ol>

<h2>格式化示例</h2>
<h3>格式化前</h3>

```xml
<root><user id="1"><name>Alice</name><email>alice@example.com</email></user></root>
```

<h3>格式化后</h3>

```xml
<root>
  <user id="1">
    <name>Alice</name>
    <email>alice@example.com</email>
  </user>
</root>
```

<h2>XML 格式化最佳实践</h2>
<p>正确格式化的 XML 提高可读性，更容易发现结构问题。使用一致的缩进，属性较短时保持在元素同一行，发送到 API 或配置文件之前始终验证 XML。</p>
