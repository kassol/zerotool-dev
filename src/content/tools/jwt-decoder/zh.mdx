<h2>使用方法</h2>
<ol>
  <li>将 JWT 令牌粘贴到输入框中。</li>
  <li>点击 <strong>Decode</strong>（或按 Ctrl+Enter / Cmd+Enter）。</li>
  <li>Header、Payload 和 Signature 将以语法高亮的形式显示。</li>
  <li>时间戳声明（<code>exp</code>、<code>iat</code>、<code>nbf</code>）会显示可读日期和过期状态。</li>
  <li>点击任意区域的 <strong>Copy</strong> 可复制该 JSON 到剪贴板。</li>
</ol>

<h2>什么是 JWT？</h2>
<p>
  JSON Web Token（JWT）是一种紧凑的、URL 安全的令牌格式，用于身份验证和信息交换。
  它由三个 Base64URL 编码的部分组成，用点号分隔：
</p>
<ul>
  <li><strong>Header：</strong>指定签名算法（如 HS256、RS256）和令牌类型。</li>
  <li><strong>Payload：</strong>包含声明——关于实体（通常是用户）的陈述和附加元数据。</li>
  <li><strong>Signature：</strong>确保令牌未被篡改。需要密钥或公钥来验证。</li>
</ul>
<p>
  常见声明包括 <code>sub</code>（主题）、<code>iss</code>（签发者）、<code>exp</code>（过期时间）、
  <code>iat</code>（签发时间）和 <code>aud</code>（受众）。
</p>
