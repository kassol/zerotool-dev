<h2>사용 방법</h2>
<ol>
  <li>JWT 토큰을 입력창에 붙여넣습니다.</li>
  <li>헤더·페이로드·서명이 자동으로 파싱되어 표시됩니다.</li>
  <li>exp 클레임이 있는 경우, 만료 시각 상태가 하이라이트됩니다.</li>
</ol>

<h2>JWT에 대하여</h2>
<p>JWT는 웹 애플리케이션의 인증·인가에 널리 사용됩니다. 사용자가 로그인하면 서버는 JWT를 발급하고, 클라이언트는 이후 요청에서 JWT를 Authorization 헤더에 포함하여 전송합니다.</p>
<p>디버깅 시 JWT의 내용(사용자 ID, 역할, 만료 시각 등)을 확인하고 싶을 때 이 도구가 유용합니다.</p>
