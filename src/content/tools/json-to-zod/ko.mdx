<h2>사용 방법</h2>
<ol>
  <li>왼쪽 패널에 <strong>JSON 데이터</strong>를 붙여넣습니다 (또는 <strong>예시</strong> 클릭).</li>
  <li><strong>루트 스키마 이름</strong>을 설정합니다 (내보내기 상수와 타입에 사용).</li>
  <li>선택적으로 <strong>엄격 모드</strong>를 활성화하여 추가 키를 불허합니다.</li>
  <li><strong>Zod 스키마 생성</strong>을 클릭합니다.</li>
  <li><strong>복사</strong>를 클릭하여 프로젝트에 붙여넣습니다.</li>
</ol>

<h2>출력 형식</h2>
<p>완전한 TypeScript 스니펫을 출력합니다:</p>
<ul>
  <li><code>import &#123; z &#125; from "zod"</code> — Zod 가져오기</li>
  <li><code>const FooSchema = z.object(&#123;...&#125;)</code> — 스키마 정의</li>
  <li><code>export type Foo = z.infer&lt;typeof FooSchema&gt;</code> — 추론된 TypeScript 타입</li>
</ul>

<h2>주요 사용 사례</h2>
<ul>
  <li><strong>API 유효성 검사:</strong> API 응답 예시에서 스키마를 생성하여 실제 응답을 런타임에 검증.</li>
  <li><strong>폼 유효성 검사:</strong> 폼 데이터 형태를 Zod 스키마로 변환하여 react-hook-form 등에서 활용.</li>
  <li><strong>설정 파일:</strong> JSON 파일에서 로드하는 설정 객체를 정의하고 검증.</li>
  <li><strong>타입 안전성:</strong> TypeScript와 결합하여 하나의 스키마에서 런타임 검증과 정적 타입 추론을 동시에.</li>
</ul>
