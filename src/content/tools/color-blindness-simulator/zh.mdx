<h2>色觉障碍简介</h2>
<p>色觉障碍（CVD）影响眼睛锥细胞对光的反应。人类色觉依赖三种锥细胞：L 型（红色）、M 型（绿色）和 S 型（蓝色）。当某一类型缺失或功能异常时，特定颜色会变得难以区分。</p>
<ul>
  <li><strong>红色盲 / 红色弱</strong>——影响 L 锥细胞（红色感知）。红色变暗，偏向棕色或绿色。约 2% 的男性受影响。</li>
  <li><strong>绿色盲 / 绿色弱</strong>——影响 M 锥细胞（绿色感知）。最常见类型，约 6% 的男性受影响。绿色和红色看起来相似。</li>
  <li><strong>蓝色盲 / 蓝色弱</strong>——影响 S 锥细胞（蓝色感知）。罕见，影响不到 0.01% 的人群。蓝色和绿色容易混淆。</li>
  <li><strong>全色盲 / 全色弱</strong>——影响所有锥细胞类型。完全性全色盲呈灰度视觉；全色弱显著降低色彩饱和度。</li>
</ul>

<h2>无障碍设计建议</h2>
<p>将此工具与<a href="/zh/tools/color-contrast-checker/">颜色对比度检测器</a>配合使用，进行完整的无障碍审查。传达信息时不要仅依赖颜色，应同时使用形状、图案或文字标签。至少通过红色盲和绿色盲模拟测试关键 UI 元素（错误状态、图表、状态指示器），因为这两种类型覆盖了大多数受影响用户。</p>
