<h2>사용 방법</h2>
<ol>
  <li><strong>색상 모드</strong>에서 HEX 색상을 입력하거나 색상 선택기를 사용하세요. 8가지 색각 이상 유형에서 해당 색상이 어떻게 보이는지 즉시 그리드로 표시됩니다.</li>
  <li><strong>이미지 모드</strong>에서 이미지를 드래그 앤 드롭하거나 클릭하여 선택하세요. 로컬에서 처리 후 원본과 8가지 시뮬레이션 결과가 나란히 표시됩니다.</li>
  <li>색상 모드에서 HEX 값을 클릭하면 클립보드에 복사됩니다.</li>
</ol>

<h2>색각 이상이란</h2>
<p>색각 이상(CVD)은 눈의 원추세포가 빛에 반응하는 방식에 영향을 미칩니다. 인간의 색각은 L-원추(적색), M-원추(녹색), S-원추(청색) 세 가지 유형에 의존합니다. 한 유형이 없거나 기능이 비정상이면 특정 색상을 구별하기 어려워집니다.</p>
<ul>
  <li><strong>적색맹 / 적색약</strong> — L-원추(적색 인식)에 영향. 빨간색이 어두워지고 갈색이나 녹색에 가까워집니다. 남성의 약 2%.</li>
  <li><strong>녹색맹 / 녹색약</strong> — M-원추(녹색 인식)에 영향. 가장 흔한 유형으로 남성의 약 6%. 녹색과 빨간색이 비슷하게 보입니다.</li>
  <li><strong>청색맹 / 청색약</strong> — S-원추(청색 인식)에 영향. 드물며 인구의 0.01% 미만. 파란색과 녹색 구별이 어렵습니다.</li>
  <li><strong>전색맹 / 부분색약</strong> — 모든 원추 유형에 영향. 완전 전색맹은 회색조 시각이 되며, 부분색약은 색상 채도가 크게 감소합니다.</li>
</ul>
