<h2>使用方法</h2>
<ol>
  <li>在输入框中输入带 CIDR 表示法的 IP 地址（如 <code>192.168.1.0/24</code>），或只输入 IP 地址并从下拉菜单选择前缀长度。</li>
  <li>结果即时显示：网络地址、广播地址、子网掩码、通配符掩码、首尾可用主机及可用主机总数。</li>
  <li>点击<strong>复制</strong>可将所有结果以纯文本格式复制到剪贴板。</li>
</ol>

<h2>输出字段说明</h2>

| 字段 | 说明 |
|------|------|
| 网络地址 | 子网第一个地址，用于标识网络本身 |
| 广播地址 | 最后一个地址，发送给子网内所有主机 |
| 子网掩码 | 前缀的点分十进制形式（如 255.255.255.0） |
| 通配符掩码 | 子网掩码的按位取反 |
| 第一个可用主机 | 网络地址 + 1 |
| 最后一个可用主机 | 广播地址 − 1 |
| 可用主机数 | 总地址数减 2（网络 + 广播） |
| CIDR 表示法 | 规范的网络/前缀形式 |
