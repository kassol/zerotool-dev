<h2>使い方</h2>
<ol>
  <li>入力欄にCIDR表記のIPアドレス（例: <code>192.168.1.0/24</code>）を入力するか、IPアドレスのみ入力してドロップダウンからプレフィックス長を選択します。</li>
  <li>ネットワークアドレス、ブロードキャストアドレス、サブネットマスク、ワイルドカードマスク、先頭/末尾の使用可能ホスト、ホスト総数が即座に表示されます。</li>
  <li><strong>コピー</strong>ボタンをクリックすると、すべての結果をプレーンテキストでクリップボードにコピーできます。</li>
</ol>

<h2>出力フィールドの説明</h2>

| フィールド | 説明 |
|-----------|------|
| ネットワークアドレス | サブネットの最初のアドレス（ネットワーク自体を識別） |
| ブロードキャストアドレス | 最後のアドレス（サブネット内の全ホストに送信） |
| サブネットマスク | プレフィックスのドット区切り10進数表現（例: 255.255.255.0） |
| ワイルドカードマスク | サブネットマスクのビット反転 |
| 最初の使用可能ホスト | ネットワークアドレス + 1 |
| 最後の使用可能ホスト | ブロードキャストアドレス − 1 |
| 使用可能ホスト数 | 総アドレス数 − 2（ネットワーク + ブロードキャスト） |
| CIDR表記 | 標準的なネットワーク/プレフィックス形式 |
