<h2>사용 방법</h2>
<ol>
  <li><strong>CSV → JSON</strong>: CSV 데이터를 입력창에 붙여넣고 <strong>변환</strong>을 클릭합니다.</li>
  <li><strong>JSON → CSV</strong>: JSON 객체 배열을 입력창에 붙여넣고 <strong>변환</strong>을 클릭합니다.</li>
  <li>결과를 <strong>복사</strong>하거나 다운로드합니다.</li>
</ol>

<h2>CSV와 JSON에 대하여</h2>
<p>CSV는 스프레드시트나 데이터베이스 내보내기에 자주 사용되고, JSON은 API와 웹 앱에서 널리 사용되는 데이터 형식입니다. 두 형식 간 변환은 데이터 처리·분석·마이그레이션 작업에서 자주 필요합니다.</p>
