<h2>使い方</h2>
<ol>
  <li>左パネルにJSONを貼り付けるか入力します。リアルタイムで検証されます。</li>
  <li>任意で<strong>ルートクラス名</strong>を設定します（デフォルト：<code>RootObject</code>）。</li>
  <li><strong>Kotlin を生成</strong>をクリックするか、リアルタイム変換を待ちます。</li>
  <li><strong>コピー</strong>をクリックして出力をクリップボードにコピーします。</li>
</ol>

<h2>型マッピング</h2>
<ul>
  <li><code>null</code> → <code>Any?</code></li>
  <li>文字列 → <code>String</code></li>
  <li>整数 → <code>Int</code></li>
  <li>浮動小数点数 → <code>Double</code></li>
  <li>真偽値 → <code>Boolean</code></li>
  <li>配列 → <code>List&lt;T&gt;</code></li>
  <li>ネストオブジェクト → 独立した <code>data class</code></li>
</ul>
