<h2>사용 방법</h2>
<ol>
  <li>JSON을 왼쪽 패널에 붙여넣거나 입력합니다. 실시간으로 유효성이 검사됩니다.</li>
  <li>선택적으로 <strong>루트 클래스 이름</strong>을 설정합니다 (기본값: <code>RootObject</code>).</li>
  <li><strong>Kotlin 생성</strong>을 클릭하거나 실시간 변환을 기다립니다.</li>
  <li><strong>복사</strong>를 클릭하여 출력을 클립보드에 복사합니다.</li>
</ol>

<h2>타입 매핑</h2>
<ul>
  <li><code>null</code> → <code>Any?</code></li>
  <li>문자열 → <code>String</code></li>
  <li>정수 → <code>Int</code></li>
  <li>부동소수점 → <code>Double</code></li>
  <li>불리언 → <code>Boolean</code></li>
  <li>배열 → <code>List&lt;T&gt;</code></li>
  <li>중첩 객체 → 독립적인 <code>data class</code></li>
</ul>
