<h2>How to Use</h2>
<ol>
  <li>Paste or type your JSON into the left panel. The tool validates it in real time.</li>
  <li>Optionally set a <strong>Root class name</strong> (default: <code>RootObject</code>).</li>
  <li>Click <strong>Generate Kotlin</strong> or wait for real-time conversion.</li>
  <li>Click <strong>Copy</strong> to copy the output to your clipboard.</li>
</ol>

<h2>Type Mapping</h2>
<ul>
  <li><code>null</code> → <code>Any?</code></li>
  <li>string → <code>String</code></li>
  <li>integer number → <code>Int</code></li>
  <li>floating-point number → <code>Double</code></li>
  <li>boolean → <code>Boolean</code></li>
  <li>array → <code>List&lt;T&gt;</code></li>
  <li>nested object → separate <code>data class</code></li>
</ul>

<h2>Example</h2>
<p>Given this JSON:</p>
<pre><code>{`{"user": {"name": "Alice", "age": 30}, "tags": ["admin"]}`}</code></pre>
<p>The tool generates:</p>
<pre><code>{`@Serializable\ndata class User(\n    val name: String = "",\n    val age: Int = 0\n)\n\n@Serializable\ndata class RootObject(\n    val user: User = User(),\n    val tags: List<String> = emptyList()\n)`}</code></pre>
