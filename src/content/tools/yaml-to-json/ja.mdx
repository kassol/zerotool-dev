<h2>使い方</h2>
<ol>
  <li><strong>YAML → JSON</strong>：左側のパネルにYAMLを貼り付け、<strong>YAML → JSON</strong>をクリックします。</li>
  <li><strong>JSON → YAML</strong>：右側のパネルにJSONを貼り付け、<strong>JSON → YAML</strong>をクリックします。</li>
  <li><strong>Ctrl+Enter</strong>（Macは<strong>Cmd+Enter</strong>）でショートカット変換できます。</li>
  <li><strong>コピー</strong>ボタンで結果をクリップボードにコピーします。</li>
</ol>

<h2>変換例</h2>

<h3>シンプルなオブジェクト</h3>
<p>入力YAML：</p>

```yaml
name: Alice
age: 30
active: true
```

<p>出力JSONにはname・age・activeフィールドが正しい型で含まれます。</p>

<h3>ネスト構造と配列</h3>
<p>入力YAML：</p>

```yaml
server:
  host: localhost
  port: 8080
  features:
    - auth
    - logging
    - cache
```

<p>出力JSONにはネストされたserverオブジェクトとfeatures配列が含まれます。</p>

<h2>YAMLとJSONの使い分け</h2>
<p>
  <strong>YAML</strong>はDocker Compose、Kubernetes、GitHub Actions、Ansible、Helm Chartsなど設定ファイルの標準形式です。
  <strong>JSON</strong>はREST APIやWebサービスのデータ交換形式として広く使われています。
  このツールで両形式を簡単に変換できます。
</p>
