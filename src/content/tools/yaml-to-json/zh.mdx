<h2>使用方法</h2>
<ol>
  <li><strong>YAML → JSON</strong>：将 YAML 数据粘贴到左侧面板，点击 <strong>YAML → JSON</strong> 按钮。</li>
  <li><strong>JSON → YAML</strong>：将 JSON 数据粘贴到右侧面板，点击 <strong>JSON → YAML</strong> 按钮。</li>
  <li>可使用 <strong>Ctrl+Enter</strong>（Mac 上为 <strong>Cmd+Enter</strong>）快捷键。</li>
  <li>点击面板上的 <strong>Copy</strong> 按钮，将结果复制到剪贴板。</li>
</ol>

<h2>转换示例</h2>

<h3>简单对象</h3>
<p>输入 YAML：</p>

```yaml
name: Alice
age: 30
active: true
```

<p>输出 JSON 包含正确类型的 name、age 和 active 字段。</p>

<h3>嵌套结构与数组</h3>
<p>输入 YAML：</p>

```yaml
server:
  host: localhost
  port: 8080
  features:
    - auth
    - logging
    - cache
```

<p>输出 JSON 包含嵌套的 server 对象，含 features 数组。</p>

<h2>YAML 与 JSON</h2>
<p>
  <strong>YAML</strong> 是配置文件的主流格式——Docker Compose、Kubernetes 清单、GitHub Actions 工作流、Ansible Playbook 和 Helm Chart 都使用它。
  <strong>JSON</strong> 是 REST API 和 Web 服务的标准交换格式。
  两者可以表示相同的数据结构。当您的工具需要不同格式时，此转换器可帮助您在两者之间切换。
</p>
