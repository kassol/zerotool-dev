<h2>사용 방법</h2>
<ol>
  <li>드롭다운에서 <strong>User-agent</strong>를 선택하거나(<code>*</code>는 모든 봇 대상) 직접 봇 이름을 입력합니다.</li>
  <li><strong>+ Allow</strong> 또는 <strong>+ Disallow</strong>를 클릭하여 경로 규칙을 추가합니다.</li>
  <li>봇별 설정이 필요하면 「+ Add User-agent Block」으로 블록을 추가합니다.</li>
  <li>Sitemap이 있다면 <strong>Sitemap URL</strong>을 입력합니다.</li>
  <li>robots.txt 내용이 실시간으로 업데이트됩니다 — <strong>Copy</strong>를 클릭해 복사하세요.</li>
</ol>

<h2>자주 쓰이는 robots.txt 패턴</h2>
<ul>
  <li><strong>모든 봇 차단:</strong> <code>User-agent: &#42;</code> + <code>Disallow: /</code></li>
  <li><strong>모든 봇 허용:</strong> <code>User-agent: *</code> + <code>Disallow:</code> (값 비워두기)</li>
  <li><strong>특정 디렉터리 차단:</strong> <code>Disallow: /admin/</code></li>
  <li><strong>Googlebot만 허용:</strong> <code>&#42;</code>를 차단한 뒤, Googlebot 블록을 별도 추가하고 <code>Disallow:</code>를 비워 두기</li>
</ul>

  <code>https://yourdomain.com/robots.txt</code>로 접근할 수 있도록 하세요.
