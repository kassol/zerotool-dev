<h2>よく使うHTMLエンティティ</h2>
<ul>
  <li><code>&amp;lt;</code> → &lt;（小なり記号）</li>
  <li><code>&amp;gt;</code> → &gt;（大なり記号）</li>
  <li><code>&amp;amp;</code> → &amp;（アンパサンド）</li>
  <li><code>&amp;quot;</code> → "（ダブルクォート）</li>
  <li><code>&amp;apos;</code> → '（シングルクォート）</li>
  <li><code>&amp;copy;</code> → ©（著作権記号）</li>
</ul>
