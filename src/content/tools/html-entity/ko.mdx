<h2>사용 방법</h2>
<ol>
  <li><strong>인코딩</strong>: HTML에 표시할 텍스트(&lt;, &gt; 등 포함)를 입력합니다.</li>
  <li><strong>디코딩</strong>: &lt;, &amp; 등의 엔티티를 포함한 HTML을 입력합니다.</li>
  <li>변환 결과를 <strong>복사</strong>합니다.</li>
</ol>

<h2>자주 사용하는 HTML 엔티티</h2>
<ul>
  <li><code>&amp;lt;</code> → &lt;(작다 기호)</li>
  <li><code>&amp;gt;</code> → &gt;(크다 기호)</li>
  <li><code>&amp;amp;</code> → &amp;(앰퍼샌드)</li>
  <li><code>&amp;quot;</code> → "(큰따옴표)</li>
  <li><code>&amp;apos;</code> → '(작은따옴표)</li>
  <li><code>&amp;copy;</code> → ©(저작권 기호)</li>
</ul>
