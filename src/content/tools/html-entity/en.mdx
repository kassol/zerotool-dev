faqItems:
  - question: "What are HTML entities?"
    answer: "HTML entities are special codes that represent characters which have meaning in HTML or are not available on a standard keyboard. For example, &lt; is written as &amp;lt; so the browser does not interpret it as an HTML tag."
  - question: "When should I encode HTML entities?"
    answer: "Encode whenever you display user-generated content, embed code snippets in HTML, or need to safely include special characters like <, >, &, and quotes inside HTML attributes or text nodes to prevent XSS attacks and rendering errors."
  - question: "What is the difference between named, decimal, and hex entities?"
    answer: "Named entities use a human-readable label (e.g. &amp;amp;). Decimal entities use the character's Unicode code point in base 10 (e.g. &amp;#38;). Hex entities use base 16 (e.g. &amp;#x26;). Browsers support all three forms."
  - question: "Is my data sent to any server?"
    answer: "No. All encoding and decoding happens entirely in your browser using JavaScript. No data leaves your machine."

<h2>How to Use</h2>
<ol>
  <li>Paste your text or HTML-encoded string into the <strong>Input</strong> box.</li>
  <li>Click <strong>Encode</strong> to convert special characters to HTML entities, or <strong>Decode</strong> to convert entities back to characters.</li>
  <li>Click <strong>Copy</strong> to copy the result to your clipboard.</li>
</ol>

<h2>Worked Examples</h2>

<h3>Encoding a code snippet for display</h3>
<p>
  You want to show <code>&lt;div class="box"&gt;</code> as visible text on a web page.
  Paste it into the input and click <strong>Encode</strong>. The output will be:
</p>
<pre><code>&amp;lt;div class=&amp;quot;box&amp;quot;&amp;gt;</code></pre>
<p>This safely renders as the literal HTML tag text in any browser.</p>

<h3>Decoding received HTML content</h3>
<p>
  You received a string from an API: <code>Tom &amp;amp; Jerry &amp;mdash; Season 1</code>.
  Paste it in and click <strong>Decode</strong>. The output will be:
</p>
<pre><code>Tom &amp; Jerry — Season 1</code></pre>

<h3>Encoding non-ASCII characters</h3>
<p>
  Paste <code>Caf&eacute; &copy; 2024</code> and click <strong>Encode</strong>.
  Non-ASCII characters are converted to numeric entities:
</p>
<pre><code>Caf&amp;#233; &amp;#169; 2024</code></pre>
