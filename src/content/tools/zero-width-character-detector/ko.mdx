<h2>제로 폭 문자 감지기 개요</h2>

<p>제로 폭 문자 감지기는 텍스트에서 화면에 표시되지 않는 Unicode 문자를 찾아 하이라이트 처리하고 제거하는 도구입니다. 복사-붙여넣기 오염, AI 생성 텍스트의 숨겨진 메타데이터, 공급망 공격 시도를 조사할 때 사용합니다.</p>

<p>텍스트를 붙여넣으면 모든 비가시 문자가 색상으로 구분되어 표시되며, 문자 종류별로 통계를 제공합니다. 한 번의 클릭으로 모든 비가시 문자를 제거하거나, 특정 범주(제로 폭만, 양방향만, 태그만)만 선택적으로 정제할 수 있습니다.</p>

<h2>감지 범위</h2>

<ul>
  <li><strong>제로 폭 공백(ZWSP, U+200B)</strong> — 너비가 없는 공백, 복사-붙여넣기 추적에 사용</li>
  <li><strong>제로 폭 비결합자(ZWNJ, U+200C)</strong> — 문자 결합을 방지하며 주로 아랍어·힌디 텍스트 처리</li>
  <li><strong>제로 폭 결합자(ZWJ, U+200D)</strong> — 문자 결합을 강제하며 이모지 변형 제어</li>
  <li><strong>바이트 순서 표시(BOM, U+FEFF)</strong> — UTF-8/UTF-16 파일의 바이트 순서 마커, 대부분 불필요</li>
  <li><strong>Word Joiner(U+2060)</strong> — 제로 폭이지만 줄 바꿈을 방지하는 특수 공백</li>
  <li><strong>양방향 제어 문자(U+200E, U+200F, U+202A-E, U+2066-2069)</strong> — 텍스트 방향성을 제어하며 Trojan-Source 공격의 핵심 벡터</li>
  <li><strong>소프트 하이픈(U+00AD)</strong> — 선택적 줄 바꿈 위치 표시, 주로 인쇄 용도</li>
  <li><strong>변형 선택자(U+FE00-FE0F, U+E0100-E01EF)</strong> — 문자 모양 변형 제어, 이모지와 CJK 문자에서 사용</li>
  <li><strong>태그 문자(U+E0000-E007F)</strong> — 메타데이터 인코딩 용도, ChatGPT 및 기타 LLM의 숨겨진 워터마크 범위</li>
</ul>

<ol>

  <li>모든 비가시 문자가 색상 코드로 하이라이트되며, 아래에 감지된 문자의 종류와 개수가 표시됩니다.</li>
  <li><strong>정제 모드</strong>를 선택합니다:<ul><li><strong>모두 제거</strong> — 모든 비가시 문자 삭제</li><li><strong>제로 폭만</strong> — ZWSP, ZWNJ, ZWJ, Word Joiner, 소프트 하이픈만 제거</li><li><strong>양방향만</strong> — U+200E, U+200F, U+202A-E, U+2066-2069만 제거</li><li><strong>태그만</strong> — U+E0000-E007F 태그 문자만 제거</li><li><strong>변형만</strong> — 변형 선택자만 제거</li></ul></li>
  <li><strong>정제</strong> 버튼을 클릭하면 선택된 범주의 비가시 문자가 제거됩니다.</li>
  <li><strong>복사</strong>로 결과를 클립보드에 복사하거나 <strong>.txt 다운로드</strong>로 파일을 저장합니다.</li>

<h2>주요 시나리오</h2>

<h3>AI 생성 텍스트 검증</h3>
<p>ChatGPT, Claude, Gemini 등의 모델은 탐지 회피 목적으로 생성 텍스트에 태그 문자를 숨기는 것으로 알려져 있습니다. 이 도구로 태그 문자(U+E0000-E007F)를 스캔하면 의도된 워터마크나 숨겨진 수정 기록을 감지할 수 있습니다.</p>

<h3>공급망 코드 리뷰</h3>
<p>소스 코드에 양방향 제어 문자(특히 U+202E RLO)가 숨겨져 있으면, 렌더링된 코드와 실제 실행 흐름이 달라지는 Trojan-Source 공격이 가능합니다(CVE-2021-42574). Pull Request 검토 시 이 도구로 미리 스캔하면 위험한 문자를 조기에 발견할 수 있습니다.</p>

<h3>피싱 메일 조사</h3>
<p>정교한 피싱 메일은 제로 폭 문자를 사용하여 표시 주소와 실제 링크를 가릴 수 있습니다. 의심스러운 이메일의 헤더나 본문을 분석하면 숨겨진 문자 조작을 감지할 수 있습니다.</p>

<h3>붙여넣기 정리</h3>
<p>Word, Google Docs, 블로그 플랫폼에서 복사한 텍스트는 종종 포매팅 유물(제로 폭 공백, 소프트 하이픈)을 포함합니다. 이 도구로 한 번에 정제하면 깔끔한 평문을 얻을 수 있습니다.</p>

<h3>텍스트 누출 조사</h3>
<p>특정 조직이 배포하는 민감한 문서(NDA, 기술 사양)에는 추적 목적으로 제로 폭 공백이 삽입되기도 합니다. 수신자가 누출자를 특정할 수 없도록 제거하기 위해 사용할 수 있습니다.</p>
