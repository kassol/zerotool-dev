<h2>零宽字符检测器</h2>
<p>
  文本中经常隐藏着肉眼看不见的 Unicode 字符——零宽字符、字节顺序标记、双向控制符、AI 隐写水印等。这些不可见字符可能来自复制粘贴、富文本编辑器、AI 生成内容或供应链攻击，造成难以排查的 bug、审计风险或数据泄露隐患。
</p>
<p>
  本工具实时扫描并高亮显示你粘贴的任何文本中的所有不可见字符，按类别统计，支持按需清除——全部删除或仅清除特定类型（零宽字符、双向控制字、标签字符等），一键复制或下载清洁文本。
</p>

<h2>检测范围</h2>
<p>本工具覆盖以下不可见 Unicode 字符类别：</p>
<ul>
  <li><strong>零宽字符</strong>（Zero-Width）：ZWSP（U+200B）、ZWNJ（U+200C）、ZWJ（U+200D）、Word Joiner（U+2060）</li>
  <li><strong>字节顺序标记</strong>（BOM）：U+FEFF，常在跨平台文件传输中出现</li>
  <li><strong>双向控制字符</strong>（Bidi Control）：LRE、RLE、PDF、LRO、RLO（U+202A-202E）、隐式方向标记（U+200E、U+200F）、隔离符（U+2066-2069），包括 Trojan-Source 攻击的常用向量</li>
  <li><strong>软连字符</strong>（Soft Hyphen）：U+00AD，在自动换行位置插入但无实际显示</li>
  <li><strong>字体变体选择符</strong>（Variation Selectors）：U+FE00-FE0F、U+E0100-E01EF，用于调整字形但不显示为字符</li>
  <li><strong>标签字符</strong>（Tag Characters）：U+E0000-E007F，AI 模型（如 ChatGPT）用于隐写水印的范围</li>
</ul>

  <li>点击「复制清洁文本」或「下载 .txt」获得结果。</li>
</ol>

<h2>常见场景</h2>
<ul>
  <li><strong>AI 输出审查</strong> — ChatGPT、Claude、Copilot 等生成的文本可能含隐写水印（Tag 字符），本工具可快速检出并清除。</li>
  <li><strong>供应链代码审查</strong> — 在接收第三方代码或依赖前扫描 Trojan-Source 攻击，特别是双向控制字符伪装的逻辑反转。</li>
  <li><strong>邮件反钉鱼</strong> — 复制自邮件、即时通讯的文本可能包含跟踪像素、格式控制符或恶意双向字符，清除后更安全。</li>
  <li><strong>复制粘贴清理</strong> — 从 Word、Google Docs、在线文章粘贴到纯文本编辑器时常残留隐形格式标记，本工具一键清洁。</li>
  <li><strong>文本泄密调查</strong> — 用 Tag 字符或其他不可见标记对敏感文档做隐式追踪，本工具帮你识别隐藏的元数据标签。</li>
</ul>

<h2>为什么需要纯客户端处理</h2>
<p>
  将含有隐形字符或水印的文本上传到服务器检测存在多重风险：
</p>
<ul>
  <li><strong>隐私泄露</strong> — 你的代码片段、邮件内容、AI 输出可能被服务端日志或分析系统捕获，违反组织敏感性要求。</li>
  <li><strong>Trojan-Source 样本无需外传</strong> — 接收的可疑代码不应离开本地环保，本地分析避免了病毒或恶意样本的传播风险。</li>
  <li><strong>完全离线可用</strong> — 本工具纯浏览器 JavaScript 运行，无需网络连接或账号登录，断网状态下也能工作。</li>
</ul>
