<h2>사용 방법</h2>
<ol>
  <li>해시화할 텍스트를 입력창에 붙여넣습니다.</li>
  <li>사용할 해시 알고리즘(MD5·SHA-1·SHA-256 등)을 선택합니다.</li>
  <li>해시 값이 즉시 표시됩니다.</li>
  <li><strong>복사</strong>로 해시 값을 클립보드에 복사합니다.</li>
</ol>

<h2>해시 알고리즘에 대하여</h2>
<p>이 도구는 5종류의 해시 알고리즘을 지원합니다:</p>
<ul>
  <li><strong>MD5</strong>(128비트): 하위 호환성을 위한 레거시 용도</li>
  <li><strong>SHA-1</strong>(160비트): 레거시 시스템과의 호환성</li>
  <li><strong>SHA-256</strong>(256비트): 일반적인 보안 용도에 권장</li>
  <li><strong>SHA-384</strong>(384비트): 높은 보안 요건</li>
  <li><strong>SHA-512</strong>(512비트): 최고 수준의 보안</li>
</ul>
