<h2>使用示例</h2>
<h3>推文长度检查</h3>
<p>粘贴推文草稿，查看<strong>字符数</strong>。X（原 Twitter）每条推文限制 280 个字符，超出则需精简。</p>

<h3>论文字数统计</h3>
<p>写 500 字的短文？粘贴草稿后关注<strong>字数</strong>计数器。同时可以检查段落结构——大多数文章至少需要 4-5 个段落。</p>

<h3>演讲计时</h3>
<p>准备 5 分钟的演讲？按每分钟 130 词计算，大约需要 650 个英文单词。粘贴讲稿后查看<strong>演讲时间</strong>是否达标。</p>

<h2>字符计数标准</h2>
<p>不同平台的字符计数方式有所不同。有些按 Unicode 码点计数（一个表情可能算 1 或 2 个），有些按 UTF-16 代码单元计数。本工具使用 JavaScript 字符串长度，即 UTF-16 代码单元——与浏览器和大多数 Web 平台一致。对于绝大多数拉丁、中日韩和西里尔文本，一个可见字符等于一个计数单位。</p>
