<h2>사용 방법</h2>
<ol>
  <li><strong>흐림</strong>·<strong>밝기</strong>·<strong>대비</strong>·<strong>회색조</strong>·<strong>색조 회전</strong>·<strong>반전</strong>·<strong>불투명도</strong>·<strong>채도</strong>·<strong>세피아</strong> 9개의 슬라이더 중 하나를 조정하면 미리보기 이미지에 실시간으로 효과가 반영됩니다.</li>
  <li><strong>이미지 업로드</strong>를 클릭하여 기본 미리보기 이미지를 원하는 이미지(JPEG, PNG, WebP, SVG 등 5MB 이하)로 교체할 수 있습니다. 이미지는 브라우저에서 완전히 처리되며 서버에 업로드되지 않습니다.</li>
  <li>각 슬라이더의 <strong>초기화</strong> 버튼으로 해당 필터만 기본값으로 되돌릴 수 있습니다. <strong>모두 초기화</strong>로 한 번에 모든 슬라이더를 리셋합니다.</li>
  <li><strong>복사</strong> 버튼으로 생성된 CSS를 복사합니다. 기본값과 다른 함수만 출력되며, 모두 기본값이면 <code>filter: none;</code>이 출력됩니다.</li>
</ol>

<h2>생성된 CSS 이해하기</h2>
<p>
  출력은 <code>.element</code> 클래스에 <code>filter</code> 속성을 설정합니다.
  <code>.element</code>를 원하는 선택자로 교체하여 이미지·div·canvas·video·가상 요소 등
  모든 HTML 요소에 적용할 수 있습니다. 필터 함수는 순서대로 평가되므로
  <code>blur(2px) brightness(120%)</code>는 먼저 흐리게 한 뒤 밝기를 높이는 순서로 처리됩니다.
</p>

<h2>주요 활용 사례</h2>
<p>
  CSS 필터는 이미지 호버 효과(기본 상태에서 회색조, 호버 시 컬러), 다크 모드 이미지 조정(반전 + 색조 회전),
  로딩 스켈레톤(회색조 + 밝기), 유리 카드 배경(흐림 적용), 세피아·고대비 등의 사진 스타일링에 널리 사용됩니다.
  필터는 GPU에서 처리되므로 대형 이미지나 복잡한 레이아웃에서도 우수한 성능을 발휘합니다.
</p>
