<h2>사용 방법</h2>
<ol>
  <li>이미지 파일을 드래그 앤 드롭하거나 <strong>파일 선택</strong>을 클릭합니다.</li>
  <li>Base64 문자열과 Data URI가 자동 생성됩니다.</li>
  <li>용도에 맞게 <strong>Base64만 복사</strong> 또는 <strong>Data URI 복사</strong>를 선택합니다.</li>
</ol>

<h2>이미지 Base64 변환의 활용</h2>
<ul>
  <li><strong>CSS/HTML에 인라인 삽입</strong>: 작은 아이콘이나 이미지를 파일 참조 없이 사용</li>
  <li><strong>이메일 첨부</strong>: MIME 형식의 이미지 삽입</li>
  <li><strong>JSON API 응답</strong>: 이미지 데이터를 JSON에 포함</li>
  <li><strong>오프라인 앱</strong>: 외부 파일 참조 없이 이미지를 사용</li>
</ul>
