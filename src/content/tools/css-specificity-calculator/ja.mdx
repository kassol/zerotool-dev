<h2>使い方</h2>
<ol>
  <li>入力欄にCSSセレクターを入力します（例: <code>#nav .item:hover</code>）。</li>
  <li>詳細度タプル <code>(a, b, c)</code> と色分けされた内訳が即座に表示されます。</li>
  <li>複数セレクターを比較するにはカンマで区切ります: <code>#nav .item, .item:hover</code>。</li>
  <li>各結果の<strong>コピー</strong>ボタンでタプルをクリップボードにコピーできます。</li>
</ol>

<h2>(a, b, c) タプルの読み方</h2>
<ul>
  <li><strong>a — ID</strong>: <code>#id</code> セレクター1つにつき +1。</li>
  <li><strong>b — クラス / 属性 / 疑似クラス</strong>: <code>.class</code>・<code>[attr]</code>・<code>:疑似クラス</code> 1つにつき +1。</li>
  <li><strong>c — 要素 / 疑似要素</strong>: タグ名または <code>::疑似要素</code> 1つにつき +1。</li>
</ul>
<p>左から順に比較します。最初に異なる列が勝敗を決めます。<code>(1,0,0)</code> は常に <code>(0,99,99)</code> に勝ちます。</p>

<h2>よくある例</h2>
<ul>
  <li><code>*</code> → <code>(0, 0, 0)</code></li>
  <li><code>p</code> → <code>(0, 0, 1)</code></li>
  <li><code>.class</code> → <code>(0, 1, 0)</code></li>
  <li><code>#id</code> → <code>(1, 0, 0)</code></li>
  <li><code>#id .class:hover</code> → <code>(1, 2, 0)</code></li>
  <li><code>div > p + span::before</code> → <code>(0, 0, 4)</code></li>
</ul>
