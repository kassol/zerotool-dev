<h2>사용 방법</h2>
<ol>
  <li>입력 필드에 CSS 선택자를 입력합니다. 예: <code>#nav .item:hover</code>.</li>
  <li>명시도 튜플 <code>(a, b, c)</code>와 색상별 분석이 즉시 표시됩니다.</li>
  <li>여러 선택자를 비교하려면 쉼표로 구분합니다: <code>#nav .item, .item:hover</code>.</li>
  <li>각 결과의 <strong>복사</strong> 버튼을 클릭하면 튜플이 클립보드에 복사됩니다.</li>
</ol>

<h2>(a, b, c) 튜플 읽기</h2>
<ul>
  <li><strong>a — ID</strong>: <code>#id</code> 선택자 하나당 +1.</li>
  <li><strong>b — 클래스 / 속성 / 가상 클래스</strong>: <code>.class</code>, <code>[attr]</code>, <code>:가상클래스</code> 하나당 +1.</li>
  <li><strong>c — 요소 / 가상 요소</strong>: 태그 이름 또는 <code>::가상요소</code> 하나당 +1.</li>
</ul>
<p>왼쪽부터 비교합니다. 처음으로 다른 열이 승패를 결정합니다. <code>(1,0,0)</code>은 항상 <code>(0,99,99)</code>를 이깁니다.</p>

<h2>일반적인 예시</h2>
<ul>
  <li><code>*</code> → <code>(0, 0, 0)</code></li>
  <li><code>p</code> → <code>(0, 0, 1)</code></li>
  <li><code>.class</code> → <code>(0, 1, 0)</code></li>
  <li><code>#id</code> → <code>(1, 0, 0)</code></li>
  <li><code>#id .class:hover</code> → <code>(1, 2, 0)</code></li>
  <li><code>div > p + span::before</code> → <code>(0, 0, 4)</code></li>
</ul>
