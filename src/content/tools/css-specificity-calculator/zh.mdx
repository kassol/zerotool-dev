<h2>使用方法</h2>
<ol>
  <li>在输入框中输入 CSS 选择器，例如 <code>#nav .item:hover</code>。</li>
  <li>优先级元组 <code>(a, b, c)</code> 和彩色分项说明会立即显示。</li>
  <li>如需对比多个选择器，用逗号分隔：<code>#nav .item, .item:hover</code>。</li>
  <li>点击任意结果旁的<strong>复制</strong>按钮，即可将元组复制到剪贴板。</li>
</ol>

<h2>理解 (a, b, c) 元组</h2>
<ul>
  <li><strong>a — ID</strong>：每个 <code>#id</code> 选择器加 1。</li>
  <li><strong>b — 类 / 属性 / 伪类</strong>：每个 <code>.class</code>、<code>[attr]</code> 或 <code>:伪类</code> 加 1。</li>
  <li><strong>c — 元素 / 伪元素</strong>：每个标签名或 <code>::伪元素</code> 加 1。</li>
</ul>
<p>从左到右对比。第一个不同的列决定胜负。<code>(1,0,0)</code> 始终胜过 <code>(0,99,99)</code>。</p>

<h2>常见示例</h2>
<ul>
  <li><code>*</code> → <code>(0, 0, 0)</code></li>
  <li><code>p</code> → <code>(0, 0, 1)</code></li>
  <li><code>.class</code> → <code>(0, 1, 0)</code></li>
  <li><code>#id</code> → <code>(1, 0, 0)</code></li>
  <li><code>#id .class:hover</code> → <code>(1, 2, 0)</code></li>
  <li><code>div > p + span::before</code> → <code>(0, 0, 4)</code></li>
</ul>
