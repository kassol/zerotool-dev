<h2>How to Use</h2>
<ol>
  <li>Paste your CSV data (with a header row) into the left panel.</li>
  <li>Click <strong>CSV → JSON</strong> to convert.</li>
  <li>Click <strong>Copy</strong> to copy the JSON result to your clipboard.</li>
  <li>The status line shows the row count or any parsing errors.</li>
</ol>

<h2>Worked Examples</h2>

<h3>Simple table</h3>
<p>Input CSV:</p>
<pre><code>name,age,city
Alice,30,London
Bob,25,Paris</code></pre>
<p>Output JSON is an array of objects with name, age, and city keys.</p>

<h3>Quoted fields with commas</h3>
<p>Input CSV:</p>
<pre><code>product,description,price
Widget,"Small, round widget",9.99
Gadget,"Includes ""turbo"" mode",24.50</code></pre>
<p>Fields containing commas or escaped quotes are handled correctly per RFC 4180.</p>

<h2>When to Use CSV to JSON</h2>
<p>
  Use this converter when you need to load spreadsheet or database export data into a web application or API.
  JSON is the standard interchange format for REST APIs and JavaScript applications, while CSV is the most common export format from Excel, Google Sheets, and databases.
  Converting CSV to JSON lets you use your tabular data directly in code without writing a parser.
</p>
