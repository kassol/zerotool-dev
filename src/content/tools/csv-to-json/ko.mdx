<h2>사용 방법</h2>
<ol>
  <li>헤더 행이 포함된 CSV 데이터를 왼쪽 패널에 붙여넣습니다.</li>
  <li><strong>CSV → JSON</strong>을 클릭하여 변환합니다.</li>
  <li><strong>복사</strong>를 클릭하여 JSON 결과를 클립보드에 복사합니다.</li>
  <li>상태 표시줄에 행 수 또는 파싱 오류가 표시됩니다.</li>
</ol>

<h2>변환 예시</h2>

<h3>간단한 테이블</h3>
<p>입력 CSV:</p>
<pre><code>name,age,city
Alice,30,London
Bob,25,Paris</code></pre>
<p>출력 JSON은 name, age, city 키를 가진 객체 배열입니다.</p>

<h3>쉼표가 포함된 따옴표 필드</h3>
<p>입력 CSV:</p>
<pre><code>product,description,price
Widget,"Small, round widget",9.99
Gadget,"Includes ""turbo"" mode",24.50</code></pre>
<p>쉼표나 이스케이프된 따옴표가 포함된 필드는 RFC 4180에 따라 올바르게 처리됩니다.</p>

<h2>활용 사례</h2>
<p>
  스프레드시트나 데이터베이스 내보내기 데이터를 웹 앱이나 API에 로드할 때 유용합니다.
  JSON은 REST API와 JavaScript 앱의 표준 형식이고, CSV는 Excel, Google 스프레드시트, 데이터베이스의 가장 일반적인 내보내기 형식입니다.
  CSV를 JSON으로 변환하면 파서를 작성하지 않고도 코드에서 직접 표 형식 데이터를 사용할 수 있습니다.
</p>
