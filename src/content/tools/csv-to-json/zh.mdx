<h2>使用方法</h2>
<ol>
  <li>将带表头的 CSV 数据粘贴到左侧面板。</li>
  <li>点击 <strong>CSV → JSON</strong> 按钮进行转换。</li>
  <li>点击 <strong>Copy</strong> 将 JSON 结果复制到剪贴板。</li>
  <li>状态栏会显示转换行数或解析错误信息。</li>
</ol>

<h2>转换示例</h2>

<h3>简单表格</h3>
<p>输入 CSV：</p>
<pre><code>name,age,city
Alice,30,London
Bob,25,Paris</code></pre>
<p>输出 JSON 为包含 name、age、city 键的对象数组。</p>

<h3>含逗号的引号字段</h3>
<p>输入 CSV：</p>
<pre><code>product,description,price
Widget,"Small, round widget",9.99
Gadget,"Includes ""turbo"" mode",24.50</code></pre>
<p>包含逗号或转义引号的字段按 RFC 4180 规范正确处理。</p>

<h2>使用场景</h2>
<p>
  当您需要将 Excel、Google 表格或数据库导出的数据加载到 Web 应用或 API 时，此工具非常有用。
  JSON 是 REST API 和 JavaScript 应用的标准格式，而 CSV 是最常见的表格导出格式。
  将 CSV 转换为 JSON 让您无需编写解析器即可在代码中直接使用表格数据。
</p>
