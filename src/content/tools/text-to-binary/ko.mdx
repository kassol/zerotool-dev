<h2>사용 방법</h2>
<ol>
  <li><strong>텍스트</strong> 패널에 텍스트를 입력하거나 붙여 넣으면 바이너리 출력이 자동으로 업데이트됩니다.</li>
  <li>바이너리를 텍스트로 변환하려면 <strong>바이너리</strong> 패널에 바이너리를 붙여 넣고 <strong>바이너리 → 텍스트</strong>를 클릭합니다.</li>
  <li>바이너리 출력 형식으로 <strong>공백 구분</strong> 또는 <strong>연속</strong>을 선택합니다.</li>
  <li><strong>복사</strong> 버튼으로 각 패널 내용을 클립보드에 복사합니다.</li>
</ol>

<h2>바이너리 표현</h2>

각 문자는 하나 이상의 바이트로 인코딩되며, 각 바이트는 8비트 이진수로 표현됩니다.

```
H = 01001000
e = 01100101
l = 01101100
o = 01101111
```

"Hello"의 공백 구분 바이너리: `01001000 01100101 01101100 01101100 01101111`

<h2>팁</h2>
<ul>
  <li>다중 바이트 UTF-8 문자(이모지 😀 = U+1F600 등)는 4개의 8비트 그룹이 됩니다.</li>
  <li>더 간결한 텍스트 안전 인코딩에는 <a href="/tools/base64">Base64</a>를 사용하세요.</li>
  <li>10진수, 16진수, 8진수 값을 다루려면 <a href="/tools/ascii-converter">ASCII 변환기</a>를 사용하세요.</li>
</ul>
