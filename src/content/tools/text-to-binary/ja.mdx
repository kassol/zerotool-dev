<h2>使い方</h2>
<ol>
  <li><strong>テキスト</strong>パネルにテキストを入力または貼り付けると、バイナリ出力が自動更新されます。</li>
  <li>バイナリをテキストに戻すには、<strong>バイナリ</strong>パネルにバイナリを貼り付けて<strong>バイナリ → テキスト</strong>をクリックします。</li>
  <li>バイナリ出力形式として<strong>スペース区切り</strong>または<strong>連続</strong>を選択します。</li>
  <li><strong>コピー</strong>ボタンで各パネルの内容をクリップボードにコピーできます。</li>
</ol>

<h2>バイナリ表現</h2>

各文字は1つ以上のバイトにエンコードされ、各バイトは8ビットの二進数で表されます。

```
H = 01001000
e = 01100101
l = 01101100
o = 01101111
```

"Hello"のスペース区切りバイナリ：`01001000 01100101 01101100 01101100 01101111`

<h2>ヒント</h2>
<ul>
  <li>マルチバイトUTF-8文字（絵文字😀 = U+1F600など）は4つの8ビットグループになります。</li>
  <li>よりコンパクトなテキストセーフなエンコードには <a href="/tools/base64/">Base64</a> をご利用ください。</li>
  <li>10進数・16進数・8進数の値を扱う場合は <a href="/tools/ascii-converter/">ASCII変換器</a> をご利用ください。</li>
</ul>
