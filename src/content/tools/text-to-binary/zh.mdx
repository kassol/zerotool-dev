<h2>使用方法</h2>
<ol>
  <li>在<strong>文本</strong>面板中输入或粘贴文本——二进制输出自动更新。</li>
  <li>要将二进制还原为文本，请在<strong>二进制</strong>面板中粘贴二进制内容，然后点击<strong>二进制 → 文本</strong>。</li>
  <li>选择二进制输出格式：<strong>空格分隔</strong>或<strong>连续</strong>。</li>
  <li>使用<strong>复制</strong>按钮将任一面板内容复制到剪贴板。</li>
</ol>

<h2>二进制表示</h2>

每个字符编码为一个或多个字节，每个字节用 8 位二进制数表示。

```
H = 01001000
e = 01100101
l = 01101100
o = 01101111
```

"Hello" 的空格分隔二进制：`01001000 01100101 01101100 01101100 01101111`

<h2>使用技巧</h2>
<ul>
  <li>多字节 UTF-8 字符（如 Emoji 😀 = U+1F600）会生成四个 8 位组。</li>
  <li>如需更紧凑的文本安全编码，请使用 <a href="/tools/base64">Base64</a>。</li>
  <li>如需处理十进制、十六进制和八进制值，请使用 <a href="/tools/ascii-converter">ASCII 转换器</a>。</li>
</ul>
