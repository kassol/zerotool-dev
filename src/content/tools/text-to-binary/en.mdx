<h2>Text to Binary Converter Online</h2>
<p>
  Convert plain text into binary <code>0</code>s and <code>1</code>s instantly, then decode binary back to readable text.
  The converter uses UTF-8, so it handles ASCII letters, numbers, punctuation, accented characters, CJK text, and emoji.
</p>
<ul>
  <li><strong>Text → Binary:</strong> turn each UTF-8 byte into an 8-bit binary group.</li>
  <li><strong>Binary → Text:</strong> paste spaced or continuous binary and decode it back to text.</li>
  <li><strong>Private by design:</strong> conversion runs locally in your browser; input is not uploaded.</li>
</ul>

<h2>How to Use</h2>
<ol>
  <li>Type or paste text in the <strong>Text</strong> panel — the binary output updates automatically.</li>
  <li>To convert binary back to text, paste binary into the <strong>Binary</strong> panel and click <strong>Binary → Text</strong>.</li>
  <li>Choose <strong>Spaced</strong> or <strong>Continuous</strong> format for the binary output.</li>
  <li>Use the <strong>Copy</strong> buttons to copy either panel to your clipboard.</li>
</ol>

<h2>Binary Representation</h2>

Each character is encoded as one or more bytes. Each byte is represented as an 8-bit binary number.

```
H = 01001000
e = 01100101
l = 01101100
o = 01101111
```

"Hello" in spaced binary: `01001000 01100101 01101100 01101100 01101111`
