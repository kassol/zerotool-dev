<h2>使用方法</h2>
<h3>Schema → 示例 JSON（默认模式）</h3>
<ol>
  <li>在左侧面板粘贴 <code>.proto</code> Schema。</li>
  <li>工具自动检测所有消息类型并填充下拉列表。</li>
  <li>选择一个消息类型，点击 <strong>生成 JSON</strong>。</li>
  <li>右侧出现带默认字段值的示例 JSON。</li>
</ol>

<h3>二进制 → JSON 模式</h3>
<ol>
  <li>使用顶部切换按钮切换到「二进制 → JSON」模式。</li>
  <li>在上方文本框粘贴 <code>.proto</code> Schema。</li>
  <li>在第二个文本框粘贴编码后的二进制数据（hex 或 base64）。</li>
  <li>选择消息类型，点击 <strong>解码二进制</strong>。</li>
</ol>

## Proto3 语法快速参考

```proto
syntax = "proto3";

package myapp;

message User {
  string id = 1;
  string name = 2;
  int32 age = 3;
  repeated string roles = 4;
  Address address = 5;
  UserStatus status = 6;
}

message Address {
  string street = 1;
  string city = 2;
  string country_code = 3;
}

enum UserStatus {
  UNKNOWN = 0;
  ACTIVE = 1;
  INACTIVE = 2;
}
```

<h2>字段类型参考</h2>
<ul>
  <li><strong>标量类型：</strong><code>double</code>、<code>float</code>、<code>int32</code>、<code>int64</code>、<code>uint32</code>、<code>uint64</code>、<code>bool</code>、<code>string</code>、<code>bytes</code></li>
  <li><strong>重复字段：</strong>在任意字段前加 <code>repeated</code>——对应 JSON 数组</li>
  <li><strong>嵌套消息：</strong>使用另一个消息类型作为字段类型</li>
  <li><strong>枚举：</strong>用 <code>enum</code> 定义，第一个值必须为 0</li>
  <li><strong>oneof：</strong>一组字段中同一时刻只能设置一个</li>
</ul>
