<h2>사용 방법</h2>
<h3>Schema → 샘플 JSON (기본 모드)</h3>
<ol>
  <li>왼쪽 패널에 <code>.proto</code> 스키마를 붙여넣습니다.</li>
  <li>도구가 모든 메시지 타입을 자동으로 감지하고 드롭다운에 표시합니다.</li>
  <li>메시지 타입을 선택하고 <strong>JSON 생성</strong>을 클릭합니다.</li>
  <li>기본 필드 값이 포함된 샘플 JSON이 오른쪽에 표시됩니다.</li>
</ol>

<h3>바이너리 → JSON 모드</h3>
<ol>
  <li>토글을 사용해 <strong>바이너리 → JSON</strong> 모드로 전환합니다.</li>
  <li>위쪽 텍스트 영역에 <code>.proto</code> 스키마를 붙여넣습니다.</li>
  <li>두 번째 텍스트 영역에 바이너리 데이터(hex 또는 base64)를 붙여넣습니다.</li>
  <li>메시지 타입을 선택하고 <strong>바이너리 디코드</strong>를 클릭합니다.</li>
</ol>

## Proto3 문법 빠른 참조

```proto
syntax = "proto3";

package myapp;

message User {
  string id = 1;
  string name = 2;
  int32 age = 3;
  repeated string roles = 4;
  Address address = 5;
  UserStatus status = 6;
}

message Address {
  string street = 1;
  string city = 2;
  string country_code = 3;
}

enum UserStatus {
  UNKNOWN = 0;
  ACTIVE = 1;
  INACTIVE = 2;
}
```

<h2>필드 타입 참조</h2>
<ul>
  <li><strong>스칼라 타입:</strong> <code>double</code>, <code>float</code>, <code>int32</code>, <code>int64</code>, <code>uint32</code>, <code>uint64</code>, <code>bool</code>, <code>string</code>, <code>bytes</code></li>
  <li><strong>반복 필드:</strong> 필드 앞에 <code>repeated</code>를 붙이면 JSON 배열로 매핑</li>
  <li><strong>중첩 메시지:</strong> 다른 메시지 타입을 필드 타입으로 사용</li>
  <li><strong>enum:</strong> <code>enum</code>으로 정의, 첫 번째 값은 반드시 0</li>
  <li><strong>oneof:</strong> 그룹 내 필드 중 한 번에 하나만 설정 가능</li>
</ul>
