<h2>사용 방법</h2>
<ol>
  <li>인코딩 또는 디코딩할 텍스트를 입력창에 붙여넣습니다.</li>
  <li><strong>인코딩</strong> 또는 <strong>디코딩</strong>을 클릭합니다.</li>
  <li><strong>복사</strong>로 결과를 클립보드에 복사합니다.</li>
</ol>

<h2>URL 인코딩에 대하여</h2>
<p>URL에서 사용할 수 있는 문자는 ASCII 문자의 일부에 한정됩니다. 공백, 한국어, 특수 기호 등은 퍼센트 인코딩으로 %XX 형식으로 변환해야 합니다.</p>
<p>폼 전송, API 요청 구성, URL 수작업 생성 등에서 자주 필요합니다.</p>
