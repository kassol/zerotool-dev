<h2>何时需要 URL 编码</h2>
<p>
  浏览器和服务器通过 URL 通信，而 URL 仅允许特定字符集。
  超出该字符集的内容——空格、Unicode 字符、<code>&amp;</code> 和 <code>=</code> 等运算符——必须经过百分号编码后才能包含在 URL 中。
</p>
<p>
  常见场景：传递包含特殊字符的查询参数、构造重定向 URL、编码表单数据、构建包含动态值的 REST API 调用。
</p>
