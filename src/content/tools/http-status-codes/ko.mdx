<h2>사용 방법</h2>
<p>
  검색 상자에 상태 코드 번호, 이름 키워드, 또는 설명을 입력하면 실시간으로 목록이 필터링됩니다. 검색을 비워두면 모든 코드를 카테고리별로 탐색할 수 있습니다.
</p>

<h2>상태 코드 카테고리</h2>
<ul>
  <li><strong>1xx 정보:</strong> 임시 응답 — 요청 수신됨, 서버가 아직 처리 중.</li>
  <li><strong>2xx 성공:</strong> 요청이 수신, 이해, 수락됨.</li>
  <li><strong>3xx 리다이렉션:</strong> 요청을 완료하려면 추가 조치 필요.</li>
  <li><strong>4xx 클라이언트 오류:</strong> 요청에 잘못된 구문이 있거나 처리 불가.</li>
  <li><strong>5xx 서버 오류:</strong> 서버가 유효한 요청 처리에 실패.</li>
</ul>
