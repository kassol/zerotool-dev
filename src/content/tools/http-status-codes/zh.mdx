<h2>使用方法</h2>
<p>
  在搜索框中输入状态码数字、名称关键字或描述，即可实时过滤列表。清空搜索框可按类别浏览全部状态码。

<h2>状态码分类</h2>
<ul>
  <li><strong>1xx 信息性：</strong>临时响应——请求已收到，服务器仍在处理中。</li>
  <li><strong>2xx 成功：</strong>请求已被接收、理解并处理完成。</li>
  <li><strong>3xx 重定向：</strong>需要进一步操作才能完成请求。</li>
  <li><strong>4xx 客户端错误：</strong>请求语法有误或无法完成。</li>
  <li><strong>5xx 服务器错误：</strong>服务器无法处理有效请求。</li>
</ul>

<h2>常用状态码速览</h2>
<ul>
  <li><strong>200 OK</strong> — 标准成功响应。</li>
  <li><strong>201 Created</strong> — 新资源已创建（常见于 POST/PUT API）。</li>
  <li><strong>204 No Content</strong> — 成功，无响应体（常见于 DELETE）。</li>
  <li><strong>301 Moved Permanently</strong> — SEO 安全的永久重定向。</li>
  <li><strong>400 Bad Request</strong> — 客户端发送了格式错误的数据。</li>
  <li><strong>401 Unauthorized</strong> — 需要身份验证。</li>
  <li><strong>403 Forbidden</strong> — 已验证但无权限。</li>
  <li><strong>404 Not Found</strong> — 资源不存在。</li>
  <li><strong>429 Too Many Requests</strong> — 超出请求频率限制。</li>
  <li><strong>500 Internal Server Error</strong> — 服务器意外故障。</li>
  <li><strong>503 Service Unavailable</strong> — 服务器宕机或过载。</li>
</ul>
