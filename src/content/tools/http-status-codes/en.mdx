<h2>Status Code Categories</h2>
<ul>
  <li><strong>1xx Informational:</strong> Provisional responses — request received, server is still processing.</li>
  <li><strong>2xx Success:</strong> The request was received, understood, and accepted.</li>
  <li><strong>3xx Redirection:</strong> Further action must be taken to complete the request.</li>
  <li><strong>4xx Client Error:</strong> The request contains bad syntax or cannot be fulfilled.</li>
  <li><strong>5xx Server Error:</strong> The server failed to fulfill a valid request.</li>
</ul>

<h2>Common Status Codes at a Glance</h2>
<ul>
  <li><strong>200 OK</strong> — Standard success response.</li>
  <li><strong>201 Created</strong> — New resource created (common in POST/PUT APIs).</li>
  <li><strong>204 No Content</strong> — Success, no response body (common in DELETE).</li>
  <li><strong>301 Moved Permanently</strong> — SEO-safe permanent redirect.</li>
  <li><strong>400 Bad Request</strong> — Client sent malformed data.</li>
  <li><strong>401 Unauthorized</strong> — Authentication required.</li>
  <li><strong>403 Forbidden</strong> — Authenticated but not permitted.</li>
  <li><strong>404 Not Found</strong> — Resource does not exist.</li>
  <li><strong>429 Too Many Requests</strong> — Rate limit exceeded.</li>
  <li><strong>500 Internal Server Error</strong> — Unexpected server failure.</li>
  <li><strong>503 Service Unavailable</strong> — Server down or overloaded.</li>
</ul>
