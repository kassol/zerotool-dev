<h2>사용 방법</h2>
<ol>
  <li>왼쪽 <strong>HTML 입력</strong> 패널에 HTML을 붙여넣거나 입력합니다.</li>
  <li>Markdown 출력이 오른쪽 패널에 실시간으로 표시됩니다(버튼 클릭 불필요).</li>
  <li><strong>복사</strong>를 클릭하여 클립보드에 복사하거나 <strong>.md 다운로드</strong>로 파일을 저장합니다.</li>
  <li><strong>예시</strong> 버튼으로 주요 변환 예제를 불러올 수 있습니다.</li>
</ol>

<h2>변환 참조표</h2>

<table>
  <thead><tr><th>HTML</th><th>Markdown</th></tr></thead>
  <tbody>
    <tr><td><code>&lt;h1&gt;제목&lt;/h1&gt;</code></td><td><code># 제목</code></td></tr>
    <tr><td><code>&lt;strong&gt;텍스트&lt;/strong&gt;</code></td><td><code>**텍스트**</code></td></tr>
    <tr><td><code>&lt;em&gt;텍스트&lt;/em&gt;</code></td><td><code>*텍스트*</code></td></tr>
    <tr><td><code>&lt;a href="url"&gt;링크&lt;/a&gt;</code></td><td><code>[링크](url)</code></td></tr>
    <tr><td><code>&lt;pre&gt;&lt;code class="language-js"&gt;</code></td><td>` ```js ` 펜스 블록</td></tr>
    <tr><td><code>&lt;table&gt;</code></td><td>GFM 파이프 표</td></tr>
    <tr><td><code>&lt;blockquote&gt;</code></td><td><code>&gt; 인용</code></td></tr>
  </tbody>
</table>

<h2>주요 사용 사례</h2>
<ul>
  <li><strong>CMS 콘텐츠 마이그레이션</strong>: WordPress HTML 콘텐츠를 Markdown으로 변환하여 Hugo, Astro 등 정적 사이트 생성기에 활용.</li>
  <li><strong>README 정리</strong>: 웹 페이지를 스크랩한 후 Markdown으로 변환하여 GitHub README나 문서 사이트에 활용.</li>
  <li><strong>LLM 입력 최적화</strong>: 웹 페이지를 Markdown으로 변환하여 토큰 수와 노이즈를 줄인 후 대규모 언어 모델에 입력.</li>
</ul>
