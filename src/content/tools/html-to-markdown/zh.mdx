<h2>使用方法</h2>
<ol>
  <li>将 HTML 代码粘贴或输入到左侧 <strong>HTML 输入</strong> 面板。</li>
  <li>Markdown 输出会即时显示在右侧面板，无需点击任何按钮。</li>
  <li>点击 <strong>复制</strong> 将 Markdown 复制到剪贴板，或点击 <strong>下载 .md</strong> 保存文件。</li>
  <li>点击 <strong>示例</strong> 按钮可加载涵盖主要转换场景的示例。</li>
</ol>

<h2>常见使用场景</h2>
<ul>
  <li><strong>CMS 内容迁移</strong>：将 WordPress 或 Drupal 的 HTML 内容转换为 Markdown，用于 Hugo、Astro 等静态站生成器。</li>
  <li><strong>README 整理</strong>：抓取网页内容后转换为 Markdown，用于 GitHub README 或文档站点。</li>
  <li><strong>LLM 输入优化</strong>：将网页转换为 Markdown 格式，减少 token 数量和噪声，再输入大语言模型。</li>
  <li><strong>笔记整理</strong>：将邮件 HTML 或富文本片段转换为 Markdown，导入 Obsidian、Notion 等笔记工具。</li>
</ul>
