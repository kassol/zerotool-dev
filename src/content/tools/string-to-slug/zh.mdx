<h2>使用方法</h2>
<ol>
  <li>在<strong>输入文本</strong>框中输入或粘贴任意文本，Slug 会实时生成。</li>
  <li>切换<strong>小写</strong>选项控制大小写。</li>
  <li>选择<strong>连字符</strong>或<strong>下划线</strong>作为单词分隔符。</li>
  <li>点击<strong>复制</strong>将结果复制到剪贴板。</li>
</ol>

<h2>Slug 规则</h2>

规范的 URL Slug 遵循以下约定：

- 仅使用小写字母（推荐）
- 单词之间用连字符或下划线分隔
- 不含空格、标点或特殊字符
- 不以分隔符开头或结尾

```
输入：  "你好世界 Hello World!"
Slug：  "hello-world"
```
