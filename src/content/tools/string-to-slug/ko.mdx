<h2>사용 방법</h2>
<ol>
  <li><strong>입력 텍스트</strong> 상자에 텍스트를 입력하거나 붙여넣으면 슬러그가 실시간으로 생성됩니다.</li>
  <li><strong>소문자</strong> 옵션으로 대소문자를 제어합니다.</li>
  <li>단어 구분자로 <strong>하이픈</strong> 또는 <strong>밑줄</strong>을 선택합니다.</li>
  <li><strong>복사</strong> 버튼으로 결과를 클립보드에 복사합니다.</li>
</ol>

<h2>슬러그 규칙</h2>

올바른 URL 슬러그는 다음 규칙을 따릅니다:

- 소문자만 사용 (일관성을 위해 권장)
- 단어 사이는 하이픈 또는 밑줄로 구분
- 공백, 구두점, 특수 문자 없음
- 구분자로 시작하거나 끝나지 않음

```
입력:  "Hello World! Ça va?"
슬러그: "hello-world-ca-va"
```
