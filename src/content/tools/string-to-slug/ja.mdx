<h2>使い方</h2>
<ol>
  <li><strong>入力テキスト</strong>ボックスにテキストを入力または貼り付けると、リアルタイムでスラグが生成されます。</li>
  <li><strong>小文字</strong>オプションでケースを制御します。</li>
  <li>単語の区切りとして<strong>ハイフン</strong>または<strong>アンダースコア</strong>を選択します。</li>
  <li><strong>コピー</strong>ボタンで結果をクリップボードにコピーします。</li>
</ol>

<h2>スラグのルール</h2>

正しいURLスラグは次の規則に従います：

- 小文字のみ（一貫性のために推奨）
- 単語はハイフンまたはアンダースコアで区切る
- スペース・記号・特殊文字は含まない
- 区切り文字で始まったり終わったりしない

```
入力:  "Hello World! Ça va?"
スラグ: "hello-world-ca-va"
```

  <li>人間が読めるタイトルからのファイル・フォルダ名生成</li>
  <li>ユーザー入力から一貫した識別子を生成</li>
  <li>SEOフレンドリーなURLパスの作成</li>
</ul>
