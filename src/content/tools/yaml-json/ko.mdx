<h2>사용 방법</h2>
<ol>
  <li><strong>YAML → JSON</strong>: YAML을 입력창에 붙여넣고 <strong>변환</strong>을 클릭합니다.</li>
  <li><strong>JSON → YAML</strong>: JSON을 입력창에 붙여넣고 <strong>변환</strong>을 클릭합니다.</li>
  <li>결과를 <strong>복사</strong>합니다.</li>
</ol>

<h2>YAML과 JSON의 구분 사용</h2>
<p>YAML은 Docker Compose, Kubernetes, GitHub Actions, Ansible, Helm Charts 등의 설정 파일에 널리 사용됩니다. JSON은 REST API, 웹 브라우저, 프로그램 간 데이터 교환에 적합합니다. 두 형식 간 변환이 필요한 상황에서 활용하십시오.</p>
