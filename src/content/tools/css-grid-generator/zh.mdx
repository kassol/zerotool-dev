faqItems:
  - question: "什么是 CSS Grid？"
    answer: "CSS Grid 是 CSS 内置的二维布局系统，可以同时定义行和列，并将元素精确放置在网格容器中。适用于页面级布局、卡片网格、图片画廊以及任何需要水平和垂直结构的 UI。"
  - question: "grid-template-columns 有什么用？"
    answer: "grid-template-columns 定义网格的列数和各列大小。生成器默认填充 repeat(N, 1fr)——N 列等宽，每列占用可用空间的一个分数单位。你也可以自定义，例如 200px 1fr 2fr 或 auto-fill minmax(200px, 1fr)。"
  - question: "column-gap 和 row-gap 有什么区别？"
    answer: "column-gap 设置列之间的间距，row-gap 设置行之间的间距，共同控制网格内部的沟槽。也可以用简写 gap: row-gap column-gap 一次设置。"
  - question: "1fr 单位是什么意思？"
    answer: "fr 是分数单位（fractional unit），按比例分配网格容器中的剩余空间。例如 1fr 2fr 1fr 创建三列，中间列宽是其他两列的两倍。它是响应式网格布局最常用的单位。"
  - question: "如何不用媒体查询实现响应式网格？"
    answer: "使用 auto-fill 或 auto-fit 配合 minmax()：grid-template-columns: repeat(auto-fill, minmax(200px, 1fr))。这会自动创建尽可能多的列（每列至少 200px），并伸展填满剩余空间——无需媒体查询。"

<h2>使用方法</h2>
<ol>
  <li>设置<strong>列数</strong>和<strong>行数</strong>——预览和模板输入框会自动更新。</li>
  <li>调整<strong>列间距</strong>和<strong>行间距</strong>以控制单元格之间的沟槽。</li>
  <li>可以用任意自定义值覆盖<strong>grid-template-columns</strong>或<strong>grid-template-rows</strong>（如 <code>200px 1fr auto</code>）。</li>
  <li>实时<strong>预览</strong>会即时反映每次更改。</li>
  <li>点击<strong>复制</strong>将生成的 <code>.container &#123; ... &#125;</code> 代码块复制到剪贴板。</li>
</ol>

<h2>理解生成的 CSS</h2>
<p>
  生成器输出一条 <code>.container</code> 规则，包含五个属性：
  <code>display: grid</code>、<code>grid-template-columns</code>、<code>grid-template-rows</code>、
  <code>column-gap</code> 和 <code>row-gap</code>。粘贴到样式表后，将 <code>.container</code> 改为你自己的类名即可。
</p>

<h2>常见网格模式</h2>
<ul>
  <li><strong>等宽列</strong> — <code>repeat(3, 1fr)</code> 创建三列等宽布局。</li>
  <li><strong>侧边栏布局</strong> — <code>250px 1fr</code> 固定侧边栏 + 弹性主区域。</li>
  <li><strong>圣杯布局</strong> — <code>200px 1fr 200px</code> 配合行定义，实现头部/底部固定。</li>
  <li><strong>响应式磁贴</strong> — <code>repeat(auto-fill, minmax(180px, 1fr))</code> 在任意屏幕宽度下自动换行。</li>
</ul>
