<h2>使い方</h2>
<ol>
  <li><strong>列数</strong>と<strong>行数</strong>を設定します — プレビューとテンプレート入力が自動更新されます。</li>
  <li><strong>列ギャップ</strong>と<strong>行ギャップ</strong>を調整してセル間のガターを制御します。</li>
  <li><strong>grid-template-columns</strong>または<strong>grid-template-rows</strong>を任意の値（例：<code>200px 1fr auto</code>）で上書きできます。</li>
  <li>ライブ<strong>プレビュー</strong>が変更を即座に反映します。</li>
  <li><strong>コピー</strong>をクリックして生成された <code>.container &#123; ... &#125;</code> ブロックをクリップボードにコピーします。</li>
</ol>

<h2>生成されるCSSについて</h2>
<p>
  ジェネレーターは <code>.container</code> ルールに5つのプロパティを出力します：
  <code>display: grid</code>、<code>grid-template-columns</code>、<code>grid-template-rows</code>、
  <code>column-gap</code>、<code>row-gap</code>。スタイルシートに貼り付け、
  <code>.container</code> を自分のクラス名に変更してください。
</p>

<h2>よく使われるグリッドパターン</h2>
<ul>
  <li><strong>等幅列</strong> — <code>repeat(3, 1fr)</code> で3列等幅レイアウト。</li>
  <li><strong>サイドバーレイアウト</strong> — <code>250px 1fr</code> で固定サイドバー＋フレキシブルメインエリア。</li>
  <li><strong>ホーリーグレイル</strong> — <code>200px 1fr 200px</code> に行定義を加えてヘッダー/フッター固定。</li>
  <li><strong>レスポンシブタイル</strong> — <code>repeat(auto-fill, minmax(180px, 1fr))</code> で任意の画面幅に自動対応。</li>
</ul>
