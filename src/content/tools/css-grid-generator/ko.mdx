faqItems:
  - question: "CSS Grid란 무엇인가요?"
    answer: "CSS Grid는 CSS에 내장된 2차원 레이아웃 시스템입니다. 행과 열을 동시에 정의하고 요소를 그리드 컨테이너 내에 정밀하게 배치할 수 있습니다. 페이지 레이아웃, 카드 그리드, 이미지 갤러리, 수평·수직 구조가 모두 필요한 UI에 적합합니다."
  - question: "grid-template-columns는 무엇을 하나요?"
    answer: "grid-template-columns는 그리드의 열 수와 크기를 정의합니다. 생성기는 기본값으로 repeat(N, 1fr)을 설정합니다. 이는 N개의 동일한 너비의 열로, 각 열이 가용 공간을 균등하게 분배합니다. 200px 1fr 2fr이나 auto-fill minmax(200px, 1fr) 같은 임의의 값으로 재정의할 수 있습니다."
  - question: "column-gap과 row-gap의 차이는 무엇인가요?"
    answer: "column-gap은 열 사이의 간격, row-gap은 행 사이의 간격을 설정하여 그리드 내부의 거터를 제어합니다. gap: row-gap column-gap 단축 속성으로 한 번에 설정할 수도 있습니다."
  - question: "1fr 단위란 무엇인가요?"
    answer: "fr은 분수 단위(fractional unit)로, 그리드 컨테이너의 남은 공간을 비율에 따라 분배합니다. 예를 들어 1fr 2fr 1fr은 3열을 만들며 가운데 열이 나머지 두 열의 2배 너비가 됩니다. 반응형 그리드 레이아웃에서 가장 많이 사용되는 단위입니다."
  - question: "미디어 쿼리 없이 반응형 그리드를 만들 수 있나요?"
    answer: "auto-fill 또는 auto-fit을 minmax()와 함께 사용합니다: grid-template-columns: repeat(auto-fill, minmax(200px, 1fr)). 최소 크기에 맞게 최대한 많은 열을 자동으로 생성하고 남은 공간을 늘려 채웁니다. 미디어 쿼리가 필요 없습니다."

<h2>사용 방법</h2>
<ol>
  <li><strong>열 수</strong>와 <strong>행 수</strong>를 설정합니다 — 미리보기와 템플릿 입력이 자동으로 업데이트됩니다.</li>
  <li><strong>열 간격</strong>과 <strong>행 간격</strong>을 조정하여 셀 사이의 거터를 제어합니다.</li>
  <li><strong>grid-template-columns</strong> 또는 <strong>grid-template-rows</strong>를 임의의 값(예: <code>200px 1fr auto</code>)으로 재정의할 수 있습니다.</li>
  <li>라이브 <strong>미리보기</strong>가 모든 변경 사항을 즉시 반영합니다.</li>
  <li><strong>복사</strong>를 클릭하여 생성된 <code>.container &#123; ... &#125;</code> 블록을 클립보드에 복사합니다.</li>
</ol>

<h2>생성된 CSS 이해하기</h2>
<p>
  생성기는 다섯 가지 속성을 가진 <code>.container</code> 규칙을 출력합니다:
  <code>display: grid</code>, <code>grid-template-columns</code>, <code>grid-template-rows</code>,
  <code>column-gap</code>, <code>row-gap</code>. 스타일시트에 붙여넣고
  <code>.container</code>를 자신의 클래스명으로 변경하세요.
</p>

<h2>자주 사용되는 그리드 패턴</h2>
<ul>
  <li><strong>동일 너비 열</strong> — <code>repeat(3, 1fr)</code>로 3열 동일 너비 레이아웃.</li>
  <li><strong>사이드바 레이아웃</strong> — <code>250px 1fr</code>로 고정 사이드바 + 유동 메인 영역.</li>
  <li><strong>성배 레이아웃</strong> — <code>200px 1fr 200px</code>에 행 정의를 추가해 헤더/푸터 고정.</li>
  <li><strong>반응형 타일</strong> — <code>repeat(auto-fill, minmax(180px, 1fr))</code>로 모든 화면 너비에 자동 대응.</li>
</ul>
