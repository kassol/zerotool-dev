<h2>使用方法</h2>
<ol>
  <li>将 HTML 粘贴到左侧文本框。</li>
  <li>JSX 输出会在右侧文本框中即时出现。</li>
  <li>点击<strong>复制</strong>将 JSX 输出复制到剪贴板。</li>
</ol>

<h2>转换规则</h2>

| HTML | JSX |
|------|-----|
| `class="..."` | `className="..."` |
| `for="..."` | `htmlFor="..."` |
| `style="color:red"` | `style={{ color: 'red' }}` |
| `onclick="..."` | `onClick="..."` |
| `onchange="..."` | `onChange="..."` |
| `tabindex` | `tabIndex` |
| `readonly` | `readOnly` |
| `maxlength` | `maxLength` |
| `colspan` | `colSpan` |
| `rowspan` | `rowSpan` |
| `crossorigin` | `crossOrigin` |
| `contenteditable` | `contentEditable` |
| `<br>` | `<br />` |
| `<img src="...">` | `<img src="..." />` |
| `<input ...>` | `<input ... />` |
| `<!-- 注释 -->` | `{/* 注释 */}` |
