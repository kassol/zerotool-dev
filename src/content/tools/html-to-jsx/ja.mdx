<h2>使い方</h2>
<ol>
  <li>左側のテキストエリアにHTMLを貼り付けます。</li>
  <li>入力と同時に右側のテキストエリアにJSX出力が即座に表示されます。</li>
  <li><strong>コピー</strong>をクリックしてJSX出力をクリップボードにコピーします。</li>
</ol>

<h2>変換ルール</h2>

| HTML | JSX |
|------|-----|
| `class="..."` | `className="..."` |
| `for="..."` | `htmlFor="..."` |
| `style="color:red"` | `style={{ color: 'red' }}` |
| `onclick="..."` | `onClick="..."` |
| `onchange="..."` | `onChange="..."` |
| `tabindex` | `tabIndex` |
| `readonly` | `readOnly` |
| `maxlength` | `maxLength` |
| `colspan` | `colSpan` |
| `rowspan` | `rowSpan` |
| `crossorigin` | `crossOrigin` |
| `contenteditable` | `contentEditable` |
| `<br>` | `<br />` |
| `<img src="...">` | `<img src="..." />` |
| `<input ...>` | `<input ... />` |
| `<!-- コメント -->` | `{/* コメント */}` |

<h2>主な用途</h2>
<ul>
  <li>静的HTMLテンプレートをReactコンポーネントに移行する</li>
  <li>デザインツール（Figma、Webflow）からエクスポートしたHTMLスニペットをJSXファイルに貼り付ける</li>
  <li>HTMLとJSXの属性名の違いを学習する</li>
  <li>HTMLメールテンプレートをReactメールライブラリ用に変換する</li>
  <li>サードパーティのHTMLウィジェットをReactプロジェクトに素早く適応させる</li>
</ul>
