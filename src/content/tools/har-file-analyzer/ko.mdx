<h2>사용 방법</h2>
<ol>
  <li>브라우저 DevTools의 Network 패널에서 세션을 <strong>.har</strong>로 내보냅니다.</li>
  <li>파일을 드롭존에 끌어다 놓거나 클릭하여 선택합니다. 파싱은 브라우저 안에서 진행됩니다.</li>
  <li><strong>개요</strong>(통계) / <strong>워터폴</strong>(타이밍) / <strong>상세</strong>(헤더·쿠키·본문) / <strong>필터 및 내보내기</strong>(부분 다운로드) 사이를 전환합니다.</li>
  <li>워터폴의 임의 행을 클릭하면 해당 요청의 상세 화면으로 바로 이동합니다.</li>
  <li><strong>리셋</strong>으로 메모리에 있는 HAR 데이터를 지웁니다.</li>
</ol>

<h2>워터폴 단계</h2>
<p>워터폴은 HAR 1.2 <code>timings</code>를 6단계로 그립니다:</p>
<ul>
  <li><strong>대기 / Stalled</strong> — <code>blocked</code></li>
  <li><strong>DNS 조회</strong> — <code>dns</code></li>
  <li><strong>초기 연결</strong> — <code>connect</code> (HAR 1.2에서 <code>ssl</code> 포함)</li>
  <li><strong>요청 전송</strong> — <code>send</code></li>
  <li><strong>대기 (TTFB)</strong> — <code>wait</code></li>
  <li><strong>콘텐츠 다운로드</strong> — <code>receive</code></li>
</ul>

<h2>개인정보 보호</h2>
<p>HAR에는 Authorization, Cookie, 응답 본문 등 민감한 정보가 자주 포함됩니다. 호스팅된 분석기에 업로드하는 것은 위험합니다. ZeroTool은 <code>FileReader</code> + <code>JSON.parse</code>로 브라우저 내에서 처리하며 업로드도 리로드를 넘어선 저장도 하지 않습니다.</p>
