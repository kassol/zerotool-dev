<h2>使用方法</h2>
<ol>
  <li>打开浏览器 DevTools，进入 Network 面板，将会话导出为 <strong>.har</strong> 文件。</li>
  <li>将文件拖到拖拽区（或点击选择）。解析过程在你的浏览器内完成。</li>
  <li>在 <strong>概览</strong>（统计）、<strong>瀑布</strong>（时序）、<strong>详情</strong>（Header / Cookie / Body）、<strong>筛选与导出</strong>（裁切下载）之间切换。</li>
  <li>点击任意瀑布行可直接跳到该请求的详情视图。</li>
  <li>点击 <strong>重置</strong> 清除内存中的 HAR 数据。</li>
</ol>

<h2>瀑布阶段</h2>
<p>瀑布按 HAR 1.2 <code>timings</code> 字段分 6 段绘制：</p>
<ul>
  <li><strong>排队 / Stalled</strong> — <code>blocked</code></li>
  <li><strong>DNS 解析</strong> — <code>dns</code></li>
  <li><strong>建立连接</strong> — <code>connect</code>（按 HAR 1.2 已包含 <code>ssl</code>）</li>
  <li><strong>请求发送</strong> — <code>send</code></li>
  <li><strong>等待（TTFB）</strong> — <code>wait</code></li>
  <li><strong>内容下载</strong> — <code>receive</code></li>
</ul>

<h2>隐私</h2>
<p>HAR 文件经常包含 Authorization、Cookie、响应体等敏感信息，上传到云端分析器有风险。ZeroTool 使用 <code>FileReader</code> + <code>JSON.parse</code> 在浏览器中完成解析，不上传、不持久化，刷新即清空。</p>
