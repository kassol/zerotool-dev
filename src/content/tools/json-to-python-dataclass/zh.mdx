<h2>使用方法</h2>
<ol>
  <li>将 JSON 粘贴或输入到左侧面板，工具会实时验证。</li>
  <li>可选设置<strong>根类名称</strong>（默认：<code>Root</code>）。</li>
  <li>选择输出模式：<strong>@dataclass</strong>、<strong>Pydantic v2</strong> 或 <strong>TypedDict</strong>。</li>
  <li>点击<strong>生成 Python</strong>，或等待实时转换结果。</li>
  <li>点击<strong>复制</strong>将输出复制到剪贴板，或点击<strong>下载 .py</strong> 保存为文件。</li>
</ol>

<h2>类型映射</h2>
<ul>
  <li><code>null</code> → <code>Optional[Any] = None</code></li>
  <li>字符串 → <code>str</code></li>
  <li>整数 → <code>int</code></li>
  <li>浮点数 → <code>float</code></li>
  <li>布尔值 → <code>bool</code></li>
  <li>数组 → <code>List[T]</code></li>
  <li>混合数组 → <code>List[Union[T1, T2]]</code></li>
  <li>嵌套对象 → 独立类</li>
</ul>

<h2>示例</h2>
<p>给定如下 JSON：</p>
<pre><code>{`{"user": {"first_name": "Alice", "age": 30}, "tags": ["admin"], "note": null}`}</code></pre>
<p>@dataclass 模式下生成：</p>
<pre><code>{`from dataclasses import dataclass\nfrom typing import Any, List, Optional\n\n@dataclass\nclass User:\n    first_name: str\n    age: int\n\n@dataclass\nclass Root:\n    user: User\n    tags: List[str]\n    note: Optional[Any] = None`}</code></pre>
