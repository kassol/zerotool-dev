<h2>How to Use</h2>
<ol>
  <li>Paste or type your JSON into the left panel. The tool validates it in real time.</li>
  <li>Optionally set a <strong>Root class name</strong> (default: <code>Root</code>).</li>
  <li>Choose an output mode: <strong>@dataclass</strong>, <strong>Pydantic v2</strong>, or <strong>TypedDict</strong>.</li>
  <li>Click <strong>Generate Python</strong> or wait for real-time conversion.</li>
  <li>Click <strong>Copy</strong> to copy the output, or <strong>Download .py</strong> to save as a file.</li>
</ol>

<h2>Type Mapping</h2>
<ul>
  <li><code>null</code> → <code>Optional[Any] = None</code></li>
  <li>string → <code>str</code></li>
  <li>integer → <code>int</code></li>
  <li>float → <code>float</code></li>
  <li>boolean → <code>bool</code></li>
  <li>array → <code>List[T]</code></li>
  <li>mixed array → <code>List[Union[T1, T2]]</code></li>
  <li>nested object → separate class</li>
</ul>

<h2>Example</h2>
<p>Given this JSON:</p>
<pre><code>{`{"user": {"first_name": "Alice", "age": 30}, "tags": ["admin"], "note": null}`}</code></pre>
<p>The tool generates (@dataclass mode):</p>
<pre><code>{`from dataclasses import dataclass\nfrom typing import Any, List, Optional\n\n@dataclass\nclass User:\n    first_name: str\n    age: int\n\n@dataclass\nclass Root:\n    user: User\n    tags: List[str]\n    note: Optional[Any] = None`}</code></pre>
