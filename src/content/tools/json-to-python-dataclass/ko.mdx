<h2>사용 방법</h2>
<ol>
  <li>왼쪽 패널에 JSON을 붙여넣거나 입력합니다. 도구가 실시간으로 유효성을 검사합니다.</li>
  <li>선택적으로 <strong>루트 클래스 이름</strong>을 설정합니다 (기본값: <code>Root</code>).</li>
  <li>출력 모드 선택: <strong>@dataclass</strong>, <strong>Pydantic v2</strong>, 또는 <strong>TypedDict</strong>.</li>
  <li><strong>Python 생성</strong>을 클릭하거나 실시간 변환을 기다립니다.</li>
  <li><strong>복사</strong>를 클릭해 출력을 복사하거나, <strong>.py 다운로드</strong>로 파일로 저장합니다.</li>
</ol>

<h2>타입 매핑</h2>
<ul>
  <li><code>null</code> → <code>Optional[Any] = None</code></li>
  <li>문자열 → <code>str</code></li>
  <li>정수 → <code>int</code></li>
  <li>부동소수점 → <code>float</code></li>
  <li>불리언 → <code>bool</code></li>
  <li>배열 → <code>List[T]</code></li>
  <li>혼합 배열 → <code>List[Union[T1, T2]]</code></li>
  <li>중첩 객체 → 별도 클래스</li>
</ul>

<h2>예시</h2>
<p>다음 JSON이 주어졌을 때:</p>
<pre><code>{`{"user": {"first_name": "Alice", "age": 30}, "tags": ["admin"], "note": null}`}</code></pre>
<p>@dataclass 모드에서 생성되는 코드:</p>
<pre><code>{`from dataclasses import dataclass\nfrom typing import Any, List, Optional\n\n@dataclass\nclass User:\n    first_name: str\n    age: int\n\n@dataclass\nclass Root:\n    user: User\n    tags: List[str]\n    note: Optional[Any] = None`}</code></pre>
