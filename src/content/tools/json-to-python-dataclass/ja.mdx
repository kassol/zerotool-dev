<h2>使い方</h2>
<ol>
  <li>左のパネルにJSONを貼り付けるか入力します。ツールがリアルタイムで検証します。</li>
  <li>必要に応じて<strong>ルートクラス名</strong>を設定します（デフォルト：<code>Root</code>）。</li>
  <li>出力モードを選択：<strong>@dataclass</strong>、<strong>Pydantic v2</strong>、または<strong>TypedDict</strong>。</li>
  <li><strong>Python を生成</strong>をクリックするか、リアルタイム変換を待ちます。</li>
  <li><strong>コピー</strong>で出力をコピー、または<strong>.py ダウンロード</strong>でファイルとして保存します。</li>
</ol>

<h2>型マッピング</h2>
<ul>
  <li><code>null</code> → <code>Optional[Any] = None</code></li>
  <li>文字列 → <code>str</code></li>
  <li>整数 → <code>int</code></li>
  <li>浮動小数点数 → <code>float</code></li>
  <li>真偽値 → <code>bool</code></li>
  <li>配列 → <code>List[T]</code></li>
  <li>混合配列 → <code>List[Union[T1, T2]]</code></li>
  <li>ネストオブジェクト → 別クラス</li>
</ul>

<h2>例</h2>
<p>このJSONが与えられた場合：</p>
<pre><code>{`{"user": {"first_name": "Alice", "age": 30}, "tags": ["admin"], "note": null}`}</code></pre>
<p>@dataclassモードで生成されるコード：</p>
<pre><code>{`from dataclasses import dataclass\nfrom typing import Any, List, Optional\n\n@dataclass\nclass User:\n    first_name: str\n    age: int\n\n@dataclass\nclass Root:\n    user: User\n    tags: List[str]\n    note: Optional[Any] = None`}</code></pre>
