<h2>使い方</h2>
<ol>
  <li>左のテキストエリアにSVGコードを貼り付けます。</li>
  <li>コンポーネント名を入力します（デフォルト：<code>MyIcon</code>）。先頭文字は自動で大文字になります。</li>
  <li>TypeScript・forwardRef・memoオプションを必要に応じて切り替えます。</li>
  <li>JSXコンポーネントが右のテキストエリアに即座に表示されます。</li>
  <li><strong>コピー</strong>ボタンでクリップボードにコピーします。</li>
</ol>

<h2>属性変換ルール</h2>

| SVG | JSX |
|-----|-----|
| `stroke-width` | `strokeWidth` |
| `stroke-linecap` | `strokeLinecap` |
| `fill-rule` | `fillRule` |
| `clip-path` | `clipPath` |
| `class` | `className` |
| `font-size` | `fontSize` |
| `text-anchor` | `textAnchor` |
| `xmlns="..."` | *(削除)* |

<h2>出力モード</h2>
<ul>
  <li><strong>デフォルト</strong>：TypeScript注釈なしのプレーンな関数コンポーネント。</li>
  <li><strong>TypeScript</strong>：<code>React.FC&lt;React.SVGProps&lt;SVGSVGElement&gt;&gt;</code>型を付与。className・style・onClickなどが型安全に使えます。</li>
  <li><strong>forwardRef</strong>：<code>React.forwardRef</code>でラップし、親コンポーネントにSVG要素のrefを公開します。</li>
  <li><strong>memo</strong>：最終エクスポートを<code>React.memo</code>でラップし、不要な再レンダリングをスキップします。</li>
</ul>

<h2>主な用途</h2>
<ul>
  <li>FigmaやIllustratorからエクスポートしたSVGをReactアイコンコンポーネントに変換</li>
  <li>SVGアイコンライブラリをJSXベースのデザインシステムに移行</li>
  <li>SVGアイコンにclassName・styleなどのReact propsを受け取れるようにする</li>
  <li>propsスプレッドでaria-labelを受け取るアクセシブルなアイコンコンポーネントの構築</li>
</ul>
