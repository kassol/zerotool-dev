<h2>使用方法</h2>
<ol>
  <li>将 SVG 代码粘贴到左侧文本框。</li>
  <li>输入组件名称（默认：<code>MyIcon</code>），首字母会自动大写。</li>
  <li>根据需要切换 TypeScript、forwardRef 或 memo 选项。</li>
  <li>JSX 组件输出会即时出现在右侧文本框。</li>
  <li>点击<strong>复制</strong>将其复制到剪贴板。</li>
</ol>

<h2>属性转换规则</h2>

| SVG | JSX |
|-----|-----|
| `stroke-width` | `strokeWidth` |
| `stroke-linecap` | `strokeLinecap` |
| `fill-rule` | `fillRule` |
| `clip-path` | `clipPath` |
| `class` | `className` |
| `font-size` | `fontSize` |
| `text-anchor` | `textAnchor` |
| `xmlns="..."` | *(已移除)* |

<h2>输出模式</h2>
<ul>
  <li><strong>默认</strong>：无 TypeScript 注解的普通函数组件。</li>
  <li><strong>TypeScript</strong>：添加 <code>React.FC&lt;React.SVGProps&lt;SVGSVGElement&gt;&gt;</code> 类型，支持 <code>className</code>、<code>style</code>、<code>onClick</code> 等 prop。</li>
  <li><strong>forwardRef</strong>：将组件包裹在 <code>React.forwardRef</code> 中，向父组件暴露 SVG 元素的 ref。</li>
  <li><strong>memo</strong>：将最终导出包裹在 <code>React.memo</code> 中，跳过不必要的重新渲染。</li>
</ul>

<h2>常见使用场景</h2>
<ul>

  <li>将 SVG 图标库迁移到基于 JSX 的设计系统</li>
  <li>让 SVG 图标接受 <code>className</code>、<code>style</code> 等 React props</li>
  <li>构建通过展开 props 接受 <code>aria-label</code> 的无障碍图标组件</li>
</ul>
