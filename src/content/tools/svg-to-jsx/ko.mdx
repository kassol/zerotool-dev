<h2>사용 방법</h2>
<ol>
  <li>왼쪽 텍스트 영역에 SVG 코드를 붙여넣습니다.</li>
  <li>컴포넌트 이름을 입력합니다(기본값: <code>MyIcon</code>). 첫 글자는 자동으로 대문자가 됩니다.</li>
  <li>필요에 따라 TypeScript, forwardRef, memo 옵션을 토글합니다.</li>
  <li>JSX 컴포넌트 출력이 오른쪽 텍스트 영역에 즉시 나타납니다.</li>
  <li><strong>복사</strong>를 클릭해 클립보드에 복사합니다.</li>
</ol>

<h2>속성 변환 규칙</h2>

| SVG | JSX |
|-----|-----|
| `stroke-width` | `strokeWidth` |
| `stroke-linecap` | `strokeLinecap` |
| `fill-rule` | `fillRule` |
| `clip-path` | `clipPath` |
| `class` | `className` |
| `font-size` | `fontSize` |
| `text-anchor` | `textAnchor` |
| `xmlns="..."` | *(제거됨)* |

<h2>출력 모드</h2>
<ul>
  <li><strong>기본</strong>: TypeScript 주석 없는 일반 함수 컴포넌트.</li>
  <li><strong>TypeScript</strong>: <code>React.FC&lt;React.SVGProps&lt;SVGSVGElement&gt;&gt;</code> 타입 적용. className, style, onClick이 타입 안전하게 작동합니다.</li>
  <li><strong>forwardRef</strong>: <code>React.forwardRef</code>로 감싸 부모 컴포넌트에 SVG 요소 ref를 공개합니다.</li>
  <li><strong>memo</strong>: 최종 내보내기를 <code>React.memo</code>로 감싸 불필요한 리렌더링을 방지합니다.</li>
</ul>

<h2>주요 사용 사례</h2>
<ul>
  <li>Figma나 Illustrator에서 내보낸 SVG를 React 아이콘 컴포넌트로 변환</li>
  <li>SVG 아이콘 라이브러리를 JSX 기반 디자인 시스템으로 마이그레이션</li>
  <li>SVG 아이콘이 className, style 등 React props를 받을 수 있도록 설정</li>
  <li>props 스프레드로 aria-label을 받는 접근성 아이콘 컴포넌트 구축</li>
</ul>
