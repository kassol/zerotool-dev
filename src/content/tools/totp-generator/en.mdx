<h2>How to Use</h2>
<ol>
  <li>Enter your Base32 secret, or click <strong>Random</strong> to generate a new one.</li>
  <li>Optionally configure digits (6 or 8), period (30s or 60s), and algorithm (SHA-1/SHA-256/SHA-512).</li>
  <li>The current TOTP code appears with a live countdown bar.</li>
  <li>Scan the QR code with any TOTP authenticator app (Google Authenticator, Authy, etc.).</li>
  <li>Copy the <code>otpauth://</code> URI if you need to configure an authenticator programmatically.</li>
</ol>

<h2>Algorithm Notes</h2>
<ul>
  <li><strong>SHA-1</strong> is the default and is used by most authenticator apps and platforms.</li>
  <li><strong>SHA-256 / SHA-512</strong> provide stronger HMAC but are less widely supported by authenticator apps.</li>
  <li>Make sure your server and this tool use the same algorithm and period for codes to match.</li>
</ul>
