<h2>사용 방법</h2>
<ol>
  <li><strong>TOML → JSON</strong>: TOML을 입력창에 붙여넣고 <strong>변환</strong>을 클릭합니다.</li>
  <li><strong>JSON → TOML</strong>: JSON을 입력창에 붙여넣고 <strong>변환</strong>을 클릭합니다.</li>
  <li>결과를 <strong>복사</strong>합니다.</li>
</ol>

<h2>TOML의 활용 사례</h2>
<p>Rust 프로젝트 설정(Cargo.toml), Python 패키지 설정(pyproject.toml), 정적 사이트 생성기 설정(Hugo, Zola) 등에서 널리 사용됩니다. 기존 JSON 설정을 TOML로 마이그레이션하거나 TOML 내용을 API에 전달할 때 변환이 필요합니다.</p>
