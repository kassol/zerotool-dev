<h2>使い方</h2>
<ol>
  <li><strong>入力テキスト</strong>フィールドにテキストを入力または貼り付けます（最大80文字）。</li>
  <li>ドロップダウンから<strong>フォント</strong>を選択してASCIIアートのスタイルを変更します。</li>
  <li>出力は自動更新されます。<strong>コピー</strong>をクリックして結果をコピーします。</li>
</ol>

<h2>利用可能なフォント</h2>

| フォント | スタイル |
|---------|---------|
| Standard | クリーンでクラシックなfigletデフォルト |
| Doom | 太字のターミナル風 |
| Slant | 斜体の対角線文字 |
| Shadow | ドロップシャドウ付き文字 |
| Big | 超大型ブロック文字 |
| Block | 塗りつぶし長方形 |
| Banner | 横長フラット文字 |
| Script | 筆記体風ストローク |
| Lean | 細身のコンデンスドグリフ |
| Mini | コンパクトな2行高さ |
| Small | Standardの小型版 |
| 3D-ASCII | 3Dパースペクティブ押し出し |
