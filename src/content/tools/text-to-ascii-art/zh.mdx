<h2>使用方法</h2>
<ol>
  <li>在<strong>输入文字</strong>栏中输入或粘贴文本，最多 80 个字符。</li>
  <li>从下拉菜单中选择<strong>字体</strong>以切换 ASCII 艺术风格。</li>
  <li>输出自动更新，点击<strong>复制</strong>即可复制结果。</li>
</ol>

<h2>可用字体</h2>

| 字体 | 风格 |
|------|------|
| Standard | 简洁经典的 figlet 默认字体 |
| Doom | 粗体终端风格 |
| Slant | 斜体对角线字母 |
| Shadow | 带阴影效果的字母 |
| Big | 超大块状字母 |
| Block | 实心矩形填充 |
| Banner | 宽扁平字符 |
| Script | 手写曲线风格 |
| Lean | 窄体压缩字形 |
| Mini | 紧凑双行高度 |
| Small | Standard 的小尺寸版本 |
| 3D-ASCII | 3D 透视立体效果 |
