<h2>사용 방법</h2>
<ol>
  <li><strong>입력 텍스트</strong> 필드에 텍스트를 입력하거나 붙여넣습니다 (최대 80자).</li>
  <li>드롭다운에서 <strong>글꼴</strong>을 선택해 ASCII 아트 스타일을 변경합니다.</li>
  <li>출력이 자동으로 업데이트됩니다. <strong>복사</strong>를 클릭해 결과를 복사합니다.</li>
</ol>

<h2>사용 가능한 글꼴</h2>

| 글꼴 | 스타일 |
|------|--------|
| Standard | 깔끔한 클래식 figlet 기본 글꼴 |
| Doom | 굵은 터미널 스타일 |
| Slant | 기울어진 사선 문자 |
| Shadow | 그림자 효과가 있는 문자 |
| Big | 초대형 블록 문자 |
| Block | 꽉 찬 직사각형 |
| Banner | 넓고 납작한 문자 |
| Script | 필기체 스트로크 |
| Lean | 좁고 촘촘한 자형 |
| Mini | 소형 2행 높이 |
| Small | Standard의 소형 버전 |
| 3D-ASCII | 3D 원근법 돌출 효과 |
