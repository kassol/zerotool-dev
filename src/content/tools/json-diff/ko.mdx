<h2>사용 방법</h2>
<ol>
  <li>원본 JSON을 <strong>「변경 전」</strong> 패널에 붙여넣습니다.</li>
  <li>업데이트된 JSON을 <strong>「변경 후」</strong> 패널에 붙여넣습니다.</li>
  <li><strong>비교</strong>를 클릭하여 차이를 실행합니다.</li>
  <li><strong>비주얼 차이</strong>에서 사람이 읽기 쉬운 변경사항을 검토합니다.</li>
  <li>애플리케이션이나 API에 사용할 <strong>JSON Patch</strong>를 복사합니다.</li>
</ol>

<h2>사용 사례</h2>
<ul>
  <li>버전 간 API 응답 변경 사항 검토.</li>
  <li>PATCH HTTP 엔드포인트용 패치 작업 생성.</li>
  <li>설정 파일 변경 감사.</li>
  <li>애플리케이션의 예상치 못한 데이터 변이 디버그.</li>
</ul>
