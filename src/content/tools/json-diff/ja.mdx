<h2>使い方</h2>
<ol>
  <li>元のJSONを<strong>「変更前」</strong>パネルに貼り付けます。</li>
  <li>更新後のJSONを<strong>「変更後」</strong>パネルに貼り付けます。</li>
  <li><strong>比較</strong>をクリックして差分を実行します。</li>
  <li><strong>ビジュアルDiff</strong>で人間が読みやすい変更を確認します。</li>
  <li><strong>JSON Patch</strong>をコピーしてアプリケーションやAPIで使用します。</li>
</ol>

<h2>ユースケース</h2>
<ul>
  <li>バージョン間のAPIレスポンスの変化を確認する。</li>
  <li>PATCH HTTPエンドポイント用のパッチ操作を生成する。</li>
  <li>設定ファイルの変更を監査する。</li>
  <li>アプリケーション内の予期しないデータ変異をデバッグする。</li>
</ul>
