<h2>使用方法</h2>
<ol>
  <li>将 JSON 粘贴或输入到左侧面板，工具会实时验证并高亮错误。</li>
  <li>可选：设置 <strong>Root interface name</strong>（默认：<code>RootObject</code>）。</li>
  <li>勾选 <strong>Make properties optional</strong> 可将所有字段标记为 <code>?</code> 可选。</li>
  <li>勾选 <strong>Use type instead of interface</strong> 可将输出关键字改为 <code>type</code>。</li>
  <li>点击 <strong>Generate TypeScript</strong> 生成接口定义。</li>
  <li>点击 <strong>Copy</strong> 将输出复制到剪贴板。</li>
</ol>

<h2>类型推断规则</h2>
<ul>
  <li><code>null</code> → <code>null</code></li>
  <li>字符串 → <code>string</code></li>
  <li>数字 → <code>number</code></li>
  <li>布尔值 → <code>boolean</code></li>
  <li>空数组 → <code>unknown[]</code></li>
  <li>对象数组 → 合并接口 + <code>ItemName[]</code></li>
  <li>混合类型数组 → 联合类型，如 <code>(string | number)[]</code></li>
  <li>嵌套对象 → 独立具名接口</li>
</ul>

<h2>示例</h2>
<p>输入以下 JSON：</p>
<pre><code>{`{"user": {"name": "Alice", "age": 30}, "tags": ["admin", "user"]}`}</code></pre>
<p>工具生成：</p>
<pre><code>{`interface User {\n  name: string;\n  age: number;\n}\n\ninterface RootObject {\n  user: User;\n  tags: string[];\n}`}</code></pre>
