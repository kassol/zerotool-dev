<h2>How to Use</h2>
<ol>
  <li>Paste or type your JSON into the left panel. The tool validates it in real time and highlights errors.</li>
  <li>Optionally set a <strong>Root interface name</strong> (default: <code>RootObject</code>).</li>
  <li>Toggle <strong>Make properties optional</strong> if you want every field marked with <code>?</code>.</li>
  <li>Toggle <strong>Use type instead of interface</strong> to switch the output keyword.</li>
  <li>Click <strong>Generate TypeScript</strong> to produce the interfaces.</li>
  <li>Click <strong>Copy</strong> to copy the output to your clipboard.</li>
</ol>

<h2>Type Inference Rules</h2>
<ul>
  <li><code>null</code> → <code>null</code></li>
  <li>string → <code>string</code></li>
  <li>number → <code>number</code></li>
  <li>boolean → <code>boolean</code></li>
  <li>empty array → <code>unknown[]</code></li>
  <li>array of objects → merged interface + <code>ItemName[]</code></li>
  <li>array of mixed primitives → union, e.g. <code>(string | number)[]</code></li>
  <li>nested object → separate named interface</li>
</ul>

<h2>Example</h2>
<p>Given this JSON:</p>
<pre><code>{`{"user": {"name": "Alice", "age": 30}, "tags": ["admin", "user"]}`}</code></pre>
<p>The tool generates:</p>
<pre><code>{`interface User {\n  name: string;\n  age: number;\n}\n\ninterface RootObject {\n  user: User;\n  tags: string[];\n}`}</code></pre>
