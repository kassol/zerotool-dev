<h2>使い方</h2>
<ol>
  <li>左側のパネルにJSONを貼り付けるか入力します。リアルタイムで検証しエラーをハイライト表示します。</li>
  <li>必要に応じて <strong>Root interface name</strong>（デフォルト: <code>RootObject</code>）を設定します。</li>
  <li><strong>Make properties optional</strong> をオンにすると、すべてのフィールドが <code>?</code> 付きのオプショナルになります。</li>
  <li><strong>Use type instead of interface</strong> をオンにすると出力キーワードが <code>type</code> になります。</li>
  <li><strong>Generate TypeScript</strong> をクリックしてインターフェースを生成します。</li>
  <li><strong>Copy</strong> をクリックして出力をクリップボードにコピーします。</li>
</ol>

<h2>型推論ルール</h2>
<ul>
  <li><code>null</code> → <code>null</code></li>
  <li>文字列 → <code>string</code></li>
  <li>数値 → <code>number</code></li>
  <li>真偽値 → <code>boolean</code></li>
  <li>空配列 → <code>unknown[]</code></li>
  <li>オブジェクト配列 → マージされたインターフェース + <code>ItemName[]</code></li>
  <li>混合プリミティブ配列 → Union型（例: <code>(string | number)[]</code>）</li>
  <li>ネストオブジェクト → 独立した名前付きインターフェース</li>
</ul>

<h2>例</h2>
<p>以下のJSONを入力すると:</p>
<pre><code>{`{"user": {"name": "Alice", "age": 30}, "tags": ["admin", "user"]}`}</code></pre>
<p>次のように生成されます:</p>
<pre><code>{`interface User {\n  name: string;\n  age: number;\n}\n\ninterface RootObject {\n  user: User;\n  tags: string[];\n}`}</code></pre>
