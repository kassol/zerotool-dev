<h2>사용 방법</h2>
<ol>
  <li>왼쪽 패널에 JSON을 붙여넣거나 입력합니다. 실시간으로 검증하고 오류를 하이라이트합니다.</li>
  <li>필요하면 <strong>Root interface name</strong>을 설정합니다(기본값: <code>RootObject</code>).</li>
  <li><strong>Make properties optional</strong>을 켜면 모든 필드가 <code>?</code> 선택적 속성이 됩니다.</li>
  <li><strong>Use type instead of interface</strong>를 켜면 출력 키워드가 <code>type</code>으로 바뀝니다.</li>
  <li><strong>Generate TypeScript</strong>를 클릭해 인터페이스를 생성합니다.</li>
  <li><strong>Copy</strong>를 클릭해 출력을 클립보드에 복사합니다.</li>
</ol>

<h2>타입 추론 규칙</h2>
<ul>
  <li><code>null</code> → <code>null</code></li>
  <li>문자열 → <code>string</code></li>
  <li>숫자 → <code>number</code></li>
  <li>불리언 → <code>boolean</code></li>
  <li>빈 배열 → <code>unknown[]</code></li>
  <li>객체 배열 → 병합된 인터페이스 + <code>ItemName[]</code></li>
  <li>혼합 기본 타입 배열 → 유니언 타입(예: <code>(string | number)[]</code>)</li>
  <li>중첩 객체 → 별도의 명명된 인터페이스</li>
</ul>

<h2>예시</h2>
<p>다음 JSON을 입력하면:</p>
<pre><code>{`{"user": {"name": "Alice", "age": 30}, "tags": ["admin", "user"]}`}</code></pre>
<p>다음과 같이 생성됩니다:</p>
<pre><code>{`interface User {\n  name: string;\n  age: number;\n}\n\ninterface RootObject {\n  user: User;\n  tags: string[];\n}`}</code></pre>
