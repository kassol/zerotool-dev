<h2>使い方</h2>
<ol>
  <li><strong>flex-direction</strong>を調整してメイン軸を設定します：row（水平）またはcolumn（垂直）。</li>
  <li><strong>flex-wrap</strong>を設定してアイテムを複数行に折り返せるようにします。</li>
  <li><strong>justify-content</strong>でメイン軸のスペース配置を設定します。</li>
  <li><strong>align-items</strong>でクロス軸上の配置を設定します。</li>
  <li>複数行の折り返し動作には<strong>align-content</strong>を調整します。</li>
  <li><strong>ギャップ</strong>でアイテム間のガターを設定します。</li>
  <li><strong>アイテム数</strong>でプレビューボックスを追加・削除してレスポンスを確認します。</li>
  <li><strong>コピー</strong>をクリックして生成された <code>.container &#123; ... &#125;</code> ルールをコピーします。</li>
</ol>

<h2>生成されるCSSについて</h2>
<p>
  ジェネレーターは7つのプロパティを持つ <code>.container</code> ルールを出力します：
  <code>display: flex</code>、<code>flex-direction</code>、<code>flex-wrap</code>、
  <code>justify-content</code>、<code>align-items</code>、<code>align-content</code>、
  <code>gap</code>。スタイルシートに貼り付け、<code>.container</code>を自分のクラス名に変更してください。
</p>

<h2>Flexbox と Grid の使い分け</h2>
<p>
  一次元レイアウト（単一の行または列に沿ったアイテムの配置）にはFlexboxを使います。
  行と列を同時に制御する二次元レイアウトにはCSS Gridが適しています。
  実践では、Flexboxはコンポーネントレベルのレイアウト（ナビゲーション、ボタングループ、フォーム行）に優れ、
  Gridはページレベルの構造に向いています。
</p>
