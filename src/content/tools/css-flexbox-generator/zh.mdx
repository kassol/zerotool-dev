faqItems:
  - question: "什么是 CSS Flexbox？"
    answer: "CSS Flexbox（弹性盒布局）是一种一维布局模型，沿单轴（行或列）分配空间。适用于导航栏、工具栏、卡片行、居中元素以及任何需要沿单一方向灵活排列子项的 UI。"
  - question: "justify-content 和 align-items 有什么区别？"
    answer: "justify-content 控制主轴（由 flex-direction 定义的方向）上的对齐方式；align-items 控制交叉轴（垂直于主轴）上的对齐方式。对于行方向的 Flex 容器：justify-content 控制水平对齐，align-items 控制垂直对齐。"
  - question: "什么时候应该使用 flex-wrap？"
    answer: "默认情况下 flex-wrap 为 nowrap——所有子项保持在一行，必要时会收缩。设置 flex-wrap: wrap 可让子项无法容纳时换行。适合响应式卡片网格或标签列表等在小屏幕上需要自动换行的场景。"
  - question: "align-content 的作用是什么？"
    answer: "align-content 控制多行 Flex 子项沿交叉轴的分布方式。仅在存在多行时（flex-wrap 设为 wrap 或 wrap-reverse）有效。若只有一行，该属性无明显效果。"
  - question: "Flexbox 中的 gap 属性是什么？"
    answer: "gap（row-gap 和 column-gap 的简写）设置 Flex 子项之间的间距，不会给容器外边缘添加 margin。该属性最初只适用于 Grid，现已在所有现代浏览器中完全支持 Flexbox。"

<h2>使用方法</h2>
<ol>
  <li>调整 <strong>flex-direction</strong> 设置主轴方向：row（水平）或 column（垂直）。</li>
  <li>设置 <strong>flex-wrap</strong> 允许子项换行到多行。</li>
  <li>配置 <strong>justify-content</strong> 控制主轴上的间距。</li>
  <li>设置 <strong>align-items</strong> 控制交叉轴上的对齐方式。</li>
  <li>调整 <strong>align-content</strong> 控制多行换行行为。</li>
  <li>更改 <strong>间距</strong> 设置子项之间的沟槽。</li>
  <li>使用<strong>子项数量</strong>增减预览框，观察响应效果。</li>
  <li>点击<strong>复制</strong>复制生成的 <code>.container &#123; ... &#125;</code> 规则。</li>
</ol>

<h2>理解生成的 CSS</h2>
<p>
  生成器输出一条包含七个属性的 <code>.container</code> 规则：
  <code>display: flex</code>、<code>flex-direction</code>、<code>flex-wrap</code>、
  <code>justify-content</code>、<code>align-items</code>、<code>align-content</code> 和 <code>gap</code>。
  粘贴到样式表后，将 <code>.container</code> 改为你自己的类名即可。
</p>

<h2>Flexbox 与 Grid 的区别</h2>
<p>
  一维布局（沿单行或单列排列子项）用 Flexbox；需要同时控制行和列的二维布局用 CSS Grid。
  实践中，Flexbox 擅长组件级布局（导航、按钮组、表单行），
  Grid 更适合页面级结构。
</p>
