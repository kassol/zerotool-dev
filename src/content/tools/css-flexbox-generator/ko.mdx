<h2>사용 방법</h2>
<ol>
  <li><strong>flex-direction</strong>을 조정하여 주축을 설정합니다: row(수평) 또는 column(수직).</li>
  <li><strong>flex-wrap</strong>을 설정하여 아이템이 여러 줄로 넘어갈 수 있게 합니다.</li>
  <li><strong>justify-content</strong>로 주축의 간격 배치를 구성합니다.</li>
  <li><strong>align-items</strong>로 교차축의 정렬 방식을 설정합니다.</li>
  <li>여러 줄 줄바꿈 동작에는 <strong>align-content</strong>를 조정합니다.</li>
  <li><strong>간격</strong>을 변경하여 아이템 사이의 거터를 설정합니다.</li>
  <li><strong>아이템 수</strong>로 미리보기 박스를 추가·제거하며 응답을 확인합니다.</li>
  <li><strong>복사</strong>를 클릭하여 생성된 <code>.container &#123; ... &#125;</code> 규칙을 복사합니다.</li>
</ol>

<h2>생성된 CSS 이해하기</h2>
<p>
  생성기는 일곱 가지 속성을 가진 <code>.container</code> 규칙을 출력합니다:
  <code>display: flex</code>, <code>flex-direction</code>, <code>flex-wrap</code>,
  <code>justify-content</code>, <code>align-items</code>, <code>align-content</code>,
  <code>gap</code>. 스타일시트에 붙여넣고 <code>.container</code>를 자신의 클래스명으로 변경하세요.
</p>

<h2>Flexbox와 Grid 비교</h2>
<p>
  1차원 레이아웃(단일 행 또는 열을 따른 아이템 배열)에는 Flexbox를 사용합니다.
  행과 열을 동시에 제어하는 2차원 레이아웃에는 CSS Grid가 적합합니다.
  실제로 Flexbox는 컴포넌트 수준 레이아웃(내비게이션, 버튼 그룹, 폼 행)에 뛰어나고,
  Grid는 페이지 수준 구조에 더 적합합니다.
</p>
