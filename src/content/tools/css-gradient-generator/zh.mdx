<h2>使用方法</h2>
<ol>
  <li>使用顶部标签选择渐变类型——<strong>线性</strong>、<strong>径向</strong>或<strong>锥形</strong>。</li>
  <li><strong>线性</strong>：拖动角度滑块或输入 0 到 360 度的值。使用方向快捷按钮（上 / 右 / 下 / 左）快速设置。</li>
  <li><strong>径向</strong>：从下拉菜单中选择形状（圆形或椭圆）和中心位置。</li>
  <li><strong>锥形</strong>：用滑块设置起始角度，并选择中心位置。</li>
  <li>在<strong>颜色节点</strong>部分，点击色块或输入十六进制值来更改每个节点的颜色。调整百分比控制其在渐变中的位置。</li>
  <li>点击 <strong>+ 添加</strong> 插入更多节点（最多六个），或点击 <strong>×</strong> 删除（最少保留两个）。</li>
  <li><strong>预览矩形</strong>实时随修改更新。</li>
  <li>点击<strong>复制</strong>将 CSS 规则复制到剪贴板。</li>
</ol>

<h2>理解生成的 CSS</h2>
<p>
  输出为 <code>.gradient</code> 类设置 <code>background</code> 属性。对于线性渐变，角度控制方向——0deg 从上到下，90deg 从左到右。
  对于径向渐变，语法在颜色列表前指定形状和位置。对于锥形渐变，<code>from</code> 设置起始角度，<code>at</code> 设置旋转中心。
</p>

<h2>常见用途</h2>
<p>
  CSS 渐变广泛用于 Hero 区域背景、卡片头部、按钮填充、进度条、装饰性遮罩层以及文字填充效果（通过 <code>background-clip: text</code>）。
  锥形渐变特别适合用纯 CSS 实现饼图和色轮选择器。
</p>
