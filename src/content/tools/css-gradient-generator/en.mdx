<h2>Understanding the Generated CSS</h2>
<p>
  The output sets <code>background</code> on a <code>.gradient</code> class. For linear gradients, the angle
  controls the direction — 0deg points upward (bottom-to-top), 90deg points right (left-to-right),
  and 180deg points downward (top-to-bottom). For radial gradients, the syntax specifies the shape
  and its position before the color list. For conic gradients, <code>from</code> sets
  the starting angle and <code>at</code> sets the sweep center.
</p>

<h2>Common Use Cases</h2>
<p>
  CSS gradients are used for hero section backgrounds, card headers, button fills, progress bars, decorative
  overlays, and text fill effects (via <code>background-clip: text</code>). Conic gradients are especially
  useful for pie charts and color-wheel pickers rendered purely in CSS.
</p>
