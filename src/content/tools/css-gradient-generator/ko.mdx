<h2>사용 방법</h2>
<ol>
  <li>상단 탭을 사용하여 그라디언트 유형을 선택합니다——<strong>선형</strong>, <strong>방사형</strong>, 또는 <strong>원뿔형</strong>.</li>
  <li><strong>선형</strong>: 각도 슬라이더를 드래그하거나 0~360도 값을 입력합니다. 방향 버튼(위 / 오른쪽 / 아래 / 왼쪽)을 빠른 설정 단축키로 사용할 수 있습니다.</li>
  <li><strong>방사형</strong>: 드롭다운에서 모양(원형 또는 타원)과 중심 위치를 선택합니다.</li>
  <li><strong>원뿔형</strong>: 슬라이더로 시작 각도를 설정하고 중심 위치를 선택합니다.</li>
  <li><strong>색상 정지점</strong> 섹션에서 색상 견본을 클릭하거나 16진수 값을 입력하여 각 정지점의 색상을 변경합니다. 퍼센트를 조정하여 그라디언트 내 위치를 제어합니다.</li>
  <li><strong>+ 추가</strong>를 클릭하여 정지점 추가(최대 6개), 또는 <strong>×</strong>로 제거(최소 2개 유지)합니다.</li>
  <li><strong>미리보기</strong> 사각형이 변경에 따라 실시간으로 업데이트됩니다.</li>
  <li><strong>복사</strong>를 클릭하여 CSS 규칙을 클립보드에 복사합니다.</li>
</ol>

<h2>생성된 CSS 이해하기</h2>
<p>
  출력은 <code>.gradient</code> 클래스에 <code>background</code> 속성을 설정합니다. 선형 그라디언트에서 각도는 방향을 제어합니다——0deg는 위에서 아래로, 90deg는 왼쪽에서 오른쪽으로입니다.
  방사형 그라디언트에서는 색상 목록 앞에 형태와 위치가 지정됩니다. 원뿔형 그라디언트에서 <code>from</code>은 시작 각도를, <code>at</code>은 스윕 중심을 설정합니다.
</p>

<h2>주요 사용 사례</h2>
<p>
  CSS 그라디언트는 히어로 섹션 배경, 카드 헤더, 버튼 채우기, 진행 표시줄, 장식 오버레이, 텍스트 채우기 효과(<code>background-clip: text</code>)에 사용됩니다.
  원뿔형 그라디언트는 순수 CSS로 파이 차트와 색상환 선택기를 렌더링하는 데 특히 유용합니다.
</p>
