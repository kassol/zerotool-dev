<h2>使用方法</h2>
<ol>
  <li>将你的 <strong>cURL 命令</strong>粘贴到左侧输入框。</li>
  <li>点击 <strong>转换</strong>。</li>
  <li>选择目标语言标签（Python、JavaScript、Go、PHP 或 Node.js）。</li>
  <li>点击 <strong>复制</strong>复制生成的代码。</li>
</ol>

<h2>支持的参数</h2>
<ul>
  <li><code>-X METHOD</code> — HTTP 方法（GET、POST、PUT、DELETE 等）</li>
  <li><code>-H "Header: Value"</code> — 自定义请求头</li>
  <li><code>-d / --data / --data-raw</code> — 请求体</li>
  <li><code>-u user:password</code> — 基本认证</li>
  <li><code>-L</code> — 跟随重定向</li>
  <li><code>--cookie "name=value"</code> — Cookie 头</li>
  <li><code>-F "field=value"</code> — 多部分表单数据</li>
</ul>
