<h2>How to Use</h2>
<ol>
  <li>Paste your <strong>cURL command</strong> into the left panel.</li>
  <li>Click <strong>Convert</strong>.</li>
  <li>Select the target language tab (Python, JavaScript, Go, PHP, or Node.js).</li>
  <li>Click <strong>Copy</strong> to copy the generated code.</li>
</ol>

<h2>Supported Flags</h2>
<ul>
  <li><code>-X METHOD</code> — HTTP method (GET, POST, PUT, DELETE, etc.)</li>
  <li><code>-H "Header: Value"</code> — Custom request headers</li>
  <li><code>-d / --data / --data-raw</code> — Request body</li>
  <li><code>-u user:password</code> — Basic authentication</li>
  <li><code>-L</code> — Follow redirects</li>
  <li><code>--cookie "name=value"</code> — Cookie header</li>
  <li><code>-F "field=value"</code> — Multipart form data</li>
</ul>
