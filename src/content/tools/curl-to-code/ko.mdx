<h2>사용 방법</h2>
<ol>
  <li>왼쪽 패널에 <strong>cURL 명령어</strong>를 붙여넣습니다.</li>
  <li><strong>변환</strong>을 클릭합니다.</li>
  <li>언어 탭(Python, JavaScript, Go, PHP, Node.js)을 선택합니다.</li>
  <li><strong>복사</strong>를 클릭하여 생성된 코드를 복사합니다.</li>
</ol>

<h2>지원 플래그</h2>
<ul>
  <li><code>-X METHOD</code> — HTTP 메서드 (GET, POST, PUT, DELETE 등)</li>
  <li><code>-H "Header: Value"</code> — 커스텀 요청 헤더</li>
  <li><code>-d / --data / --data-raw</code> — 요청 본문</li>
  <li><code>-u user:password</code> — 기본 인증</li>
  <li><code>-L</code> — 리다이렉트 추적</li>
  <li><code>--cookie "name=value"</code> — 쿠키 헤더</li>
  <li><code>-F "field=value"</code> — 멀티파트 폼 데이터</li>
</ul>

<h2>주요 사용 사례</h2>
<ul>
  <li><strong>API 통합:</strong> 문서의 cURL 예제를 원하는 언어로 변환.</li>
  <li><strong>디버깅:</strong> 브라우저 DevTools에서 복사한 cURL을 코드로 변환하여 테스트.</li>
  <li><strong>팀 협업:</strong> cURL에 익숙하지 않은 팀원에게 요청 내용 설명.</li>
  <li><strong>빠른 개발:</strong> 보일러플레이트 없이 HTTP 요청 빠르게 구현.</li>
</ul>
