<h2>対応フラグ</h2>
<ul>
  <li><code>-X METHOD</code> — HTTPメソッド（GET・POST・PUT・DELETEなど）</li>
  <li><code>-H "Header: Value"</code> — カスタムリクエストヘッダー</li>
  <li><code>-d / --data / --data-raw</code> — リクエストボディ</li>
  <li><code>-u user:password</code> — Basic認証</li>
  <li><code>-L</code> — リダイレクト追跡</li>
  <li><code>--cookie "name=value"</code> — Cookieヘッダー</li>
  <li><code>-F "field=value"</code> — マルチパートフォームデータ</li>
</ul>
