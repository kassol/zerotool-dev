<h2>使い方</h2>
<ol>
  <li><strong>カラーピッカー</strong>を使うか、<strong>HEX値</strong>（例：<code>#3b82f6</code>）を直接入力してベースカラーを設定します。</li>
  <li>パレットが即座に更新され、最も明るい（50）から最も暗い（900）まで10シェードが表示されます。</li>
  <li>各シェードカードに <strong>HEX</strong>・<strong>HSL</strong>・<strong>RGB</strong> 値が表示されます。</li>
  <li>シェードカードをクリックすると、その<strong>HEX値をクリップボードにコピー</strong>します。</li>
  <li><strong>CSS変数としてすべてコピー</strong>をクリックすると、全シェードを <code>:root</code> CSSブロックとしてコピーします。</li>
</ol>

<h2>コピー全体の出力形式</h2>

```css
:root {
  --color-50: #EFF6FF;
  --color-100: #DBEAFE;
  --color-200: #BFDBFE;
  --color-300: #93C5FD;
  --color-400: #60A5FA;
  --color-500: #3B82F6;
  --color-600: #2563EB;
  --color-700: #1D4ED8;
  --color-800: #1E40AF;
  --color-900: #1E3A8A;
}
```

<h2>デザインシステムでのシェード活用</h2>
<p>10段階シェードスケールは以下に最適です：</p>
<ul>
  <li><strong>プライマリ/アクセントカラー：</strong>500をメインブランドカラーとして使用し、明るいシェードを背景・ホバー状態に、暗いシェードをテキスト・ボーダーに使用</li>
  <li><strong>ステートフィードバック：</strong>50–100を警告背景のティントに、700–900をその上のテキスト色に</li>
  <li><strong>ダークモード：</strong>スケールを反転させ、900を背景に、50をテキストに</li>
  <li><strong>コンポーネントバリアント：</strong>ボタンのデフォルト・ホバー・アクティブ状態をそれぞれ500・600・700に</li>
</ul>
