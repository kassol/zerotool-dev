<h2>使用方法</h2>
<ol>
  <li>使用<strong>颜色选择器</strong>或输入 <strong>HEX 值</strong>（如 <code>#3b82f6</code>）设置基础颜色。</li>
  <li>调色板即时更新，显示从最浅（50）到最深（900）的 10 个色阶。</li>
  <li>每个色块显示 <strong>HEX</strong>、<strong>HSL</strong> 和 <strong>RGB</strong> 值。</li>
  <li>点击任意色块即可将其 <strong>HEX 值复制</strong>到剪贴板。</li>
  <li>点击<strong>复制全部（CSS 变量格式）</strong>将所有色阶复制为 <code>:root</code> CSS 块。</li>
</ol>

<h2>复制全部的输出格式</h2>

```css
:root {
  --color-50: #EFF6FF;
  --color-100: #DBEAFE;
  --color-200: #BFDBFE;
  --color-300: #93C5FD;
  --color-400: #60A5FA;
  --color-500: #3B82F6;
  --color-600: #2563EB;
  --color-700: #1D4ED8;
  --color-800: #1E40AF;
  --color-900: #1E3A8A;
}
```

<h2>在设计系统中使用色阶</h2>
<p>10 步色阶适用于：</p>
<ul>
  <li><strong>主色 / 强调色：</strong>用 500 作为主品牌色，浅色阶用于背景和悬停态，深色阶用于文字和边框</li>
  <li><strong>状态反馈：</strong>50–100 用于警告背景色调，700–900 用于其上的文字颜色</li>
  <li><strong>暗色模式：</strong>反转刻度方向——用 900 作背景，50 作文字</li>
  <li><strong>组件变体：</strong>按钮的默认、悬停、激活状态分别可用 500、600 和 700</li>
</ul>
