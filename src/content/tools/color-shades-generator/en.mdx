<h2>Using Shades in Design Systems</h2>
<p>A 10-step shade scale is ideal for:</p>
<ul>
  <li><strong>Primary / accent colors:</strong> use 500 as the main brand color, lighter shades for backgrounds and hover states, darker shades for text and borders</li>
  <li><strong>State feedback:</strong> use 50–100 for tinted alert backgrounds, 700–900 for text on those backgrounds</li>
  <li><strong>Dark mode:</strong> swap scale direction — use 900 as the background and 50 as text</li>
  <li><strong>Component variants:</strong> a button's default, hover, and active states can be 500, 600, and 700 respectively</li>
</ul>

<h2>HSL Color Model Explained</h2>
<ul>
  <li><strong>Hue (H):</strong> the color angle on the color wheel, 0–360°. Red is 0°, green is 120°, blue is 240°.</li>
  <li><strong>Saturation (S):</strong> the intensity of the color. 0% is grey, 100% is fully vivid.</li>
  <li><strong>Lightness (L):</strong> 0% is black, 50% is the "pure" color, 100% is white.</li>
</ul>
