<h2>사용 방법</h2>
<ol>
  <li><strong>색상 선택기</strong>를 사용하거나 <strong>HEX 값</strong>(예: <code>#3b82f6</code>)을 직접 입력하여 기본 색상을 설정합니다.</li>
  <li>팔레트가 즉시 업데이트되어 가장 밝은(50)부터 가장 어두운(900)까지 10단계 색상을 표시합니다.</li>
  <li>각 쉐이드 카드에 <strong>HEX</strong>, <strong>HSL</strong>, <strong>RGB</strong> 값이 표시됩니다.</li>
  <li>쉐이드 카드를 클릭하면 해당 <strong>HEX 값이 클립보드에 복사</strong>됩니다.</li>
  <li><strong>모두 CSS 변수로 복사</strong>를 클릭하면 모든 쉐이드를 <code>:root</code> CSS 블록으로 복사합니다.</li>
</ol>

<h2>복사 전체 출력 형식</h2>

```css
:root {
  --color-50: #EFF6FF;
  --color-100: #DBEAFE;
  --color-200: #BFDBFE;
  --color-300: #93C5FD;
  --color-400: #60A5FA;
  --color-500: #3B82F6;
  --color-600: #2563EB;
  --color-700: #1D4ED8;
  --color-800: #1E40AF;
  --color-900: #1E3A8A;
}
```

<h2>디자인 시스템에서 쉐이드 활용</h2>
<p>10단계 쉐이드 스케일은 다음에 적합합니다:</p>
<ul>
  <li><strong>기본/강조 색상:</strong> 500을 주 브랜드 색상으로 사용하고, 밝은 쉐이드는 배경과 호버 상태에, 어두운 쉐이드는 텍스트와 테두리에 사용</li>
  <li><strong>상태 피드백:</strong> 50–100을 알림 배경 색조에, 700–900을 그 위의 텍스트 색상에 사용</li>
  <li><strong>다크 모드:</strong> 스케일 방향을 반전시켜 900을 배경으로, 50을 텍스트로 사용</li>
  <li><strong>컴포넌트 변형:</strong> 버튼의 기본, 호버, 활성 상태에 각각 500, 600, 700 사용</li>
</ul>
