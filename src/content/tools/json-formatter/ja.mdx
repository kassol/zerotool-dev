<h2>使い方</h2>
<ol>
  <li>JSONを入力欄に貼り付けます。</li>
  <li><strong>整形</strong>をクリックするとスペース2つのインデントで整形されます。</li>
  <li><strong>圧縮</strong>をクリックすると1行に圧縮されます。</li>
  <li><strong>検証</strong>をクリックすると構文エラーをチェックします。</li>
  <li><strong>コピー</strong>をクリックすると結果をクリップボードにコピーします。</li>
</ol>

<h2>JSONフォーマットについて</h2>
<p>JSON（JavaScript Object Notation）は最も広く使われているデータ交換フォーマットです。フォーマッターはインデントを追加してネスト構造を読みやすくし、圧縮ツールは空白を除去してデータサイズを削減します。どちらの操作もデータの損失はありません。</p>
<p>このツールはブラウザ上で完全に動作し、データがデバイス外に出ることはありません。</p>
