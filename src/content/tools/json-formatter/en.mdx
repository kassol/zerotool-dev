<h2>How to Use</h2>
<ol>
  <li>Paste your JSON into the input box.</li>
  <li>Click <strong>Format</strong> to prettify with 2-space indentation.</li>
  <li>Click <strong>Minify</strong> to compress into a single line.</li>
  <li>Click <strong>Validate</strong> to check for syntax errors without modifying output.</li>
  <li>Click <strong>Copy</strong> to copy the result to clipboard.</li>
</ol>

<h2>About JSON Formatter</h2>
<p>
  JSON (JavaScript Object Notation) is the most common data interchange format.
  A formatter makes nested structures readable by adding indentation, while a minifier
  strips whitespace to reduce payload size. Both operations are lossless — the parsed
  value is identical.
</p>
<p>
  This tool runs entirely in your browser. No data ever leaves your machine.
</p>
