<h2>사용 방법</h2>
<ol>
  <li>JSON을 입력창에 붙여넣습니다.</li>
  <li><strong>정렬</strong>을 클릭하면 공백 2칸 들여쓰기로 정렬됩니다.</li>
  <li><strong>압축</strong>을 클릭하면 한 줄로 압축됩니다.</li>
  <li><strong>검증</strong>을 클릭하면 구문 오류를 확인합니다.</li>

<h2>JSON 포매팅에 대하여</h2>
<p>JSON(JavaScript Object Notation)은 가장 널리 사용되는 데이터 교환 형식입니다. 포매터는 들여쓰기를 추가하여 중첩 구조를 읽기 쉽게 하고, 압축 도구는 공백을 제거하여 데이터 크기를 줄입니다. 어느 작업도 데이터 손실은 없습니다.</p>
<p>이 도구는 브라우저에서 완전히 동작하며, 데이터가 기기 밖으로 나가는 일은 없습니다.</p>
