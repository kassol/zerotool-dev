<h2>使用方法</h2>

<h3>生成模式</h3>
<ol>
  <li>在<strong>明文密码</strong>输入框中输入密码。</li>
  <li>调整<strong>计算轮次</strong>滑块（默认：12），预估耗时实时更新。</li>
  <li>点击<strong>生成 Hash</strong>。Web Worker 在后台计算时界面保持响应。</li>
  <li>点击<strong>复制</strong>将哈希值复制到剪贴板。</li>
</ol>

<h3>验证模式</h3>
<ol>
  <li>切换到<strong>验证</strong>标签。</li>
  <li>输入明文密码和已存储的 bcrypt 哈希值。</li>
  <li>点击<strong>验证</strong>，结果显示<strong>匹配</strong>或<strong>不匹配</strong>。</li>
</ol>

<h2>安全提示</h2>
<ul>
  <li>切勿用 MD5、SHA-1 或无盐 SHA-256 存储密码——它们速度过快，容易被彩虹表攻击。</li>
  <li>bcrypt 哈希输出中包含计算轮次，已存储的旧哈希在提升轮次后仍可正确验证。</li>
  <li>本工具使用 <a href="https://github.com/dcodeIO/bcrypt.js" target="_blank" rel="noopener">bcryptjs</a> 库——与所有 bcrypt 实现兼容的纯 JavaScript 移植版。</li>
</ul>
