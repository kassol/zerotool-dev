<h2>사용 방법</h2>

<h3>생성 모드</h3>
<ol>
  <li><strong>비밀번호</strong> 필드에 평문 비밀번호를 입력합니다.</li>
  <li><strong>비용 계수</strong> 슬라이더를 조정합니다(기본값: 12). 예상 처리 시간이 실시간으로 업데이트됩니다.</li>
  <li><strong>해시 생성</strong>을 클릭합니다. Web Worker가 계산하는 동안 UI는 응답성을 유지합니다.</li>
  <li><strong>복사</strong>를 클릭하여 해시를 클립보드에 복사합니다.</li>
</ol>

<h3>검증 모드</h3>
<ol>
  <li><strong>검증</strong> 탭으로 전환합니다.</li>
  <li>평문 비밀번호와 저장된 bcrypt 해시를 입력합니다.</li>
  <li><strong>검증</strong>을 클릭합니다. 결과에 <strong>일치</strong> 또는 <strong>불일치</strong>가 표시됩니다.</li>
</ol>

<ul>
  <li>비밀번호에 MD5, SHA-1, 또는 솔트 없는 SHA-256을 절대 사용하지 마세요. 너무 빠르고 레인보우 테이블 공격에 취약합니다.</li>

</ul>
